# Payment Gateways

Use payment gateways to pass authorization, payments, and settlement data securely to and from the merchant's website to the merchant's processor.

For more information about payment gateways, see [Payment Gateways](https://knowledgecenter.zuora.com/Billing/Billing_and_Payments/M_Payment_Gateways).
