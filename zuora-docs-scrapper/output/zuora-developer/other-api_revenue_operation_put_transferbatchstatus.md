## Update transfer batch status

Updates the transfer batch status to `Transferred` in Zuora Revenue.

Request

##### path Parameters

| batch_idrequired | integerThe batch ID of the transfer accounting batch. |
| --- | --- |

##### header Parameters

| tokenrequired | stringThe valid authentication token that is generated by the Authentication operation. |
| --- | --- |

Responses

200

The status of the transfer batch is changed to `Transferred`.

400

Only the batch in Ready TO Transfer status can be updated.
