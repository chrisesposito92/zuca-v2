## Get field mapping

Gets the field mapping information in the upload template definitions for transactions, events, or bundle configuration from Zuora Revenue.

Request

##### query Parameters

| templatename | stringName of the upload template that contains the field mapping information. |
| --- | --- |

##### header Parameters

| tokenrequired | stringThe valid authentication token that is generated by the Authentication operation. |
| --- | --- |

Responses

200

Field mapping data is returned.

400

Error fetching the mapping details.

get/api/integration/v1/upload/mapping
