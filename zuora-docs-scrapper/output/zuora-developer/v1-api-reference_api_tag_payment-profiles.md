# Payment Profiles

Payment profiles enable payment flexibility for subscribers by allowing payment methods to be associated with specific subscriptions.

To learn more about the configuration and usage of Payment Profiles, see [Payment Profiles](https://knowledgecenter.zuora.com/Zuora_Payments/Payment_Operations/Payment_Profiles).
