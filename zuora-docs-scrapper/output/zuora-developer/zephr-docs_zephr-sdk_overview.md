#

Zephr SDK

Zephr uses web APIs for all its functionality. These are split into two categories: the Admin API and the Public API. All functionality of the system can be controlled by the Admin API. The Public API is used for client-side implementations and is tightly linked to a user’s session.

You can read the Admin API and Public API specifications online.

The Zephr SDK is a wrapper around the Zephr APIs allowing for rapid integration development. The Zephr SDK is supplied in JavaScript, Browser SDK, Swift, and PHP.

For Zephr SDK tutorials, see the following:

-   [Server-side Integration Guide using Zephr JavaScript SDK](https://developer.zuora.com/zephr-docs/zephr-sdk/zephr-sdk-tutorials/server-side-integration-guide-using-zephr-javascript-sdk "Server-side Integration Guide using Zephr JavaScript SDK")  for integration instructions.
-   [OAuth on iOS using the Swift SDK](https://developer.zuora.com/zephr-docs/zephr-sdk/zephr-sdk-tutorials/oauth-on-ios-using-the-swift-sdk "OAuth on iOS using the Swift SDK") for integration with Zephr’s OAuth social sign-in.

For Zephr SDK references, see the following:

-   [JavaScript](https://developer.zuora.com/zephr-docs/zephr-sdk/zephr-sdk-reference/javascript-sdk "JavaScript SDK")
-   [Browser SDK](https://developer.zuora.com/zephr-docs/zephr-sdk/zephr-sdk-reference/browser-sdk "Browser SDK")
-   [Swift](https://developer.zuora.com/zephr-docs/zephr-sdk/zephr-sdk-reference/swift-sdk "Swift SDK")  (recommended for iOS development)
-   [PHP](https://developer.zuora.com/zephr-docs/zephr-sdk/zephr-sdk-reference/php-sdk "PHP SDK")

**Note**: For Android, we recommend making HTTP calls to our Web API using the HTTP client of your choice.
