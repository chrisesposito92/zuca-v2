## Get report list

Gets a list of available reports that are created on a specified date.

Request

##### query Parameters

| createddate | string <date>The date when reports are created. |
| --- | --- |

##### header Parameters

| tokenrequired | stringThe valid authentication token that is generated by the Authentication operation. |
| --- | --- |

Responses

200

Operation is successful.

400

The specified date format is invalid.

get/api/integration/v1/reports/list
