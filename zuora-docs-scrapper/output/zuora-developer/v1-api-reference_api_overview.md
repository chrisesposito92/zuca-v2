Welcome to the v1 API reference for the Zuora Billing, Payments, and Platform!

To learn about the common use cases of the Zuora v1 API, check out the [API Tutorials](https://developer.zuora.com/docs/get-started/tutorials/).

**Important**: This API Reference only showcases the parameters, fields, and behaviors available for the latest API minor version (`2025-08-12`). For more information about Zuora API versions, see [API versions](https://developer.zuora.com/v1-api-reference/api-versions/) and [API upgrades](https://developer.zuora.com/v1-api-reference/api-upgrades/).

We recommend that you upgrade to the latest API version to take advantage of the latest Zuora capabilities. For instructions about upgrading the API version, see [API upgrade guide](https://developer.zuora.com/v1-api-reference/api-upgrade-guide/).

### More OpenAPI specification options

To support your integration with our OpenAPI specifications, whether using in-house solutions or third-party tools, we provide the following resources:

-   **Order-to-Cash operations (detailed)**: Download [Order-to-Cash OpenAPI spec](https://developer.zuora.com/yaml/zuora-openapi-for-otc.yaml), which includes full descriptions and examples
-   **Order-to-Cash operations (compact)**: Download [Order-to-Cash Compact OpenAPI spec](https://developer.zuora.com/yaml/zuora-openapi-for-otc-compact.yaml), which excludes descriptions and code examples.
-   **Full OpenAPI spec (compact)**: Download [Compact OpenAPI Spec](https://developer.zuora.com/yaml/zuora-openapi-full-compact.yaml), which covers the entire API set without descriptions or samples.

If you need a different API specification beyond the options above, contact [Zuora Global Support](https://support.zuora.com), and we will provide a custom spec tailored to your needs.
