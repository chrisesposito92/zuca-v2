# Tax Health

Zuora System Health dashboard for Tax Health (Tax Health dashboard) collects and displays usage, failure, and performance data about Tax in near real time. It enables you to view all the tax details in your Zuora tenant within the last 30 days. For more information, see [Tax Health Dashboard](https://knowledgecenter.zuora.com/Zuora_Billing/B_Set_up_Zuora_Billing/Apply_taxes/A_Zuora_Tax).
