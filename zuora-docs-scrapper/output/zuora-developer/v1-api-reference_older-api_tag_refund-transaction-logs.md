# Refund Transaction Logs

This section contains the [CRUD: Retrieve a refund transaction log](https://developer.zuora.com/api-references/older-api/operation/Object_GETRefundTransactionLog/) operation. This operation allows you to export a log of all transactions from Zuora to the gateway for the refund.
