# Billing Documents

Billing documents include invoices, credit memos, and debit memos.

**Note**: Credit memos and debit memos are only available if you have the [Invoice Settlement](https://knowledgecenter.zuora.com/Billing/Billing_and_Payments/Invoice_Settlement) feature enabled.
