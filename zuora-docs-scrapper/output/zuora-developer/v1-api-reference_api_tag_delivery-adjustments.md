# Delivery Adjustments

Delivery Adjustments are used to handle end customer delivery complaints for the Delivery Pricing charge model. For more information, see [Delivery Adjustments](https://knowledgecenter.zuora.com/Zuora_Billing/Bill_your_customers/Delivery_Adjustments).
