# BI Views v1

The BI Views V1 operations are used to download data for Zuora Revenue standard BI views.

**Note:**

-   Only the standard Zuora Revenue BI views are supported.
-   If the data volume to be processed exceeds 100k records per query, it is recommended to use [BI Views V2](https://developer.zuora.com/other-api/revenue/tag/BI-Views-v2/) operations.
