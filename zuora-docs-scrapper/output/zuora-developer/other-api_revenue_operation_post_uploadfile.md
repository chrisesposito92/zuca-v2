## Upload file

Upload the event or transaction data in a CSV file to Zuora Revenue in UTF-8.

**Note:** It is recommended to have up to 60k lines per request in this operation.

Request

##### header Parameters

| tokenrequired | stringThe valid authentication token that is generated by the Authentication operation. |
| --- | --- |
| templatenamerequired | stringName of the upload template that you want to use. |
| Content-Typerequired | stringMust be set to "multipart/form-data".Value: "multipart/form-data" |

##### Request Body schema: multipart/form-data
required

| filerequired | string <binary>The path to the CSV file that is to be uploaded. |
| --- | --- |

Responses

200

The file is uploaded to Zuora Revenue.

400

Exception occurs. Please contact Zuora Revenue Support.

post/api/integration/v1/upload/file
