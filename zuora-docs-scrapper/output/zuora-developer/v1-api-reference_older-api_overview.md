# Older API Reference (2025-12-17)

Download OpenAPI specification:[Download](https://developer.zuora.com/yaml/older_api.yaml)

E-mail: [docs@zuora.com](mailto:docs@zuora.com)

This API Reference contains the API operations that we no longer recommend you use.

Although we do not recommend these API operations, you can continue to use these API operations if you have integrated with them. You are not required to make changes to your existing integration.

For more information about this change, see [this Community post](https://community.zuora.com/discussion/upcoming-reorganization-of-zuora-api-reference-december-2022#bm0903f6c6-38c9-4c8c-a371-9438cc64bd34).
