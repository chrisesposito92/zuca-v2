# Payment Method Updater

Zuora Payment Method Updater (PMU) enables merchants to automatically incorporate changes made to a customer's credit cards. For more information about Zuora PMU, see [Payment Method Updater](https://knowledgecenter.zuora.com/Zuora_Payments/Payment_Method_Updater).
