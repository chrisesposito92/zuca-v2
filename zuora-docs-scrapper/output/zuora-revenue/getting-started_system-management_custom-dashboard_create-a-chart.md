# Create a chart

Learn how to create and configure a chart for your custom dashboard by following these steps.

1.  Navigate to Setups > Application .
2.  Click the left pointing arrow to open the side menu and then click Report Setup . All reports are listed.
3.  Locate the report based on which you want to build your own dashboard and click the Edit icon .
4.  In the Layouts tab of the Edit Report window, click the add (+) icon to create a new layout.
5.  Configure the layout details as necessary in the Fields , Field Details , and Filters tab. For more information, see [Create Layouts](/zuora-revenue/month-end-process/reports/work-with-reports#concept-dlkb68eg__title-412) .
6.  In the Drill Layouts tab, select the layout that you just created from the Available Layouts section and move it to the Selected Layouts section.
7.  In the Chart Config tab, complete the following steps:
    1.  In the Chart Category field, specify whether to use line chart, bar chart and pie chart in your dashboard.
    2.  In the Chart Type field, select the appropriate chart type.
    3.  In the Attribute field, select the field name based on which the chart is to be categorized.
    4.  In the Metric field, select the field name whose value is to be displayed for each unit.
8.  Click the Preview icon to preview the chart, make changes as necessary, and then close the preview chart.
9.  Click the save icon in the New Layout window and then close the window.

    Note: This chart will be available to be added to your custom dashboard as instructed in Step 4 ​[here](/zuora-revenue/getting-started/system-management/custom-dashboard/create-a-chart/create-custom-dashboard)​. Create as many charts as you need before you proceed to create the custom dashboard.
