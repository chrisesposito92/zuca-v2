# Create custom dashboard

Learn how to create a custom dashboard in Zuora Revenue by following a series of steps, including selecting templates, adding charts, and configuring settings.

Complete the following steps to create a custom dashboard in Zuora Revenue:

1.  Navigate to Dashboard > Configurator . All your configured dashboards are listed on the Dashboard page.
2.  To create a custom dashboard, click the New Dashboard icon .
3.  In the Dashboard Template tab, click one of the layout templates to specify the way that you want Zuora Revenue to present the information. Your selection is highlighted in green.
4.  Click Continue to Charts to open the Charts tab. The available charts are listed for your selection.
5.  Drag and drop the listed charts one by one to the layout template on the right.
6.  Click Continue To Save to open the Save tab.
7.  Provide a unique name for the custom dashboard and a short description.
8.  Use the toggle switches in the Save tab to specify the following things:

    -   Enabled: Whether to enable this dashboard.
    -   System Default: Whether this dashboard is the default dashboard to display for the system.
    -   User Default: Whether this dashboard is the default dashboard to display for the current user.
    -   Private: Whether this dashboard is visible to other users.

9.  After you are done, click Save .
10.  To display the newly created dashboard, log out and log in again.

     The custom dashboard is created. If this dashboard is set to be the default, it is displayed after you log into Zuora Revenue. Otherwise, navigate to Dashboard > Summary Dashboard and then use the side menu to locate and open the dashboard.
