# Zuora Revenue cloud conversion

Zuora Revenue versions 2.3 to 3.6 will reach EOL on February 1, 2020, with support until July 31, 2021. Upgrade to the latest cloud version for enhanced features and compliance.

Beginning from February 1, 2020, Zuora Revenue versions from 2.3 to 3.6 will reach the end of life (EOL) date. Support and bug fixes will still be provided for the next 18 months until July 31, 2021. During this period, no enhancements will be delivered for these versions.

If you use a previous version of Zuora Revenue for revenue management, it is recommended to upgrade to the latest version of Zuora Revenue, which is a cloud-based solution and designed to support ASC 606 compliance with scalability, automated product updates, lower IT costs, greater mobility, and more security. It is also recommended for on-premises Zuora Revenue users to migrate to the cloud environment. To help you make decisions, a list of frequently asked questions about the Zuora Revenue cloud platform and migration from on-premises to cloud is provided. For more information, see FAQs .
