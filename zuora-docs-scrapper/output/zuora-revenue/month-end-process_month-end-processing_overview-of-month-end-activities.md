# Overview of month-end activities

Learn about the month-end activities in Zuora Revenue, including prerequisite, period-close, and period-open tasks, as well as optional application setups to streamline the process.

As part of the accounting period end process, you must perform the month-end activities within Zuora Revenue. The month-end activities are categorized into the following:

-   Prerequisite activities to be performed before the period-close process can be started, including:

    -   Report
    -   Program
    -   Workbench
    -   Manual task
-   [Period-close activities](/zuora-revenue/month-end-process/period-close-activities) to be performed to close a period, including:

    -   Program
    -   Report
-   Period-open activities to be performed to open a period, including:

    -   Program
    -   Manual task

Zuora Revenue also provides optional application setups to facilitate month-end activities. For example, you can set up a soft freeze for the period-close or period-open process to restrict users from making updates which may cause data discrepancies. Another option is to create a period-close or period-open template to list users' tasks. For more information, see Application setups for month-end process .
