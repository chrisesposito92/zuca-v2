# IP addresses used by Zuora Revenue

Zuora Revenue utilizes a fixed set of IP addresses for communication requests, which must be allowed in your network setup to ensure remote access via VPN.

A fixed set of IP addresses is used by Zuora Revenue for inbound and outbound communication requests. As part of your network setup, you must ensure the IP addresses are not blocked in the network. To provide remote access to Zuora Revenue, set up the VPN profile to add these IP addresses to the VPN routing with the help of your IT team.
