# Zuora Revenue network

Ensure Zuora Revenue IP addresses are unblocked, test connectivity, and enable unified authentication for seamless UI experience.

As part of your network setup, you must ensure the IP addresses used by Zuora Revenue are not blocked in the network. You can test the network connectivity and subscribe to the community to get maintenance notifications. You can also choose to enable unified authentication to have a seamless UI experience across different Zuora products.
