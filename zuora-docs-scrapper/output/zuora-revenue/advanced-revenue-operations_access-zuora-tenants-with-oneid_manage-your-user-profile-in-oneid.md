# Manage your user profile in OneID

This guide explains how to manage your user profile, view user information, and reset your password in OneID.

This article describes how to view or edit your user profile, and reset your password in OneID. These operations apply to all users in OneID.

## View your user profile

To view your user profile, take the following steps:

1.  Click your avatar in the upper right of the OneID portal, and click Profile. The user profile page opens.
2.  View your user information in the Basic Information section.
3.  View the linked tenant user and tenant information in the Linked Users section: For more information about linking the Zuora tenants to your user profile, see Link tenant users to OneID.
    -   Tenant Id: you can click the tenant ID to access this tenant. For more information, see [Access Zuora tenants with OneID](/zuora-revenue/advanced-revenue-operations/access-zuora-tenants-with-oneid).
    -   Tenant Name: you can click the tenant name to access this tenant. For more information, see [Access Zuora tenants with OneID](/zuora-revenue/advanced-revenue-operations/access-zuora-tenants-with-oneid).
    -   Entity Id: the entity ID.
    -   Entity Name: the entity name.
    -   Domain: the URL to access the tenant.
    -   Status: the linking status of the tenant.
    -   Default tenant: indicates the default tenant of your user profile. When you log in to OneID via single sign-on (SSO), OneID will automatically access the default tenant. For more information about the default tenant, see Set the default tenant in OneID.For more information about linking the Zuora tenants to your user profile, see Link tenant users to OneID.
4.  View user group information in the User Groups section. All the user groups you belong to are listed here. Each row in the table represents a tenant that you have access to.
    -   Tenants: the entity name
    -   User Group: the user group that contains access to the tenant
    -   Tenant Id: the tenant ID
    -   User Role: the user role from which you obtain permissions for the tenant

## Edit your user profile

To edit the basic information of your user profile, take the following steps:

1.  Click your avatar in the upper right of the OneID portal, and click Profile. The user profile page opens.
2.  Click Edit in the Basic Information section.
3.  Update the basic information fields as needed.
4.  Click Save. You can find your user profile is updated on the user profile page.

Note that as a standard user, you can edit your profile on the user profile page. However, as an organization admin, you can edit any user's profile on the Users page under Admin Console. For more information, see [Manage users in OneID](/zuora-revenue/advanced-revenue-operations/manage-users-in-oneid).

## Reset your password

To reset your password, take the following steps:

1.  Click your avatar in the upper right of the OneID portal, and click Profile. The user profile page opens.
2.  Click Change Password. The Change Password window pops up.
3.  Provide your current and new password in the window:

    1.  Enter your current password.
    2.  Enter your new password twice.
    3.  Click Change Password.

After the reset, your new password will take effect immediately.
