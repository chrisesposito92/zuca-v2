# Integration Hub

Zuora Integration Hub offers a platform for launching subscription businesses with pre-built applications and tools.

Zuora Integration Hub is a platform that hosts pre-built applications, extensions, interfaces, and tools from technology partners, system integrators, and developers across the globe, making it easier for you to launch subscription businesses based on the Zuora solution.

Zuora Integration Hub is available in the left navigation menu of Zuora.

To ask a question or view discussions about an app, visit Zuora Community .

Currently, the following limitations apply:

-   Integration Hub apps are not available in the EU Data Center. For details, see Zuora Data Centers .

-   Integration Hub apps are not available in Production Copy Environment. See Zuora Environments for more information.
