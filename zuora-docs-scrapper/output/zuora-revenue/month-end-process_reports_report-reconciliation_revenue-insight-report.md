# Revenue Insight Report

Ensure the Revenue Insight Report's R Ttl Rev Activity matches the net revenue for the current period in the Waterfall Report.

To reconcile the Revenue Insight Report, ensure that the R Ttl Rev Activity value in the Revenue Insight Report is the same as the net revenue for the current period in the Waterfall Report.

| Report name | Field in report | Matched value | Matched report |
| --- | --- | --- | --- |
| Revenue Insight Report | R Ttl Rev Activity | Current Period Net Revenue | Waterfall Report |
