# Release Notes based on RevPro 36.011.00

This article provides a summary of enhancements and resolved issues for Billing - Revenue Integration based on RevPro 36.011.00 and subsequent minor versions.

## Enhancements

This section includes the enhancements for RevPro version 36.011.00 and the subsequent minor versions.

## 36.011.00

## Transformation Job UI now available under Data Interface

The UI for the Data Transformation job configuration is now available under Data Interface > Data Sync . This enhancement unifies the entry point for the jobs specific to Billing - Revenue Integration, and facilitates your job scheduling.
