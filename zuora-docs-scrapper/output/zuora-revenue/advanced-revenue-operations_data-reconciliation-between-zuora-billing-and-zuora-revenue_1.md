# Data reconciliation between Zuora Billing and Zuora Revenue

Data reconciliation can be a complicated process that usually takes hours of effort to compare transaction data between two systems. To save you from the error-prone and time-consuming reconciliation work, Billing - Revenue Integration offers you the following reports for automated reconciliation:

-   Billing to Revenue Booking Report

-   Billing to Revenue Billing Report


These reports automatically compare the booking data and billing data respectively between Zuora Billing and Zuora Revenue. With these reports, you can easily identify the transaction mismatch between the two systems as well as the underlying issues.

See the following articles to learn how to reconcile transaction data between the two systems:

-   [Reconcile booking data](/zuora-revenue/advanced-revenue-operations/data-reconciliation-between-zuora-billing-and-zuora-revenue/reconcile-booking-data-between-billing-and-revenue).

-   [Reconcile billing data](/zuora-revenue/advanced-revenue-operations/data-reconciliation-between-zuora-billing-and-zuora-revenue/reconcile-billing-data-between-billing-and-revenue).
