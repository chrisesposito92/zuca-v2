# Manually create a revenue contract

Create a revenue contract manually for orphan POBs, transaction lines, or RCs using the Workbench.

If there are orphan POBs, transaction lines, or RCs, you can manually create a revenue contract for them.

1.  Navigate to Workbench > Revenue Contracts .
2.  In the Create Manual RC window, select the appropriate RC grouping template from the Grouping Template list.
3.  Select the RC, POBs, or transaction lines from different tabs in the same window and then click Create .
4.  In the Link Comments window, provide your comments and then click Create .

After the RC is created, a message is displayed with the RC ID provided.
