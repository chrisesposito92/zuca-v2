# Reports

Learn how to run, schedule, and download reports using the Reports menu options.

You can use the Reports menu options to do the following things:

-   Manually run reports based on predefined or customized report layouts.

-   Schedule reports to run in the background and have the report results available for download.

-   Download reports that are scheduled for downloads.
