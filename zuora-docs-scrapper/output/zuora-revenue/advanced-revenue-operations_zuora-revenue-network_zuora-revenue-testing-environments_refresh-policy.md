# Refresh policy

This document outlines the refresh policy applicable to various refresh requests, including production copy snapshots and software updates for new releases.

The following table explains the refresh policy that is applicable for different types of refresh requests.

| Request type | Refresh policy guidelines |
| --- | --- |
| Copy the snapshot of the production copy environment | You may refresh the sandbox with the production copy by navigating to Setups > Refresh Sandbox . |
| Update the software for new releases | Before the production release, one sandbox will be updated 4 to 5 weeks before the production release date.The rest of the sandboxes will be updated hours after the production release. |
