# Integration service (BI Views V2)

The BI Views V2 integration service offers new API specifications for downloading Zuora Revenue data, featuring new capabilities, usage patterns, and sample codes.

In 36.008.00, a new version of the integration service is released. It is a new set of BI Views API specifications and referred to as BI Views V2 in the documentation. The BI Views APIs are used to download Zuora Revenue data into your data warehouse systems. This section provides details about the new features of BI Views V2, the suggested pattern to consume these APIs, sample codes, and FAQs.
