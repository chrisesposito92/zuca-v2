# Configure accounting structure

Learn how to configure the accounting structure in Zuora Revenue to ensure successful accounting transfer by creating account segments, account types, and mapping them appropriately.

Configuring accounting structure is an important setup in Zuora Revenue. Based on the accounting structure, when accounting entries are created, Zuora Revenue can derive the appropriate account number to associate with transactions, which determines the success of further accounting transfer.

To configure the accounting structure in Zuora Revenue, the following tasks are involved:

1.  Create account segments.
2.  Create account types.
3.  Map the segments to appropriate account types.
