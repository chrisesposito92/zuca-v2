# Data validation process

Ensure transaction data contains appropriate data points for processing in Zuora Revenue during the revenue recognition process.

When the transaction data is uploaded to Zuora Revenue for revenue recognition process, it is important to ensure that the transaction data contains the appropriate data points that are to be processed in Zuora Revenue.
