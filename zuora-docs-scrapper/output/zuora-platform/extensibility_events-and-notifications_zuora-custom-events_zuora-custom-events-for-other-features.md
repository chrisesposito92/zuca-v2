# Zuora custom events for other features

Describes Zuora custom events for features other than Orders, including Prepaid with Drawdown and Unbilled Usage.

| Name | Related event | Description |
| --- | --- | --- |
| Prepaid Balance Threshold Reached | Prepaid Balance Threshold Reached | The number or percentage of the remaining balance reaches the threshold. |
| Unbilled Usage Threshold Reached | Unbilled Usage Threshold Reached | The rating amount or usage quantity of a rating result reaches the threshold. |
