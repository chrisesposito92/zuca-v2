# Configure security policy settings for your organization

Learn how to configure security policy settings to enhance password protection and tailor security measures to your organization's needs.

This configuration aims to implement robust password protection measures and enable administrators to tailor security policies to fit the organization's requirements and risk tolerance.

1.  Navigate to the OneID portal and click on your user profile located at the top right corner.
2.  Click Settings.
3.  Select Security Policies from the menu.
4.  Click on Edit on the Security Policies page.
5.  Adjust and update the security policy parameters as needed.
6.  Click Save .
