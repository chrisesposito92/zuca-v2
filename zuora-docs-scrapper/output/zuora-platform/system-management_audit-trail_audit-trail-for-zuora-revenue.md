# Audit Trail for Zuora Revenue

The Audit Trail report in Zuora Revenue provides admins with access and activity logs, detailing system configuration changes and user actions over a 7-year retention period.

Zuora Revenue's Audit Trail report allows admins to view access and activity logs. This report displays system configuration changes for the period you specify. The Audit Trail data retention period in Zuora Revenue is 7 years by default.

Note: According to the applicable privacy laws, if you want any of your audit trail data/records deleted, Zuora recommends you open a ticket with support. We will assist you in logging into AWS and manually deleting it all.

When a user or a system adds or changes a record in your environment, a row is added to the Audit Trail report. Each change adds the following information to the report:

-   Action

-   Category

-   Category Name

-   Modified By

-   Modified On

-   Modified Time

-   New Value

-   Old Value

-   Type

-   Type Name
