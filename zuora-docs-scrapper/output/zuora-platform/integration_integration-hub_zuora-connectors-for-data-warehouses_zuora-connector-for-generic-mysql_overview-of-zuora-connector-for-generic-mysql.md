# Overview of Zuora Connector for Generic MySQL

Introduction to Zuora Connector for Generic MySQL

Note:

MySQL Version Support Disclaimer

Effective September 2025:

-   Standard support for MySQL 5.7 is ending.

-   Minimum supported version: MySQL 8.0 (Aurora v3).

-   The MySQL connector will continue to run on 5.7, but treated as best-effort only. New features and enhancements are not guaranteed to work. Issues on 5.7 will receive best-effort support, but fixes may not include backward compatibility.

Enable your organization to access Zuora data directly through Generic MySQL. This integration simplifies the setup process, eliminating common technical hurdles related to data extracts, integrations, and schema updates.

## How the Zuora Connector for Generic MySQL works

The Zuora Connector for Generic MySQL facilitates a direct connection between your Zuora account and your Generic MySQL environment. This connection is established through a one-time setup process during which you configure your MySQL credentials.

Once configured, the connector automates the synchronization of your Zuora data into your specified Generic MySQL environment on a daily basis. Tailored to seamlessly integrate with Generic MySQL, this connector enhances your data storage and analytics capabilities within your infrastructure.
