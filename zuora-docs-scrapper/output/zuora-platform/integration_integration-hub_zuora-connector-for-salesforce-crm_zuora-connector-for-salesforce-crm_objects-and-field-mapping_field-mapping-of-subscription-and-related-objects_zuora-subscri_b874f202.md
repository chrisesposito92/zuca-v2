# Zuora Subscription Fields Synced with Different Values in Zuora Connector

Zuora Subscription Fields are synchronized with varying values in the Zuora Connector for Salesforce CRM, detailing terms and period types.

| Zuora Subscription Field Name | Field Value in Zuora Connector for Salesforce CRM |
| --- | --- |
| Current Term | 12 |
| Current Term Period Type | Month (s) |
| Initial Term | 12 |
| Initial Term Period Type | Month (s) |
| Renewal Term | 6 |
| Renewal Term Period Type | Month (s) |
