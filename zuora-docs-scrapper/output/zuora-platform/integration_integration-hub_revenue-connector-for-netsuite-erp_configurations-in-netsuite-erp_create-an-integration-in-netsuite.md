# Create an integration in NetSuite

Learn how to create a new integration in NetSuite to integrate with Zuora Revenue.

1.  Navigate to .
2.  Click New.
3.  Enter Zuora Revenue in the Name field.
4.  Enter Integration with Zuora Revenue in the Description field.
5.  Enter the preferred concurrency in the Concurrency limit field.
6.  Select the applicable value from the State drop-down field.
7.  Select the Token-based Authentication checkbox in the Authentication section.
8.  Select REST WebServices and SuiteAnalytics Connect checkboxes from the OAuth 2.0 section.
9.  Save your entries.
10.  Copy and save the Client ID/ Consumer Key and Client Secret/Consumer Secret generated after you saved the integration.
