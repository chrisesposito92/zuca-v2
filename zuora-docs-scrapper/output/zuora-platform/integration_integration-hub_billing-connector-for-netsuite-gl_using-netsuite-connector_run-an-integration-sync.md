# Run an integration sync

Learn how to run an integration sync

To run an integration sync, complete this procedure.

1.  In Zuora Integration Hub, navigate to My Connect > Tenants .
2.  Hover on the Zuora Login in Tenant Logins list and click Apps .
3.  Click Apps tab in the login details page.
4.  Hover on the connector instance that you use to run the sync. Click Launch .
5.  Click Logs in your instance of Zuora Connector for NetSuite.
6.  Click Run Now .

    The sync process will run in the background and the results will be displayed on the screen. See [View sync activity](/zuora-platform/integration/integration-hub/billing-connector-for-netsuite-gl/using-netsuite-connector/view-sync-activity) for more information. The length of the sync process will vary, depending on the number of records to sync. The process can require up to 30 minutes or more.
