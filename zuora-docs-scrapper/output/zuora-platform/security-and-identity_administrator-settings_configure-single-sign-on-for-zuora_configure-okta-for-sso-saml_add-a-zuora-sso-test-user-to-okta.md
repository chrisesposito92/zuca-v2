# Add a Zuora SSO test user to Okta

Learn how to add a Zuora SSO test user to Okta.

Add a test user to your company's Okta account so that you can test SSO authentication against Zuora. You need to provide this test user's login information to Zuora for the Zuora SSO provisioning process.

1.  From your Okta dashboard, click the People tab.
2.  Click Add Person .
3.  Select a user type in the User type list or accept the default, then complete the required configuration. See [Okta documentation](https://help.okta.com/en-us/content/topics/users-groups-profiles/usgp-add-users.htm) for details.
4.  Click Save.
