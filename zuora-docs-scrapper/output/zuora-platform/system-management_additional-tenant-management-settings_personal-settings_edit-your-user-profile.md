# Edit your user profile

Learn how to edit your user profile settings, including your email, user role of each product, locale, and language used in the Zuora UI.

1.  Click your avatar in the upper right of the Zuora UI and then click Profile. The Personal Settings page opens.
2.  Click Manage Your Profile.
3.  Edit the following information as needed:

    | Field | Description |
    | --- | --- |
    | First Name | Specify your first name. |
    | Last Name | Specify your last name. |
    | Work Email | Enter the email address that you want Zuora to use when sending notifications. This field is required. |
    | Zuora Platform Role | This read-only field displays your Zuora Platform Role. This is set by your tenant administrator. See Manage users for more information. |
    | Billing Role | Select your Billing role, if multiple roles have been configured for your tenant. |
    | Payments Role | Select your Payments role, if multiple roles have been configured for your tenant. |
    | Finance Role | Select your Finance role, if multiple roles have been configured for your tenant. |
    | Commerce Role | Select your Commerce role, if multiple roles have been configured for your tenant. |
    | Reporting Role | Select your Reporting role, if multiple roles have been configured for your tenant. |
    | Locale | Select your locale. It controls date formatting in your tenant. By default, the locale is English (United States). See Locale setting and date formatting for details. |
    | Language | Select the language that is used on the Zuora UI. Zuora supports the following languages. By default, the language is English (United States).EnglishFrenchGermanJapaneseSpanishSee Language setting for details. |
    | Job Title | Specify your job title. Your job title does not affect your Zuora business logic, business objects, or your role permission. Zuora uses this information to improve our services and user interactions. |
    | Job Function(s) | Specify your job functions. You can select multiple functions of your job. If you select the last item with a blank text box, you must fill in the blank. This is a required field.Your job functions do not affect your Zuora business logic, business objects, or your role permission. Zuora uses this information to improve our services and user interactions. |
    | Tenant ID | This read-only field shows the ID of your Zuora tenant. In some cases, when you contact Zuora Support with an issue, the person helping you may ask you to provide the tenant ID. |
    | Zuora ID | This read-only field provides an easier way for you to find status details on the Zuora System Status Page and determine if your Zuora tenant is impacted by any incident. If your Zuora ID is included in an incident post or a post claims that it affects all customers, it indicates that your implementation is affected. |

4.  Click Save.
