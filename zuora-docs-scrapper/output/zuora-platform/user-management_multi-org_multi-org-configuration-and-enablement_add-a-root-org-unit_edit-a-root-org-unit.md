# Edit a root org unit

Learn how to edit the root organization unit.

1.  Navigate to Administration > Organization Hierarchy Management .
2.  Click the ellipsis icon next to the root organization's name and select Edit.
3.  Make the required changes and click Save

    Note: The reporting currency of the root org cannot be changed.
