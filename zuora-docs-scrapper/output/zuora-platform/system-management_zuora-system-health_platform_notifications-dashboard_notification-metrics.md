# Notification metrics

Descriptions of the notification metrics

The following table provides descriptions of the notification metrics.

| Metric | Description |
| --- | --- |
| Notification Usage | All the email and callout notifications issued within a specified time range. |
| Notification Performance | The response time data (maximum, minimum and mean) of the email and callout notifications within a specified time range. |
| Notification Failures | All the failed email and callout notifications issued within a specified time range. |

Note that you can configure notifications based on the Notification Performance and Failures metrics. For more information, see [Standard events for Zuora Central Platform](/zuora-platform/extensibility/events-and-notifications/standard-events/standard-events-for-zuora-platform).
