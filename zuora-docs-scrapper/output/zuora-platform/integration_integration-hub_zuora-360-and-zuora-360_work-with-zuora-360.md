# Work with Zuora 360+

Zuora 360+ uses the Real-Time Sync mode to sync object data, which allows record changes to be synced automatically from Zuora to Salesforce in real time. To understand the Real-Time Sync and its limitations, see [Real-time sync](/zuora-platform/integration/integration-hub/zuora-360-and-zuora-360/zuora-360-overview/synchronize-data-from-zuora-to-salesforce/real-time-sync-for-zuora-360).
