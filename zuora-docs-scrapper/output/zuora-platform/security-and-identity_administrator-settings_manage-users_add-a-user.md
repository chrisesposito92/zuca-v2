# Add a user

Learn how to create a user account for your tenant.

1.  Click your username at the top right, and navigate to Administration Settings > Manage Users.
2.  Click Add single user .
3.  Complete the required user information and optional information as needed. See Personal settings for more information about the fields that you can set when creating a user.

    -   You can select a Zuora Platform Role and other roles for the user. You define and manage user roles on the Manage User Roles settings page.

    -   The Locale field controls date formatting for that user. By default, the locale is English (United States).
