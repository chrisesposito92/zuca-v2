# Subscription Charge Delivery Schedule data source

Data source to export the subscription charge delivery schedule data

## Overview

Use this data source to export the subscription charge delivery schedule data, such as the delivery days and delivery frequency.

## Objects available in the data source

The following sections list the objects available in the data source.

## Base object

| Object | Description |
| --- | --- |
| Subscription Charge Delivery Schedule | Fields on a subscription charge delivery schedule. It includes the following fields:Created By IDCreated DateDayOfMonthFrequency - The subscribed product or service is delivered every week.FridayIDMondayMonthOfYearSaturdaySundayThursdayTuesdayUpdated By IDUpdated DateWednesday |
