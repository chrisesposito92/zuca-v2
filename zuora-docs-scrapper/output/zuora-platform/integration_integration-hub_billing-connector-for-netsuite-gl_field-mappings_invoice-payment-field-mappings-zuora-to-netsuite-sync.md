# Invoice Payment Field Mappings: Zuora to NetSuite Sync

This reference describes the mappings of the invoice payment information sync from Zuora to NetSuite. Note that a payment may be applied to more than one invoice.

In some cases, the Source and Destination fields are associated with different objects.
