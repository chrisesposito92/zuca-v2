# Configure S/4HANA public cloud destination

Learn how to configure the S/4HANA public cloud destination by navigating through the Integration Hub, selecting SAP GL (Billing), and adding a new destination with the required details.

1.  Navigate to Marketplace > Integration Hub .
2.  Search and select SAP GL (Billing) .
3.  Select the Settings tab.
4.  Click the Add a Destination + icon.
5.  Enter the Customer ID , Destination Name , and Destination System .
6.  Click Add .
7.  Click Save .
