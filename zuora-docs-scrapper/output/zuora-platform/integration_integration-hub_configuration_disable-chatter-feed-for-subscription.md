# Disable Chatter Feed for Subscription

Learn how to disable the Salesforce Chatter feed for subscription updates.

By default, the Salesforce Chatter feed for subscription updates are enabled.

To disable the Chatter feed on subscriptions :

1.  In your Salesforce org, navigate to Setup > Build > Develop > Custom Settings and click Manage next to Zuora Config .
2.  Click Edit .
3.  Select the Disable chatter for subscriptions field.
4.  Click Save .
