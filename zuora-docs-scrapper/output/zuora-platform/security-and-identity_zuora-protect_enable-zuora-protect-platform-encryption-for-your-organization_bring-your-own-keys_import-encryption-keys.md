# Import encryption keys

Learn how to import encryption keys from your external key store into Zuora.

To import your encryption keys from your external key store:

1.  Navigate to the key store details page and click Connect .
2.  Enter the KMS Key ID and Key Name.
3.  Click Save.

All the imported keys will be listed under the corresponding Key Store in Zuora.
