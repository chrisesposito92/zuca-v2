# Metrics available in the Billed Usage

This document provides descriptions of metrics available in the Billed Usage tab, including trends and breakdowns for detailed analysis.

## Billed Usage

The following table provides descriptions of the metrics available in the Billed Usage tab:

| Metrics | Description |
| --- | --- |
| Billed Usage Trends | This metric displays the patterns and changes in billed usage over a selected time period.To compare the analysis with the previous month or year, select the applicable value from the Compare To drop-down field.To review the anomaly or forecast analysis, click Anomaly+Forecast button. |
| Billed Usage Breakdown | Displays a detailed breakdown of billed usage by product and rate plan, segmented by month, week, or day for granular revenue analysis. |
