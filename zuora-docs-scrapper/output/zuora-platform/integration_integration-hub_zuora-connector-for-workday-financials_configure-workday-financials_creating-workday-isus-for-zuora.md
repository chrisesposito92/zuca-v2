# Creating Workday ISUs for Zuora

Guides you through the process of creating Integration System Users (ISUs) in Workday for use with Zuora.

1.  Log in and access the Workday console using your administrator credentials.
2.  Type "Create Integration System User" in the search bar and press Enter .
3.  Enter appropriate values for the following fields:

    -   User Name: ISU\_Zuora\_Get

    -   New Password

    -   New Password Verify

    -   Check the Do Not Allow UI Sessions box.


4.  Click OK .
