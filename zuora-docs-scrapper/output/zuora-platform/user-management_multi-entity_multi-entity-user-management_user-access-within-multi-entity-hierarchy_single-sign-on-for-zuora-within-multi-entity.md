# Single sign-on for Zuora within Multi-entity

Learn to manage single sign-on (SSO) for entity users in a Zuora multi-entity environment.

You can enable or disable single sign-on (SSO) for any entity users in Zuora multi-entity environment.

If you get this feature enabled for your global entity, all the sub-entity will get this feature enabled too. Zuora also allows you to disable this feature. If you disable this feature for your global entity, all sub-entities that have this feature enabled will be disabled too.

Navigate to Administration > Manage Users to configure SSO for users.

See Configure Single Sign-On for Zuora for more information.
