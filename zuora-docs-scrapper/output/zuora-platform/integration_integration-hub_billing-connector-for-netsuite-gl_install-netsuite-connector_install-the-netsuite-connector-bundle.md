# Install the NetSuite Connector bundle

Learn how to install the NetSuite Connector bundle

The NetSuite Connector "bundle" is a predefined set of configurations used to distribute the customizations required by NetSuite Connector. The customizations include custom fields, lists, forms, and other customizations. Updated versions of the bundle will be delivered automatically to all NetSuite Connector client accounts. The bundle can be installed only by a NetSuite Administrator user.

1.  Install the NetSuite Connector bundle by selecting Customization > SuiteBundler > Search & Install Bundles.
2.  Select Repository from the Location list.
3.  Enter `zuora` in the Keywords field.
4.  Click Search.
5.  Click the NetSuite Connector Product bundle.
6.  In the Bundle Details screen, click Install to begin the installation process.
7.  Use the installation wizard to complete the installation process.
8.  If the installation completes successfully, the bundle will add custom fields to NetSuite. See NetSuite Custom Fields for a complete list.
