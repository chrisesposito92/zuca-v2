# Configure multiple organizations in Quote Studio

Learn how to configure multiple organizations in Quote Studio using Multi-Org for enhanced data and user segmentation.

Zuora now supports customer data and user segmentation using Multi-Org in Quote Studio. This means that data and users can be segmented according to the organizational hierarchy defined in the Multi-Org enabled Zuora Billing tenant, ensuring better control, visibility, and alignment with your business structure.

[For more information, see Multi-Org in Quote Studio](/zuora-cpq/manage-subscriptions/advanced-cpq-x-functionalities/multi-org).
