# Available Templates

Download CSV templates from the Developer Tools UI or the table below. Mac OS users must save the .csv file in MS-DOS Comma Separated format.

You can download the CSV templates from the Developer Tools UI or from the following table. Note that if you are using Microsoft Excel on Mac OS, you must save the .csv file in MS-DOS Comma Separated (.csv) format .

| Object | Template |
| --- | --- |
| Account & Contact | Account & Contact Loader Template |
| Accounting Period | Accounting Period Loader Template |
| Amendment | Amendment Loader Template |
| Product Catalog | Product Catalog Loader Template |
| Subscription | Subscription Loader Template |
| Orders | Orders Loader Template |
