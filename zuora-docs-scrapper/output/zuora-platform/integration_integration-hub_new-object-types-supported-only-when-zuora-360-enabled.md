# New object types supported only when Zuora 360+ enabled

The new object types are supported only with the Zuora 360+ feature, including Subscription, Rate Plan, RatePlanCharge, and Rate Plan Charge Tier history objects.

The following are the new object types that are supported only when you enable the Zuora 360+ feature. These object types are not supported in Zuora 360.

| Invoice Item: Zuora__InvoiceItem__c | Payment Part: Zuora__PaymentPart__c |
| --- | --- |
| Taxation Item: Zuora__TaxationItem__c | Refund Part: Zuora__RefundPart__c |
| Credit Memo: Zuora__CreditMemo__c | Subscription: Zuora__SubscriptionHistory__c |
| Credit Memo Item: Zuora__CreditMemoItem__c | Rate Plan : Zuora__SubscriptionRatePlanHistory__c |
| Credit Memo Part: Zuora__CreditMemoPart__c | RatePlanCharge: Zuora__SubscriptionProductChargeHistory__c |
| Debit Memo: Zuora__DebitMemo__c | Rate Plan Charge Tier: Zuora__SubscriptionChargeTierHistory__c |
| Debit Memo Item: Zuora__DebitMemoItem__c |  |
