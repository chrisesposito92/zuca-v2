# View Zuora custom events

Learn how to view Zuora custom events in the Zuora UI.

1.  Navigate to in the left navigation menu.
2.  Click the Custom Events tab.
    All Zuora custom events, custom events, and custom scheduled events are listed on the page. The Type column indicates the type of each event. You can sort the list by clicking the column name.
