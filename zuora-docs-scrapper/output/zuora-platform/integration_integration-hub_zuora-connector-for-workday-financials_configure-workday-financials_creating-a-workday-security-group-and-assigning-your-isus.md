# Creating a Workday Security Group and assigning your ISUs

Guides you through creating a Workday Security Group and assigning Integration System Users (ISUs).

1.  Login and access the Workday console using your administrator credentials.
2.  Type "Create Security Group" in the search bar and press Enter .
3.  Select Integration System Security Group (Unconstrained) from the Type of Tenanted Security Group drop-down field.
4.  Type "ISSG\_Zuora" in the Name field.
5.  Click OK to access the Edit Integration System Security Group (Unconstrained) section.
6.  Type "ISU\_Zuora\_Get" in the Integration System Users field and press Enter .
7.  Type "ISU\_Zuora\_Put" in the Integration System Users field and press Enter .
8.  Click OK .
9.  Click Done
