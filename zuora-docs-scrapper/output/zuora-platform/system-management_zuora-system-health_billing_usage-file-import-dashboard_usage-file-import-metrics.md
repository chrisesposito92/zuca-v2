# Usage File Import metrics

The following table provides descriptions of the Usage File Import metrics.

| Metric | Description |
| --- | --- |
| Usage File Import | The number of usage file imports within a specified time range, including successful, failed, and under-processing imports.When hovering over each time interval bar, you can view the breakdown data of that time range.Failed : The number of failed usage file imports.Processing : The number of usage file imports that are under processing.Success : The number of successful usage file imports. |
| Usage File Import Errors | The number of failed usage file imports within a specified time range.When hovering over each time interval bar, you can view the breakdown data of that time range. |
| Usage File Import Performance | Usage file import handling time within a specified time range.Usage file import handling time means the duration from the Submission Time to the Completion Time of a usage file import.P95 : 95th percentile of the handling time dataP90 : 90th percentile of the handling time dataP50 : 50th percentile of the handling time dataWhen hovering over each time interval point, you can view the breakdown data of that time range. |
