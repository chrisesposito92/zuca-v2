# View custom fields

Learn how to view custom fields defined on a particular object via the Object Manager.

1.  Navigate to in the left navigation menu. In the Objects tab, Zuora lists the information on all custom objects and standard objects, including the number of custom fields, filterable fields, indexed fields, and records. Note that Zuora displays only the standard objects from the product areas you have permission to access.
2.  Click the object name or the Edit icon ![Edit icon](https://zuora.deploy.heretto.com/v4/deployments/QPAZk6lsgXwvotedNERE/object/6c4955ce-f413-4102-9a7d-92c4df7969a1?jwt=eyJ0eXAiOiJKV1QiLCJhbGciOiJIUzI1NiJ9.eyJodHRwczovL2pvcnNlay5jb20vZXpkX29yZ2FuaXphdGlvbiI6Inp1b3JhIiwiaHR0cHM6Ly9qb3JzZWsuY29tL2V6ZC9vYmplY3RfdXVpZCI6IjZjNDk1NWNlLWY0MTMtNDEwMi05YTdkLTkyYzRkZjc5NjlhMSIsImV4cCI6MTc2NjY0MDA2OSwianRpIjoiMGZhMjgxODljNDhmNDk2MWE1OGU5OThhMmVhMzFlNjAiLCJodHRwczovL2pvcnNlay5jb20vZXpkX2ZpbGVzZXQiOiJWZHdCeUJjM0lBa01wRU9LSFdxZCJ9.ujAc1P0c_V_1AV56evHGGetP8EN5lMZCYc8WLbakmco) to open the detail page of the object you want to view.
3.  View the field details in the Custom Fields tab on the object detail page. Zuora lists all custom fields defined on this object in the Custom Fields tab on the object detail page. A false value in the Legacy column indicates that the field is an advanced custom field. If the value is true, it is a legacy custom field.
