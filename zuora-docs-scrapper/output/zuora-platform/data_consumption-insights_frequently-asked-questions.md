# Frequently Asked Questions

This document provides answers to common questions about Consumption Insights, including data update frequency, forecasting models, and feature availability.

## FAQs

How frequently is the Consumption Insights data updated?

-   The data is refreshed daily, with timestamp in the dashboard.

What forecasting model does Consumption Insights use?

-   We use machine learning models that predict time-series data as well as detect anomalies in time-series data. We score accounts using a variety of algorithms (e.g. Mann-Kendall Test) to score individual accounts.

Will users have to additionally purchase this feature?

-   No, this feature is included as part of of your Billing subscriptions.
