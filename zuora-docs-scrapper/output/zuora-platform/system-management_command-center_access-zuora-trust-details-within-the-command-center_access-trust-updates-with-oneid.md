# Access Trust updates with OneID

Access personalized Trust updates with a unified login experience using OneID across multiple Zuora environments.

If you have OneID access, you can enjoy a unified login experience across multiple Zuora environments and access personalized Trust updates.

To view your personalized Trust updates:

1.  Log in to Command Center at one.zuora.com using your credentials or via SSO SAML.
2.  Navigate to Trust Status on the left-hand menu.
3.  View your personalized Trust updates.
