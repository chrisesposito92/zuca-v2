# Find the Salesforce account IDs

Learn how to generate and export a list of Salesforce Account IDs for active accounts.

To generate a list of Salesforce Account IDs :

1.  On the Reports tab in Salesforce, open the Account and Contact Reports folder.
2.  Click Active Accounts . ![Screenshot of active accounts](/db/organizations/zuora/repositories/prod-sitemap/content/documents/external_publications/platform/integration_hub/zuora_360_plus_and_zuora_360/configuring_topics/Active_Accounts_Reports.png)
3.  Drag and drop Account ID from the Account General folder to the table in the Preview section to add the Account ID field to the report. ![Screenshot of account ID](/db/organizations/zuora/repositories/prod-sitemap/content/documents/external_publications/platform/integration_hub/zuora_360_plus_and_zuora_360/configuring_topics/Add_AccountID.png)
4.  Click Run Report . Account IDs for active Salesforce accounts appear in the report. ![Screenshot of Salesforce account](/db/organizations/zuora/repositories/prod-sitemap/content/documents/external_publications/platform/integration_hub/zuora_360_plus_and_zuora_360/configuring_topics/SFDCAccountID.png)
5.  If you have a large number of accounts to map, export the list for a bulk import into Zuora.
