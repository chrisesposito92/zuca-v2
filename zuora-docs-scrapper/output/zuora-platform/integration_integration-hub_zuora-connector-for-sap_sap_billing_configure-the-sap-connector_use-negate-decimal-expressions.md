# Use negate decimal expressions

Learn how to navigate the Integration Hub, and edit field mappings for whole number or decimal types.

1.  Navigate to Marketplace > Integration Hub .
2.  Search and select the SAP GL (Billing).
3.  Click Field Mapping .
4.  Click the Edit Mappings button.
5.  Select a field of type WHOLE NUMBER or DECIMAL from the Source Path dropdown.
6.  Click Save Mappings .
