# Custom Object: Delete Record

This reference describes the Custom Object: Update Record task.

This task deletes a custom object record.

When creating the task, you need to select the custom object definition and specify the ID of the object.

To learn more about custom object definitions and records, see [custom object definitions](/zuora-platform/extensibility/custom-objects/custom-object-definition-management/create-a-new-custom-object-definition) and [custom object records](/zuora-platform/extensibility/custom-objects/custom-object-record-management/create-a-new-custom-object-record).
