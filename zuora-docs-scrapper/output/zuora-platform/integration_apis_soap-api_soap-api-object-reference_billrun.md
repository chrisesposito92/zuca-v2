# BillRun

The BillRun object allows you to create ad hoc bill runs and Post, Query, Cancel, and Delete bill runs.

This API object does not let you create a scheduled bill run.

The BillRun object is different from the Billing Run data source.

You must be on WSDL version 63.0 or higher to use this feature.

## Supported Calls

You can use this object with the following calls:

-   create()

-   post()

-   delete()

-   query()

-   update()
