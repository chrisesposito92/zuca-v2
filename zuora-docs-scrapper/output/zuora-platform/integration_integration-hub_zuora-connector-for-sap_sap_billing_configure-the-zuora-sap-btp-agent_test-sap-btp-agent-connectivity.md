# Test SAP-BTP agent connectivity

Learn how to test connectivity by logging into the BTP Cockpit, navigating to the Destinations menu, and performing a connectivity check.

1.  Log on to the BTP Cockpit using the user name created for your email ID.
2.  Expand the Connectivity menu item and click Destinations .
3.  Click the Check Destination button to perform a connectivity check, which will return “200 OK” if successful.
