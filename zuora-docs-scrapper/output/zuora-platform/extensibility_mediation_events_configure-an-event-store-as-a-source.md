# Configure an Event Store as a source

Learn how to configure an event store as a source for a custom meter.

To create a custom meter with an event store as the source:

1.  Create a Custom Meter.
2.  From the Sources, select Event Store.
3.  In the Event Store settings page, specify a Name for the source.
4.  Select an existing event store from the list of event stores. You can also create a new event store that is based on an existing event definition for the source data.
5.  Click Save.

The event store is configured as a source for the meter.
