# Configure an Event Store as a source

Learn how to configure an event store as a source for a custom meter.

1.  Navigate to
2.  Create a Custom meter.
3.  From the Sources, select Event Store.
4.  In the Event Store settings page, specify a Name for the source.
5.  Select an existing event store from the list of event stores. You can also create a new event store that is based on an existing event definition for the source data.
6.  Click Save.
