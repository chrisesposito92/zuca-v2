# Configure a SAP destination for GL connector

Learn how to configure a SAP destination by navigating through the Integration Hub and adding necessary details.

1.  Navigate to Marketplace > Integration Hub .
2.  Search and select SAP GL (Revenue) .
3.  Select the Destination tab.
4.  Click the Add a Destination + icon.
5.  Enter the Customer ID , Destination Name , and Destination System .
6.  Click Add .
7.  Click Save .
