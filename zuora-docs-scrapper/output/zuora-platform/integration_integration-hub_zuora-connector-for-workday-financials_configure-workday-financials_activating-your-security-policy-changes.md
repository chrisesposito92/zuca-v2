# Activating your security policy changes

Learn how to activate pending security policy changes in Workday.

1.  Log in and access the Workday console using your administrator credentials.
2.  Type "Activate Pending Security Policy Changes" in the search bar and press Enter .
3.  Type "Approved" in the Comment field and click OK .
4.  Check the Confirm checkbox and click OK
