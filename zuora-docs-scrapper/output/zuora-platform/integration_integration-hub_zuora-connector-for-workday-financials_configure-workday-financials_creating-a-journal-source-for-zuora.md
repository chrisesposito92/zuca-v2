# Creating a Journal Source for Zuora

Guides you through the process of creating a journal source for Zuora in Workday.

1.  Login and access the Workday console using your administrator credentials.
2.  Type "Maintain Journal Sources" in the search bar and press Enter .
3.  Press the + icon and type "Zuora Billing" in the Journal Source Name field.
4.  Check the Accounting Source checkbox.
5.  Click OK .
