# Create user for NetSuite integration with Zuora Revenue

Learn how to create a user for NetSuite integration with Zuora Revenue.

1.  Navigate to .
2.  Click New.
3.  Enter Zuora\_IntegrationUser in the Name field.
4.  Enter the user email ID in the Email field.
5.  Select the administrator role in the Role field on the Access tab.
6.  Save your entries.
