# Data Query metrics

This document provides descriptions and details of Data Query metrics, including usage, failures, and performance within specified time ranges.

The following table provides descriptions of the Data Query metrics:

| Metric | Description |
| --- | --- |
| Data Query Usage | The number of submitted queries within a specified time range, including accepted, completed, failed, and in-progress queries.When hovering over each time interval bar, you can view the breakdown data of that time range.Accepted: queries that are accepted by the query service, or cancelled by the user.Success: queries that are processed successfully by the query service.Failed: queries that are unable to be processed by the query service.In Progress: queries that are being processed by the query service. |
| Data Query Failures | The number and detailed information of failed queries within a specified time range.When hovering over each time interval bar, you can view the breakdown data of that time range. |
| Data Query Performance | The execution time and detailed information of submitted queries within a specified time range.When hovering over each time interval bar, you can view the breakdown data of that time range. |
