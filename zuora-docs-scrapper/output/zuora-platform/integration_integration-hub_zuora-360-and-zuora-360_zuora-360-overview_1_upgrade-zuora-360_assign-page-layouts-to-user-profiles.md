# Assign page layouts to user profiles

Learn how to assign the latest page layouts to sync user profiles using the provided table.

Assign the latest page layouts listed in the following table to your sync user profiles.

| Object | Page Layout to User |
| --- | --- |
| Billing Account (Zuora__CustomerAccount__c) | Billing Account Z-Force Layout v.2.9 |
| Invoice (Zuora__ZInvoice__c) | ZInvoice Z-Force Layout v2.100 |
| Invoice Payment (Zuora__PaymentInvoice__c) | PaymentInvoice Z-Force Layout v2.5 |
| Payment (Zuora__Payment__c) | Payment Z-Force Layout v2.7 |
| Subscription (Zuora__Subscription__c) | Subscription Z-Force Layout v2.6 |
| Subscription Product & Charge (Zuora__SubscriptionProductCharge__c) | Subscription Charge Z-Force Layout 2.9 |

1.  Navigate to Setup > Create > Objects .
2.  Click the object.
3.  In the Page Layouts section, click Page Layout Assignment .
4.  Click Edit Assignment .
5.  Click the profile.
6.  With the profile, click the Page Layout To Use field, and select the page layout shown in the table above.
7.  Repeat the Step #6 to assign all applicable user profiles and page layouts.
8.  Click Save .
