# Create a communication arrangement

Learn to create a communication arrangement in the Communication Management system to integrate with SAP for posting journal entries.

1.  Under Communication Management , choose the Communication Arrangements tile.
2.  On the Communication Arrangements screen, click New and select the scenario Finance - Posting Integration (SAP\_COM\_0002) .
3.  Click Create .
4.  On the next screen, select the Communication System that you prefer to use, then choose a User belonging to the Communication System.
5.  Click Save to save the Communication Arrangement .

Once saved, the URL will be displayed. This URL can be used to configure the Zuora SAP-BTP Agent for posting Journal Entries from Zuora Revenue.
