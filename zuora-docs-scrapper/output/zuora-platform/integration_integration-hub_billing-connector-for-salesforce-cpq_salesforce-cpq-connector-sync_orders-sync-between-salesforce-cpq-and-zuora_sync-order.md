# Sync Order

Learn how to sync orders from Salesforce CPQ to Zuora, including individual and account-level processes.

1.  Perform the following steps for individual Order sync.
    1.  Navigate to the specific order in Salesforce CPQ.
    2.  Click the top-right dropdown.
    3.  Select the Send to Zuora button created during Connector Setup
2.  Perform the following steps to sync all Orders for an Account.
    1.  Navigate to the Salesforce CPQ account.
    2.  Click the Send to Zuora button.
