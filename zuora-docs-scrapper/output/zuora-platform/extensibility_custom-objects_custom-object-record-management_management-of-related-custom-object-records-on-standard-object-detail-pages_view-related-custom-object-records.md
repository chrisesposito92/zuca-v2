# View related custom object records

Learn how to view related custom object records on a standard object detail page.

1.  Open the standard object detail page. For example, if you want to view related custom object records of the Account object, navigate to in the left navigation menu and click a customer name.
2.  View related custom object records:
    1.  Scroll to the Related Objects section. All custom object definitions related to the current standard object are displayed as tabs.
    2.  Click the tab for the custom object that the records belong to. The related custom object records are displayed. You can use the UI Builder for each custom object to configure the columns. For more information, see [UI Builder](/basics/get-familiar-with-zuora-user-interface/zuora-user-interface/configure-a-page-through-ui-builder) .
    3.  If you want to view all related custom object records, click View All.
