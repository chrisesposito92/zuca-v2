# Authentication credentials for NetSuite and Zuora Revenue

Refer to the list of credentials you should save for NetSuite and for Zuora Revenue authentication.

## NetSuite authentication

-   Consumer key and consumer secret

-   Access token details, specifically token ID and token secret

-   NetSuite tenant Account ID

-   NetSuite tenant API services URL, specifically NetSuite Suite Talk API and REST services URL available in the Company URLs tab on the company information page


## Zuora Revenue authentication

-   User name

-   Password

-   Role and client name

-   Revenue tenant host URL


Zuora Revenue users must have access to Revenue web services. Can be an API user or a service account or a business admin persona.

Ensure the user role has full access privileges for Revenue Inbound and Outbound Webservices.
