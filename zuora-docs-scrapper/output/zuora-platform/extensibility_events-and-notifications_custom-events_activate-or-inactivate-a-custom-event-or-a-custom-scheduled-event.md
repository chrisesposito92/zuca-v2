# Activate or inactivate a custom event or a custom scheduled event

Learn how to activate or inactivate a custom event or a custom scheduled event in the Zuora UI.

1.  Navigate to in the left navigation menu.
2.  Click the Custom Events tab.
3.  On the Custom Events page, locate the event you want to activate or inactivate.
4.  Click Activate or Inactivate of the event.
5.  Click Activate or Inactivate in the confirmation dialog. Then you can find the status of the event is updated to Inactive or Active in the Active column.
