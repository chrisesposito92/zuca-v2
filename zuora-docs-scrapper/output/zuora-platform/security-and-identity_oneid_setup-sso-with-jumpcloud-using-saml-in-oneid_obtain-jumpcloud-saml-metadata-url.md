# Obtain JumpCloud SAML metadata URL

Learn how to obtain the JumpCloud SAML metadata URL for Zuora OneID integration.

1.  Log in to JumpCloud and go to the SSO Applications tab.
2.  Click the application for Zuora OneID.
3.  Click the SSO tab.
4.  Under the JumpCloud Metadata section, click Copy Metadata URL .
5.  Submit the metadata URL to OneID. For more information, see [Manage single sign-on configurations](/zuora-platform/security-and-identity/oneid/oneid-settings/manage-single-sign-on-configurations)
