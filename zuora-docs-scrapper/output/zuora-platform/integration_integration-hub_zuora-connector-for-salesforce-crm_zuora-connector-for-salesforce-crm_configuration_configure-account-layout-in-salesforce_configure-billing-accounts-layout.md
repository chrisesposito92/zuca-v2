# Configure billing accounts layout

Learn how to configure the billing accounts layout by adding fields and adjusting settings in the Related List Properties Dialog.

To configure the Billing Accounts layout:

1.  In the Billing Accounts area, click the wrench icon to open the Related List Properties Dialog. ![Billing_Account_Page_Layout.png](/db/organizations/zuora/repositories/prod-sitemap/__sandbox/documents/_MT_Content_Migration/Zuora_Platform/Integration/Integration_Hub/Zuora_360__and_Zuora_360/C_Configuring_Z-Force_360_plus/A_Configure_the_Account_Layout_in_Salesforce/Billing_Account_Page_Layout.png)
2.  Select the following fields in the Available Fields list, then click the right ( Add ) arrow to add them to the Selected Fields list. This adds the fields to the Billing Accounts layout. ![Related_List_Property.png](/db/organizations/zuora/repositories/prod-sitemap/__sandbox/documents/_MT_Content_Migration/Zuora_Platform/Integration/Integration_Hub/Zuora_360__and_Zuora_360/C_Configuring_Z-Force_360_plus/A_Configure_the_Account_Layout_in_Salesforce/Related_List_Property.png)

    -   Bill to Account Balance

    -   Last Invoiced

    -   AutoPay

    -   Status

    -   Payment Term

    -   Credit Card Expiration

    -   Credit Card Number

    -   Credit Card Type


3.  On the Buttons heading, click the + sign.
4.  Uncheck New .
5.  Click OK to save your changes.
