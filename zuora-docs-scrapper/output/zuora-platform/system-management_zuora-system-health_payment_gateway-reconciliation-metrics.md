# Gateway Reconciliation metrics

Descriptions of the Gateway Reconciliation metrics

The following table provides descriptions of the Gateway Reconciliation metrics.

| Metric | Description |
| --- | --- |
| Gateway Reconciliation Overview (Status Breakdown) | The number of reports sent within a specified time range and details of each request, including completed and error runs.When hovering over each time interval bar, you can view the breakdown data of that time range.Completed Sync : The number of successful runs.Error Sync : The number of failed runs. |
| Gateway Reconciliation Failures | The number of failed reports within a specified time range.When hovering over each time interval bar, you can view the breakdown data of that time range. |

-   On the Overview tab, you can view the below details:

    -   Job Number

    -   Period Start

    -   Period End

    -   Gateway Name

    -   Status

    -   Last Attempt

    -   Completed On

-   On the Failures tab, you can view the below details:

    -   Job Number

    -   Attempt

    -   Start Date

    -   End Date

    -   Source

    -   Gateway

    -   Result

    -   Reason
