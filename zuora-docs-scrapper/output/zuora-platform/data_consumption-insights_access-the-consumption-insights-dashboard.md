# Access the Consumption Insights dashboard

dsd

## Overview

To view the dashboard, navigate to Analytics > Consumption Insights in your Zuora tenant. The dashboard opens and displays key subscription metrics as cards.

Note:

This analysis is based solely on usage-based billing data. If your account includes other billing models (such as recurring charges), those are not reflected in these metrics.

The following table provides descriptions of the metrics available in the Overview tab:

| Metrics | Description |
| --- | --- |
| Total Accounts | Total number of active customer accounts for the current month. |
| Last Month's Total Billed Usage | Total billed usage accumulated over the entire previous month. |
| Last Year's Total Billed Usage | Total billed usage accumulated over the entire previous year. |
| Top 5 Usage Growth Accounts | Top 5 accounts ranked by a proprietary 0–100 growth score based on usage trends, velocity, and consistency. |
| Top 5 Usage Risk Accounts | Top 5 accounts ranked by a 0–100 risk score that measures drops in usage, inconsistent activity, and churn risk. |
| Billed Usage Trends | This metric displays the patterns and changes in billed usage over a selected time period.Hover-over the time period to review the metric for the applicable month. |
| Top 5 Performing Usage Products | Top 5 Performing Usage Products |
| Top 5 Performing Usage Customer Accounts | Top 5 Performing Usage Customer Accounts |

Select the applicable country and customer name from the Country and Customer Name drop-down field. By default, the dashboard displays all the accounts that have growth opportunities or at risk. The following table provides descriptions of the metrics available in the Account Insights tab:

| Metrics | Description |
| --- | --- |
| Accounts with Growth Opportunities | Accounts with steady usage growth and engagement, suggesting cross-sell or upsell potential. |
| Accounts at Risk | Accounts with falling or unstable usage patterns that signal disengagement or churn risk. |

On the applicable account, click View in the Actions. The following table provides descriptions of the metrics available for the customer account:

| Metrics | Description |
| --- | --- |
| Risk Score | A 0 to 100 score indicating the likelihood of account churn based on declining usage and behavioral patterns. |
| Growth Score | A 0 to 100 score indicating the potential for account growth based on rising usage and consistent engagement. |
| Last Year's Billed Usage | A usage-based metric showing the total amount billed to the customer over the past year. |
| Last Month's Billed Usage | A usage-based metric reflecting the total amount the customer utilized and was charged for over the past month, based solely on billed consumption |
| Usage Breakdown Analysis by Product Rate Plan | A usage-based metric showing the total amount billed to the customer over the past month. |
| Billed Usage Trends | This metric displays the patterns and changes in billed usage over a selected time period.To compare the analysis with the previous month or year, select the applicable value from the Compare To drop-down field.To review the anomaly or forecast analysis, click Anomaly+Forecast button. |

Note: Select the applicable period of analysis by selecting 3, 6, or 12 months.
