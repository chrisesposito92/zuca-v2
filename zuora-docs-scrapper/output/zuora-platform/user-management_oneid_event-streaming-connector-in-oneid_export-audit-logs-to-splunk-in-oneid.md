# Export Audit Logs to Splunk in OneID

Learn how to export audit logs to Splunk in OneID by configuring event streaming and monitoring.

You must have configured event streaming and monitoring in oneID. For more detailed information, see [Configure event streaming and monitoring in OneID](/zuora-platform/user-management/oneid/event-streaming-connector-in-oneid/configure-event-streaming-and-monitoring-in-oneid)

To export the Audit events to Splunk:

1.  Click Audit Logs from the admin console.
2.  Use filters to select the data to export.
3.  Click the Push to Splunk button.

The audit events are pushed to Splunk.
