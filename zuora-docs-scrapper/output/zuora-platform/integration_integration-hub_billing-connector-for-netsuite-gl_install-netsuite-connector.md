# Install NetSuite Connector

Learn how to install NetSuite Connector

## Overview

This guide describes the initial steps to perform in both NetSuite and Zuora to enable and use NetSuite Connector.

## NetSuite Connector installation process

Perform the following steps to install and configure NetSuite Connector:

1.  [Pre-installation Tasks](/zuora-platform/integration/integration-hub/billing-connector-for-netsuite-gl/install-netsuite-connector/pre-installation-tasks)
2.  [Install the NetSuite Connector Bundle](/zuora-platform/integration/integration-hub/billing-connector-for-netsuite-gl/install-netsuite-connector/install-the-netsuite-connector-bundle)
3.  [Configure NetSuite for NetSuite Connector](/zuora-platform/integration/integration-hub/billing-connector-for-netsuite-gl/install-netsuite-connector/configure-netsuite-for-netsuite-connector)
4.  [Configure NetSuite Forms](/zuora-platform/integration/integration-hub/billing-connector-for-netsuite-gl/install-netsuite-connector/configure-netsuite-forms)
5.  [Create a NetSuite Connector User](/zuora-platform/integration/integration-hub/billing-connector-for-netsuite-gl/install-netsuite-connector/create-a-netsuite-connector-user)
6.  [Reconcile Data Between Zuora and NetSuite](/zuora-platform/integration/integration-hub/billing-connector-for-netsuite-gl/install-netsuite-connector/reconcile-data-between-zuora-and-netsuite)
7.  Configure the NetSuite Connector in Zuora Integration Hub:
    1.  [Create a New Login for NetSuite](/zuora-platform/integration/integration-hub/billing-connector-for-netsuite-gl/install-netsuite-connector/create-a-new-login-for-netsuite)
    2.  [Create a New Instance of Zuora Connector for NetSuite](/zuora-platform/integration/integration-hub/billing-connector-for-netsuite-gl/install-netsuite-connector/create-a-new-instance-of-zuora-connector-for-netsuite)
    3.  [Configure Sync Preferences](/zuora-platform/integration/integration-hub/billing-connector-for-netsuite-gl/install-netsuite-connector/configure-sync-preferences)
    4.  [Configure Advanced Settings](/zuora-platform/integration/integration-hub/billing-connector-for-netsuite-gl/install-netsuite-connector/configure-advanced-settings)
    5.  [Configure Scheduler](/zuora-platform/integration/integration-hub/billing-connector-for-netsuite-gl/install-netsuite-connector/configure-scheduler)
8.  [Migrate Data from NetSuite to Zuora](/zuora-platform/integration/integration-hub/billing-connector-for-netsuite-gl/install-netsuite-connector/migrate-data-from-netsuite-to-zuora)

## Enabling NetSuite Connector

Contact Zuora Technical Support to provision NetSuite Connector.
