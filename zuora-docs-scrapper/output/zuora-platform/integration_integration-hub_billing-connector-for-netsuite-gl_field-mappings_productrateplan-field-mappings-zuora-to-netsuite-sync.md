# ProductRatePlan Field Mappings: Zuora to NetSuite Sync

This reference details the mappings for syncing Product Rate Plan information from Zuora to NetSuite, highlighting potential differences in Source and Destination fields.

This reference describes the mappings of the Product Rate Plan information sync from Zuora to NetSuite.

In some cases, the Source and Destination fields are associated with different objects.

The Destination Field column lists the default names for the fields in NetSuite. Your fields may have been modified from the default values.
