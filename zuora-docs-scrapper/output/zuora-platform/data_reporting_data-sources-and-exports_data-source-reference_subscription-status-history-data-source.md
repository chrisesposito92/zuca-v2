# Subscription Status History data source

Data source to report on the status history information for a given subscription

Use the Subscription Status History data source to report on the status history information for a given subscription.

The following objects are available in the data source.

| Object | Description |
| --- | --- |
| Subscription Status History | Information about the subscription status history.This is the base object for the subscription status history data source export. Contains the following fields:Created By IDCreated DateEnd DateIDSort OrderStart DateStatusSubscription NumberUpdated By IDUpdated Date |
