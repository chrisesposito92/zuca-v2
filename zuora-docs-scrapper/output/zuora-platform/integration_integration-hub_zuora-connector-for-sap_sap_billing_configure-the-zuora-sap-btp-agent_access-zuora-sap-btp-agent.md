# Access Zuora SAP-BTP agent

Learn how to access and configure the Zuora SAP-BTP agent by opening the URL, logging in, and selecting your Customer ID.

1.  Open the Zuora SAP-BTP Agent URL.
2.  Log on to your Zuora SAP-BTP Agent using the username created for your email ID.
3.  Click the Zuora SAP-BTP Agent tile.
4.  Click the Configuration tile.
5.  Select your Customer Id.
