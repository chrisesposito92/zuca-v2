# Sample code for previewing a subscription

This reference provides the sample code for previewing a subscription using the subscribe() call.

## Overview

Starting with version 25.0 of the SOAP API, customers can preview invoices that would be generated by the `subscribe()` call. If you have configured taxation rules in [Zuora Tax](/zuora-billing/set-up-zuora-billing/apply-taxes/overview-of-zuora-tax), the subscription preview will calculate the tax amounts for charges in the subscription.
