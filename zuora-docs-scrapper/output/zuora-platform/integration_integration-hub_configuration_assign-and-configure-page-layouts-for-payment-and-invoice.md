# Assign and Configure Page Layouts for Payment and Invoice

Explains the process of assigning and configuring page layouts for Payment and Invoice objects in Zuora 360.

This article describes how to assign and configure account related page layouts and configure to support the Payment and Invoice Payment objects.

In Zuora 360 version 2.4, multiple page layouts were added to support the Payment and Invoice Payment objects.

In Zuora 360 version 2.5, the page layout was added to support the Refund object.

To use these layouts, you must configure and assign them to user profiles in your Salesforce org.
