# REST API

The Zuora v1 REST API provides access to all features of Zuora Billing, Payments, Platform, and Finance, enabling you to integrate, extend, and automate Zuora across your business systems.

Using the [Zuora v1 REST API](https://developer.zuora.com/v1-api-reference/api/overview/), you can build applications and integrations that interact directly with Zuora to manage transactional data, such as customer accounts, subscriptions, payments, and financial data, through standard HTTP requests and JSON payloads.

## Supported Zuora products

The Zuora v1 REST API supports the following product areas:

-   Zuora Billing: Create and manage product catalogs, customer accounts, orders and subscriptions, and billing documents such as invoices, credit memos, and debit memos.

-   Zuora Payments: Process payments, manage payment methods, handle refunds, and retrieve payment transaction data.

-   Zuora Platform: Access core platform features such as Workflows, Events and Notifications, Data Query, and Custom Objects.

-   Zuora Finance: Retrieve financial data to support revenue recognition, accounting, reporting, and downstream financial integrations.


Zuora also provides REST API support for other products, such as Zuora Revenue and Zephr. For more information, see the [Zuora Revenue API Reference](https://developer.zuora.com/other-api/revenue/overview/) and [Zuora Zephr API Reference](https://developer.zuora.com/zephr-api-reference/zephr-api-introduction/).
