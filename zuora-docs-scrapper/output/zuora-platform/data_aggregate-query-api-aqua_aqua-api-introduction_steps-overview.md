# Steps overview

Learn how to submit, retrieve, and view results of an aggregate query job using the Zuora API.

1.  [Submit an aggregate query job](https://developer.zuora.com/v1-api-reference/api/operation/POST_BatchQueryJob/)
2.  [Retrieve the job results](https://developer.zuora.com/v1-api-reference/api/operation/GET_BatchQueryJob/)
3.  [Retrieve the result file](https://developer.zuora.com/v1-api-reference/api/operation/GET_Files/)
4.  View AQuA requests and results in the [AQuA Job Finder](https://knowledgecenter.zuora.com/Zuora_Platform/Data/Aggregate_Query_API_\(AQuA\)/BE_AQuA_Job_Finder) (optional)
