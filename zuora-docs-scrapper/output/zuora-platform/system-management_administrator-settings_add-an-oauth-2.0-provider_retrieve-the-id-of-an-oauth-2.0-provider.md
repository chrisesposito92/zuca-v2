# Retrieve the ID of an OAuth 2.0 provider

Learn how to retrieve the ID of an OAuth 2.0 provider.

1.  Navigate to .
2.  On the All OAuth 2.0 Providers page, locate the provider whose ID you want to retrieve.
3.  Click the name of the OAuth 2.0 provider. The OAuth 2.0 Provider Details page opens.
    You can find the 32-character ID in the page URL. For example, the ID is `8ad099158032b8ce01803fce92647f63` in the following URL: `https://www.zuora.com/apps/Oauth2.do?method=view&id=8ad099158032b8ce01803fce92647f63`
