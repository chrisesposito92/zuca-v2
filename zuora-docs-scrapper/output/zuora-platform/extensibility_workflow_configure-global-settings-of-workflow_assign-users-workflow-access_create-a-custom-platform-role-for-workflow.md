# Create a custom platform role for Workflow

Learn how to create a custom platform role for Workflow.

1.  Log in to Zuora and navigate to Administration > Manage User Roles
2.  Select Platform from the user role list.
3.  Click Add new role.
4.  In the window that opens, specify a name for the role and select the access permissions this role will have.

    For example, to create a role that can only view the workflows:

    1.  Set the Role Name to "Workflow Viewer"

    2.  Select the checkboxes for the following Platform permissions: UI Access, API Write Access, and Workflow View Access.


5.  Click Save .
