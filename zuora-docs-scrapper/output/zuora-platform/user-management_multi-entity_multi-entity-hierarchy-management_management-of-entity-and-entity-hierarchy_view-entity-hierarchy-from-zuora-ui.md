# View entity hierarchy from Zuora UI

Access and viw entity hierarchy in Zuora UI.

You can view the multi-entity hierarchy from:

-   The Entity Switcher

-   The Manage Entity Hierarchy page


## From the Entity Switcher

Log in as an entity user. You can view the entity hierarchy with all the provisioned entities from the [entity switcher](/db/organizations/zuora/repositories/prod-sitemap/content/documents/external_publications/platform/user_management/multi_entity/overview_multi-entity.ditamap). The entities that you do not have permission to access are grayed out.
