# Enable Consumption Insights by yourself

Learn how to enable the Consumption Insights feature using the Zuora UI by following a series of simple steps.

Perform the following steps to enable Consumption Insights using the Zuora UI:

1.  Click the drop-down next to your user name from the top right corner of the screen.Select Billing > Manage Features.
2.  Find Consumption Insights from the list of available feature. click Enable.
3.  Click Enable.
4.  Click Enable Feature to activate the feature.

The Consumption Insights feature is enabled.
