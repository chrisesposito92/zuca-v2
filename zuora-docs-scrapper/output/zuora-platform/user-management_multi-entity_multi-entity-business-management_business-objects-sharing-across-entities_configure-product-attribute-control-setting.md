# Configure product attribute control setting

Learn to configure the Product Attribute Control Level Setup to manage updates on shared product attributes.

After sharing products with other entities, you might need to update information of the shared products. You can control the updates of each product attribute by configuring the Product Attribute Control Level Setup setting. Note that you must configure this setting before sharing the products.

1.  Log in as an administrator of the global entity.
2.  Navigate to the global entity from the [entity switcher](/zuora-platform/user-management/multi-entity/overview-of-multi-entity).
3.  Click your username at the top right and navigate to Administration > Product Attribute Control Level Setup .
4.  In the Product Attribute Control Level Setup page, select the control level of product attributes:

    -   source entity

    -   target entity


5.  Click save .
6.  Click activate .
