# Import tenant roles

To seamlessly transition the existing user roles into global roles, you can use the Import Tenant Roles feature. You can import all the existing roles created in various tenants along with the permission settings.

To seamlessly transition the existing user roles into global roles, you can use the Import Tenant Roles feature. You can import all the existing roles created in various tenants along with the permission settings.

Note: To avoid role duplication, the import roles feature will not import roles with a similar set of permissions.

1.  Navigate to OneID > Admin Console > User roles and permissions.
2.  Click the Import Tenant Roles button on the top right.
3.  Using the dropdown, select a tenant to import all the existing roles created in that tenant.
4.  Click Convert.
