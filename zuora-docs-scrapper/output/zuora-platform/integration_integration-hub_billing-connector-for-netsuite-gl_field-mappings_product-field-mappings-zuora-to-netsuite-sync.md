# Product Field Mappings: Zuora to NetSuite Sync

This document provides a reference for the mappings of Product information from Zuora to NetSuite, highlighting potential differences in field associations and default field names.

This reference describes the mappings of the Product information sync from Zuora to NetSuite.

In some cases, the Source and Destination fields are associated with different objects.

The Destination column lists the default names for the fields in NetSuite. Your fields may have been modified from the default values.
