# Create a new instance of Zuora Connector for NetSuite

Learn how to create a new instance of Zuora Connector for NetSuite

Each NetSuite connector license includes two permanent connector instances. If additional permanent instances are required, an extra license must be purchased. To use a different sandbox for testing, you can reuse the same connector instance credentials in the new environment. For temporary testing, temporary accounts with a 120-day expiration period can be provisioned.

To sync data between your Zuora tenant and NetSuite, you must create a new instance of Zuora Connector for NetSuite UI. To create a new instance of the connector, follow this procedure.

1.  Follow the instructions in [Install an App From Within Zuora Marketplace](/zuora-platform/integration/integration-hub/get-started-with-integration-hub/install-an-app-you-purchased-in-zuora) to create a new instance of Zuora Connector for NetSuite UI.
2.  Hover over the new instance of the connector and click Launch in the Actions menu.
