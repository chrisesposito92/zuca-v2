# Get started with Integration Hub

Provides an overview of the Zuora Integration Hub

Zuora Integration Hub is a platform that hosts pre-built applications, extensions, interfaces, and tools from technology partners, system integrators, and developers across the globe, making it easier for you to launch subscription businesses based on the Zuora solution.

You can access the Zuora Integration Hub from the Zuora menu.

Currently, the following limitations apply:

-   Integration Hub apps are not available in the EU Data Center. For details, see [Zuora Data Centers](/basics/environments/zuora-data-centers).

-   Integration Hub apps are not available in Production Copy Environment. See [Zuora Billing testing environments](/basics/environments/zuora-billing-testing-environments).


Note:

The Subscriber Portal app is in maintenance mode. Zuora is not actively enhancing the functionalities.
