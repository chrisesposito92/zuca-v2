# Update a function

Learn how to update a function.

1.  Navigate to in the left navigation menu.
2.  Locate the function you want to edit and then click the function name. The function editor opens.
3.  Update the JavaScript code.
4.  Optional: Test the function. For more information, see [Test a function](/zuora-platform/extensibility/custom-logic/function/function-management/test-a-function).
5.  Click Activate or Deactivate to update the function status.
6.  Click Save.
