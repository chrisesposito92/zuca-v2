# Validate transactions in Zuora Revenue after connector run

Learn how to validate the transactions in Zuora Revenue.

After a connector is run, you receive a notification with the details of what transactions were ingested in Zuora Revenue to validate.

1.  From Zuora Revenue, navigate to the File Transaction upload template.
2.  Verify the transactions ingested by the connector by using the Upload File name created by the connector.
3.  Navigate to the Inbound Transaction Stage UI to verify the transactions.
