# Deprecated fields

This topic lists the fields that are deprecated and will not be synced in Zuora 360+.

The following fields are deprecated and will not be synced in Zuora 360+.

| SFDC Field Name | SFDC API Name |
| --- | --- |
| Applied To Charge Number | Zuora__AppliedToChargeNumber__c |
| Product | Zuora__Product__c |
| Product Description | Zuora__ProductDescription__c |
| Product Rate Plan Charge Id | Zuora__ProductRatePlanChargeId__c |
| Revenue Recognition Code | Zuora__RevRecCode__c |
| Revenue Recognition Trigger Condition | Zuora__RevRecTriggerCondition__c |
| Unit Of Measure | Zuora__UOM__c |
