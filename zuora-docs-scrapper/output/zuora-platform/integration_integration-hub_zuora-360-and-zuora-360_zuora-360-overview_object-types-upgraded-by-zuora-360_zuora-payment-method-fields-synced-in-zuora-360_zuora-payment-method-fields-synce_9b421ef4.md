# Zuora Payment Method fields synced in Zuora 360 but not in Zuora 360+

The topic lists fields synced in Zuora 360 but not in Zuora 360+, including Default Payment Method and deprecated Mandate Dates.

The following fields are synced in Zuora 360, but not in Zuora 360+.

| SFDC Field Name | SFDC API Name |
| --- | --- |
| Default Payment Method | Zuora__DefaultPaymentMethod__c |
| Mandate Creation Date (Deprecated) | Zuora__MandateCreationDate__c |
| Mandate Update Date (Deprecated) | Zuora__MandateUpdateDate__c |
| Name | Zuora__Name__c |
