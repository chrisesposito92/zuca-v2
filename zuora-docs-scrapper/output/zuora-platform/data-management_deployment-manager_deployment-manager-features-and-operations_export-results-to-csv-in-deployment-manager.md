# Export results to CSV in Deployment Manager

Learn how to export comparison results to a CSV file in Deployment Manager.

Use the export feature on the compare screen to export the comparison results to a CSV (Comma Separated Values) file. The download CSV feature in the deployment manager has the following benefits:

1.  You can share, save and analyze the results.
2.  You have offline access to compare results.
3.  You can archive and document the results for auditing purposes.
4.  Version control feature available to track and manage the compare results. You can name the CSV files using specific naming conventions to save and organize the files.
