# Revenue Recognition Events Transaction data source

Data source to report on information for a given revenue recognition events transaction

Use the revenue recognition events transaction data source to report on information for a given revenue recognition events transaction. See [Revenue Recognition Events Transaction](/zuora-billing/set-up-zuora-billing/enable-order-to-revenue/overview-of-revenue-recognition-events-transaction) for more information about the booking transaction object.
