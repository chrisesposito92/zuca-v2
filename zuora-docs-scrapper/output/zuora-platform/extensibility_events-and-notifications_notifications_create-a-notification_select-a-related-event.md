# Select a related event

Learn how to select the related event when creating a notification definition.

Complete [Define trigger](/zuora-platform/extensibility/events-and-notifications/notifications/create-a-notification/define-trigger).

1.  Select a related event by using one of the following methods:

    -   Select an existing event from the dropdown list.

    -   Click Create New Event and specify the event details. Available fields vary depending on the trigger type you selected in Step 1: Define trigger. For more information about the description of each field, see [Create a custom event](/zuora-platform/extensibility/events-and-notifications/custom-events/create-a-custom-event) and [Create a custom scheduled event](/zuora-platform/extensibility/events-and-notifications/custom-scheduled-events/create-a-custom-scheduled-event).


2.  Click Next.

See [Set system and custom filters](/zuora-platform/extensibility/events-and-notifications/notifications/create-a-notification/set-system-and-custom-filters).
