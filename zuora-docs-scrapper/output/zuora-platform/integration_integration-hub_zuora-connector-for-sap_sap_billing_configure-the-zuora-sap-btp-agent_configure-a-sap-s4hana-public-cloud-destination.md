# Configure a SAP S/4HANA public cloud destination

Learn how to configure a destination for SAP S/4HANA public cloud, including setting system ID, name, type, and authentication details.

1.  Select the GL Summary Posting capability.
2.  Click the Create a destination button (+).

    -   Enter the ID of the SAP system in the System ID field.

    -   Enter the name of the target SAP system in the Name field.

    -   Enter a description in the Description field.

    -   Enter “HTTP” in the Type field.

    -   Enter “Internet” in the Proxy Type field.


3.  Select Public from the Target System dropdown.
4.  Enter “Basic Authentication” in the Authentication field.
5.  Enter the Communication User name in the User Name field.
6.  Enter the Communication User password in the Password field.
7.  Enter the Communication Arrangement URL in the URL field.
8.  Click Save .

Once saved, the destination will be created on the provider subaccount and you can test connectivity.
