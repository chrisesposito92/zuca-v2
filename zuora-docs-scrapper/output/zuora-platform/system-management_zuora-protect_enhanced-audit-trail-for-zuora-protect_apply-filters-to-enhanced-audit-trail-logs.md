# Apply filters to Enhanced Audit Trail logs

Learn how to apply filters to Enhanced Audit Trail logs to efficiently manage and analyze data by focusing on relevant actions, events, or user activities.

Filtering Enhanced Audit Trail logs is essential for efficiently managing and analyzing large volumes of data. By applying specific filters, you can narrow down the logs to focus on relevant actions, events, or user activities, thereby improving the accuracy and speed of audits.

1.  Once the logs are generated, use the Add New Filter button and select a filter.
2.  Click Apply.
3.  To view the detailed log, click the arrow in the Action column.

To further refine the audit logs, you can use the following filters:

-   Action

-   CreatedvBy ID

-   Event ID

-   Namespace

-   Object Name

-   Object Type

-   Token ID

-   User ID
