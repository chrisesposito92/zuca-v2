# Revenue accounting code update

This reference document list all fields in the Update Revenue Accounting Code data dictionary.

Note: Fields with an asterisk mark indicate mandatory fields.

| Name | Description | Value |
| --- | --- | --- |
| Adjustment Liability Account* | The name of the account where the Account Type is "Adjustment Liability". | string <= 100 characters |
| Adjustment Revenue Account* | The name of the account where the Account Type is "Adjustment Revenue" | string <= 100 characters |
| Contract Asset Account* | The name of the account where the Account Type is "Contract Asset". | string <= 100 characters |
| Contract Liability Account* | The name of the account where the Account Type is "Contract Liability". | string <= 100 characters |
| Product RatePlan ChargeId* | The ID of your product rate plan charge. | string |
| Recognized Revenue Account* | The name of the account where the Account Type is "Recognized Revenue". | string <= 100 characters |
| Unbilled Receivables Account* | The name of the account where the Account Type is "Unbilled Receivables". | string <= 100 characters |
