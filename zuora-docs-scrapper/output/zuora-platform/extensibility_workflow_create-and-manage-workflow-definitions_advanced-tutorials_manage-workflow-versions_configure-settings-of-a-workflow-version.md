# Configure settings of a workflow version

High-level introduction about configuring settings of a workflow version.

You can configure the settings of a workflow version in the Settings tab. Available settings include execution priority, triggers, notifications, cleanup policy, security, and input parameters. For more information, see [Settings of a workflow version](/zuora-platform/extensibility/workflow/create-and-manage-workflow-definitions/get-started-with-workflow/configure-the-settings-of-a-workflow-definition/workflow-specific-settings).
