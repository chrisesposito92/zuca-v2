# Install Zuora 360 package

Learn how to install the Zuora 360 package in your Salesforce org by following these step-by-step instructions.

To install the Zuora Salesforce Connector package :

1.  Navigate to the URL provided by Zuora and log in to your Salesforce org.
2.  On the Package Installation Details page, review the information and click Continue .
3.  When prompted to approve third-party access, select Yes to grant access and click Continue .
4.  On the Approve Package API Access page, click Next .
5.  On the Choose Security Level page, select Select Security Settings to set the level of access that you want to provide for user groups. If you select Grant access to admins only , you must update your security settings in Salesforce to grant permissions to other users to be able to access Zuora Connector for Salesforce CRM.

7.  Click Next .
8.  On the Install Package page, click Install . The installation process starts, and you will receive an automated email when the process completes.
9.  Select Zuora 360 Sync from the menu at the top right-hand corner to change to the Zuora 360 view.
