# Field descriptions of Refund

This reference provides the description of the fields of the Refund object.

All field names are case sensitive. Check enumerated values in descriptions to confirm capitalization and spacing.

| AccountId | optional | Create Query Delete Filter | The ID of the account associated with this refund. Specify a value for this field only if you're creating an electronic non-referenced refund.
