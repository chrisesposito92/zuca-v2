# Edit the unprovisioned sub-entity information from Zuora UI

Learn how to edit unprovisioned sub-entity information in Zuora UI.

Only the global entity administrators can edit unprovisioned sub-entities information.

1.  Log in as an administrator of the global entity.
2.  Navigate to the global entity from the [entity switcher](https://knowledgecenter.zuora.com/Zuora_Platform/User_Management/Multi-entity/A_Overview_of_Multi-entity#Entity_Switcher).
3.  Click your user name at the top right and navigate to Administration > Manage Entity Hierarchy.
4.  On the Manage Hierarchy of Entities page, click Edit next to the entity that you want to edit.
5.  In the Edit Entity dialog, edit the entity information.
6.  Click save.
