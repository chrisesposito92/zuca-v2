# Create value mappings

Create value mappings using the Integration Hub in the Marketplace.

1.  Navigate to Marketplace > Integration Hub .
2.  Search and select the SAP GL (Billing).
3.  Click Field Mapping .
4.  Click the Edit Mappings button.
5.  Click the Open the Expressions Wizard icon.
6.  In the Expressions Wizard, select Value Mapping .
7.  In the Expressions Wizard, select a source field and create your value mapping.
8.  Click the Submit button.
9.  Click Save Mappings .
