# Add custom fields to custom object definitions

Learn how to add a custom field to a custom object definition.

1.  Navigate to in the left navigation.
2.  Click the name of the custom object definition to which you want to add the custom field. The custom object detail page opens.
3.  Click the Create icon ![Create icon](https://zuora.deploy.heretto.com/v4/deployments/QPAZk6lsgXwvotedNERE/object/30244a20-151c-4f5f-b60c-96d79c2aadd2?jwt=eyJ0eXAiOiJKV1QiLCJhbGciOiJIUzI1NiJ9.eyJodHRwczovL2pvcnNlay5jb20vZXpkX29yZ2FuaXphdGlvbiI6Inp1b3JhIiwiaHR0cHM6Ly9qb3JzZWsuY29tL2V6ZC9vYmplY3RfdXVpZCI6IjMwMjQ0YTIwLTE1MWMtNGY1Zi1iNjBjLTk2ZDc5YzJhYWRkMiIsImV4cCI6MTc2NjY0MDE4MiwianRpIjoiMmYzOTcwMWQ2ZGJkNGRmODgwODlhMmIzNzk2OGJmMWYiLCJodHRwczovL2pvcnNlay5jb20vZXpkX2ZpbGVzZXQiOiJWZHdCeUJjM0lBa01wRU9LSFdxZCJ9.76L2qS4BDzecTPlgJgw8Tm2aT6KTm8nGc9Jj1koX_f8). The Add New Custom Field dialogue opens.

    -   A custom object can contain at least 1 and at most 50 custom fields

    -   All custom fields in a custom object are indexed.


4.  Enter the information of the custom field. The Name and API Name are required. Note that you can only add a required field, a relationship type field, or a unique constraint to an existing custom object with no records.
5.  Click Save and Close to save the new field.
