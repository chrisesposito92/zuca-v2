# Configure an Event Store as a target

Learn how to configure an Event Store as a target for a custom meter by selecting and mapping event fields.

To create a meter with an Event Store as the target:

1.  Create a custom meter.
2.  Select Event Store as the target.
3.  In the Event Store settings page, select an existing event store. You can also create a new event store based on an event definition.
4.  Map the event fields from the event store event definition. to target fields. The `startDateTime` field is required for the event store.
5.  Click Save.

The event store os configured as a target.
