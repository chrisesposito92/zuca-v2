# Configure an Event Store as a target

Learn how to configure an Event Store as a target for a custom meter by selecting and mapping event fields.

1.  Navigate to
2.  Create a Custom meter.
3.  Select Event Store as the target.
4.  In the Event Store settings page, select an existing event store. You can also create a new event store based on an event definition.
5.  Map the event fields from the event store event definition. to target fields. The `startDateTime` field is required for the event store.
6.  Click Save.
