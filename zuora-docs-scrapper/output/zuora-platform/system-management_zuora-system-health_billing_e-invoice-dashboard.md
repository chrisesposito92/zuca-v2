# E-Invoice dashboard

Collects and displays data about e-invoicing usage, failure, and transactions being processed in near real time

Zuora System Health dashboard for [E-Invoicing](/zuora-billing/bill-your-customer/leverage-advanced-capabilities/e-invoicing/e-invoicing-overview) (E-Invoice dashboard) collects and displays data about e-invoicing usage, failure, and transactions being processed in near real time. It enables you to view data about all the e-invoicing transactions in your Zuora tenant within the last 30 days. You can leverage this dashboard to identify and troubleshoot e-invoicing issues.

For more information about the System Health feature, see [Zuora System Health](/zuora-platform/system-management/zuora-system-health).

## Prerequisites

For more information, see [Zuora System Health prerequisites](/zuora-platform/system-management/zuora-system-health#concept-00qhbv71__system_health_prerequisites).

## Access the E-Invoice dashboard

To access the E-Invoice dashboard, navigate to Administration > System Health in the left navigation menu and click E-Invoice in the Billing section.

For more information on the E-Invoice dashboard, refer to these topics:

-   [View data in the E-Invoice dashboard](/zuora-platform/system-management/zuora-system-health/billing/e-invoice-dashboard/view-data-in-the-e-invoice-dashboard)
