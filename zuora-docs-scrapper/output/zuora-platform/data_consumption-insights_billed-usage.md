# Billed Usage

Review and analyze billed usage patterns over a selected time period using the Billed Usage tab.

This metric displays the patterns and changes in billed usage over a selected time period. Perform the following tasks to review the metrics in the Billed Usage tab:

1.  Select the applicable period of analysis by selecting 3, 6, or 12 months.
2.  Select product or customer from the Analysed by drop-down field.
3.  Search or select the applicable product or customer account from the field next to the Analysed by drop-down field.
