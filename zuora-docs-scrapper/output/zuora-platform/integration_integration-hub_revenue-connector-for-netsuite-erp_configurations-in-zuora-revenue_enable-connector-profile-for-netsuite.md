# Enable connector profile for NetSuite

Learn how to enable the Integration Hub Marketplace in the Zuora Revenue menu to be able to run the NetSuite Inbound connector.

1.  Navigate to .
2.  Enable the ENABLE\_ZR\_CONNECTORS profile in Zuora Revenue as follows.
    1.  Type ENABLE\_ZR\_CONNECTORS in the Profile Name field.
    2.  Select System as the Profile Level.
    3.  Select Text as the Profile Type.
    4.  Enter NS\_IB in the System Level Value field.
