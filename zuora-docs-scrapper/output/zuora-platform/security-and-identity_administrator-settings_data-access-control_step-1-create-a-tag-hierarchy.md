# Step 1: Create a tag hierarchy

Learn how to create a tag hierarchy as part of the Data Access Control implementation process.

1.  Navigate to .
2.  Enter the following hierarchy information.

    | Field | Description |
    | --- | --- |
    | Label | Name of the hierarchy. |
    | API Name | This is an internal reference name used for integrations, This field should start with a character and consist of only numbers and characters in the English language. The name should end with _h. Note that changing the API name can affect existing integrations. |
    | Active | Check this box if you want the Data Access Control hierarchy enabled. |
    | Description | Description of the hierarchy. |

3.  Click save.
4.  Create the tags.
5.  Click save.

After you have entered all your tags, your hierarchy definition will be displayed.
