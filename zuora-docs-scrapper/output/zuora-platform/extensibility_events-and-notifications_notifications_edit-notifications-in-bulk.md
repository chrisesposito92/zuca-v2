# Edit notifications in bulk

Learn how to edit notifications in bulk in the Zuora UI.

1.  Navigate to in the left navigation menu.
2.  Click Edit at the upper right of the notification definition list.
3.  Update the following notification settings of the notification definitions you want to update:

    -   Status

    -   Name

    -   Communication Profile

    -   Email status

    -   Email template

    -   Callout status

    -   Callout template


4.  Click Save at the upper right of the notification definition list.
