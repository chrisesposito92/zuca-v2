# Configure a SAP destination for SAP-BTP agent

Learn how to configure an SAP destination by selecting capabilities, entering system details, and setting authentication methods.

1.  Select the GL Summary Posting capability.
2.  Click the Create a destination button (+).
3.  Enter the ID of the SAP system in the System ID field.
4.  Enter the name of the target SAP system in the Name field.
5.  Enter a description in the Description field.
6.  Enter “HTTP” in the Type field.
7.  Enter “Internet” in the Proxy Type field.
8.  Select Public from the Target System dropdown for S/4HANA Public Cloud systems .
9.  Select Private from the Target System dropdown for ECC or S/4HANA On-Premise systems.
10.  Enter “Basic Authentication” in the Authentication field.
11.  Enter the Communication User name in the User Name field.
12.  Enter the Communication User password in the Password field.
13.  Enter the Communication Arrangement URL in the URL field.
14.  Click Save.

Once saved, the destination will be created on the provider subaccount and you can test connectivity.
