# Prepare SAP S/4HANA private cloud

Learn how to deploy the Zuora Edge Component to integrate Zuora’s Integration Hub with SAP ECC, S/4HANA on-premise, and Private Cloud systems.

The Zuora Edge Component assists you with the integration of Zuora’s Integration Hub platform with SAP ECC, S/4HANA on-premise customers, and Private Cloud systems. The following steps describe how to deploy the Zuora Edge Component.

1.  Log in to the applicable system.
2.  Type cg3z in the T-code field.
3.  Press Enter.
4.  Select the applicable file from your downloads. ile from you
5.  Select the Target file.
6.  Click the Upload button.
7.  Click the Allow button.
8.  Select the Data file.
9.  Click the Upload button.
10.  Click the Allow button.
11.  Enter stms in the T-code field in a new SAP GUI window.
12.  Press Enter.
13.  Select Import Overview.
14.  Double-click on the applicable system to access it.
15.  Navigate to Extras \> Other Requests \> Add .
16.  Select the transport from the list or paste the transport number into the Transport Request field and press Enter.
17.  Click Yes to import the queue.
18.  Select the transport from the list.
19.  Select the applicable client.
20.  Select all the options from the Import Options section.
21.  Click Next.
22.  Click Yes.
23.  Double-click the RC to verify if the transport is imported.
