# Manage role-based access

Learn how to manage role-based access in Zuora by following these steps.

Perform the following steps to manage role-based access:

1.  Log into your Zuora tenant as an administrator.
2.  Click the drop-down list next to your user name and select Administration.
3.  Click Manage User Roles.
4.  Select Reporting from View Role List of drop-down field.
5.  Click View on the applicable role.
6.  In the Analytics section under the Reporting Permissions section, select the View Consumption Insights check box.

The user's role is changed as configured.
