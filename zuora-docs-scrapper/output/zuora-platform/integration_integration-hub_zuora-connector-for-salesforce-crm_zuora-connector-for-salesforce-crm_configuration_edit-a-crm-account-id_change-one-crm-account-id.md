# Change one CRM account ID

Learn how to change the CRM Account ID in a customer account using Zuora.

To change the CRM Account ID in the customer account:

1.  In Zuora, navigate to .
2.  Click the customer account you want to edit.
3.  Click edit .
4.  Update the CRM Account ID for the applicable customer account. See Find the Salesforce Account IDs for retrieving the CRM Account ID.
5.  Click Save .
6.  Re-synchronize the Accounts & Related Objects data as described in Synchronize Data from Zuora .
