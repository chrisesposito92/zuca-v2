# Modify tenant names on the Zuora OneID home page

Learn how to modify tenant names on the Zuora OneID home page as an administrator.

To update tenant names as an administrator:

1.  In Zuora OneID, navigate to Tenants .
2.  Click Edit next to the tenant name that needs to be changed.
3.  Update the tenant name as desired and click Save.

To verify the updated tenant name, go to the MyApps page and check the changes in the Zuora products, such as Billing or Revenue .
