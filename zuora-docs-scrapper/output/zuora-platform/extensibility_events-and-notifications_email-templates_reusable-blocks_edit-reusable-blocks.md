# Edit reusable blocks

Learn how to update reusable blocks. Updates made to a reusable block are automatically applied across all linked email templates.

1.  Navigate to C in the left navigation menu. The Setup Profiles, Notifications and Email Templates page opens.
2.  Click the Email Templates tab.
3.  Click the Reusable Blocks sub-tab.
4.  Locate the reusable block you want to edit and click Edit in the last column of the table. The Edit Reusable Block dialog opens.
5.  Modify the following settings as needed:

    -   Name

    -   Block Number

    -   Category

    -   Tag(s)

    -   Active

    -   Block content


6.  Click Save.
