# Run a report with data warehouse from the report list

Learn how to run a report with data warehouse from the report list

To run a report with data warehouse from the report list, perform this procedure.

1.  In the left navigation section, navigate to Reporting > Reporting.
2.  Click the folder that contains the report you want to update. For more information about folders in reporting, see [Folders and sharing](/zuora-platform/data/reporting/use-reporting/folders-and-sharing).
3.  Click the triangle icon ![triangle-white.png](https://zuora.deploy.heretto.com/v4/deployments/QPAZk6lsgXwvotedNERE/object/e6353483-28dc-4e1c-a4c1-462b346347fb?jwt=eyJ0eXAiOiJKV1QiLCJhbGciOiJIUzI1NiJ9.eyJodHRwczovL2pvcnNlay5jb20vZXpkX29yZ2FuaXphdGlvbiI6Inp1b3JhIiwiaHR0cHM6Ly9qb3JzZWsuY29tL2V6ZC9vYmplY3RfdXVpZCI6ImU2MzUzNDgzLTI4ZGMtNGUxYy1hNGMxLTQ2MmIzNDYzNDdmYiIsImV4cCI6MTc2NjY4ODEzMSwianRpIjoiMmIyMDk2NmI4NDI5NDU1OTlkOGFkZDI2YzJmYmMzYzMiLCJodHRwczovL2pvcnNlay5jb20vZXpkX2ZpbGVzZXQiOiJWZHdCeUJjM0lBa01wRU9LSFdxZCJ9.3c4DxMvLsZk4xc_NFWzjEk_UZOR_CJL5HDvvEpYiHZs) to the right of the report you want to run. The drop-down list opens.
4.  From the drop-down list, select the option for the data warehouse you want to run this report with. For example, if you want to run the report with Zuora Warehouse \*, select Run Detail Report with Warehouse or Run Summary Report with Warehouse as needed.

    \*Applicable only to reports based on saved data query.
