# Edit a custom event or a custom scheduled event

Learn how to edit a custom event or a custom scheduled event in the Zuora UI.

1.  Navigate to in the left navigation menu.
2.  Click the Custom Events tab.
3.  In the Custom Events tab, locate the event you want to edit, and then click Edit of that event.
4.  Edit the desired fields of the event. The following fields are not editable:

    -   Custom event: API Name and Base Object.

    -   Custom scheduled event: API Name, Base Object, and Base Field.


5.  Click Save.
