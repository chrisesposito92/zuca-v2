# Edit an attachment

Learn how to update the file name and description of an attachment associated with a custom account, subscription, invoice, credit memo, or debit memo object.

Note:

You cannot edit the content in an attachment file. To edit the file content, delete the attachment and upload another updated attachment.

1.  Open the detail page of the target object, scroll down to the Attachments section.
2.  Click the title of the attachment you want to edit. The file name and description fields become editable.
3.  Make the changes and click Save.
