# Accept the user access request

Learn how to accept a user access request in the desired entity.

You must accept the user access request in the entity that the user wants to access.

1.  Log in as the administrator of the entity.
2.  Navigate to the entity from the [entity switcher](/zuora-platform/user-management/multi-entity/overview-of-multi-entity#concept-pwmj5n5e__section-243).
3.  Click your username at the top right and navigate to Administrator > Manage Users.
4.  On the All Users page, click the user first name or last name.
5.  On the user page, click Grant Access.
6.  Click Yes on the pop-up window.

    To accept a user access request from the REST API, see [Accept user access](https://www.zuora.com/developer/api-references/older-api/operation/PUT_AcceptUserAccess?_gl=1*1kb4wcn*_gcl_au*MzAzMjUyNDQ2LjE3NTU3Nzk0MDc.*_ga*NTY4ODk1NTI2LjE3NTk3NTA4NjE.*_ga_MY8CQ650DH*czE3NTk5OTMyODckbzI2JGcxJHQxNzU5OTk0NDk5JGo1MyRsMCRoMTg5MDc5NDM5Nw..).
