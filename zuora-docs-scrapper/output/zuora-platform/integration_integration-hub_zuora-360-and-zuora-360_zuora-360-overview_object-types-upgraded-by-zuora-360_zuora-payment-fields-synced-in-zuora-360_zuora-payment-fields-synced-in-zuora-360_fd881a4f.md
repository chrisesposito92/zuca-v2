# Zuora Payment fields synced in Zuora 360 but not in Zuora 360+

This topic lists the fields that are synced in Zuora 360 but not in Zuora 360+, including Effective Date, External ID, Invoice, Invoice Number, and Payment Method Name.

The following fields are synced in Zuora 360, but not in Zuora 360+.

| SFDC Field Name | SFDC API Name |
| --- | --- |
| Effective Date (Deprecated) | Zuora__EffectiveDate__c |
| External ID(Deprecated) | Zuora__External_Id__c |
| Invoice | Zuora__Invoice__c |
| Invoice Number | Zuora__InvoiceNumber__c |
| Payment Method Name | Zuora__PaymentMethod__c |
