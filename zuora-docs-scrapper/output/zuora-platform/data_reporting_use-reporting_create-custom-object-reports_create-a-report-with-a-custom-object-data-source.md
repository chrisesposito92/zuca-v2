# Create a report with a custom object data source

Learn how to create a report with a custom object data source

To create a report with a custom object data source, perform this procedure.

1.  Navigate to Reporting > Reporting in the left navigation menu.
2.  Click +Create New Report . The Datasources page opens.
3.  On the Datasources page, select the custom object you want to create the report for.
4.  Build and preview the report. For more information, see [Build and preview the report](/zuora-platform/data/reporting/use-reporting/create-a-summary-report#concept-retjrr5i__title-614). When building the report, you can use fields from the source custom object and fields from other objects that are directly related to the source custom object.
5.  Run and view the report. For more information, see [Run and view the report](/zuora-platform/data/reporting/use-reporting/create-a-summary-report#concept-retjrr5i__title-1469).
