# Set up API user in Revenue

Learn how to set up the role privileges and permissions for an API user in Zuora Revenue.

1.  Navigate to .
2.  Create a role and enter a name API Role.
3.  Click the Role Privileges tab.
4.  Expand the Privilege Data interface.
5.  Select Full Access for the privilege named Web Services Inbound.
6.  Select Full Access for the privilege named Web Services Outbound.
7.  Save your entries.
8.  Navigate to .
9.  Create an API user.
10.  From the Role Assignments tab, assign the API role you created.
11.  To change the password, navigate to .
