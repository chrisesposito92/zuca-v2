# Create a global user report in OneID

1.  Log in as administrator in the OneID account.
2.  Navigate to the Users menu and click the download button from the top-right corner of the page.
3.  In Export User Report, enter a Report Name.
4.  Set the Report Type to Global Level and click Export.
5.  You can download the report when the Export Status changes from Pending to Complete.
