# Enable Zuora default email server

Learn how to enable Zuora default email server to send email notifications.

1.  Navigate to in the left navigation menu.
2.  Click the Settings tab.
3.  In the Email Settings section, click the Basic SMTP tab.
4.  In the Basic SMTP tab, select the Select Default Zuora E-Mail Server checkbox.
