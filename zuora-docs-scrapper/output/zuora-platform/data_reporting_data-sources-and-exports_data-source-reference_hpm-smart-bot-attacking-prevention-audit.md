# HPM Smart Bot Attacking Prevention Audit

data source to export the data of reCAPTCHA security setting changes made by the HPM Smart Bot Attacking Prevention feature

Use this data source to export the data of reCAPTCHA security setting changes made by the HPM Smart Bot Attacking Prevention feature. Each row in the export represents one change record.

For details about how to generate a data source export, see Generate a data source export .
