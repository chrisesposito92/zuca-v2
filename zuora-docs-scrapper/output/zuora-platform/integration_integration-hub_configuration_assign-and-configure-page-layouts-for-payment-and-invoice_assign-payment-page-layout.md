# Assign Payment Page Layout

Learn how to customize and assign the latest default Payment page layout to user profiles.

As the admin user, you can customize the latest default Payment page layout and assign the page layout to the user profiles to access the page.

The new page layouts are used by the payment and invoice payment sync process. This sync process is a controlled release feature.

Submit a request at Zuora Global Support to enable this feature or service.

To assign the Payment page layouts to user profiles:

1.  Navigate to Setup > Build > Create > Objects .
2.  Select Payment . Scroll to the Page Layouts section and click Page Layout Assignment .
3.  Click Edit Assignment , and select the user profiles that you want to assign the page layouts to.
4.  In the Page Layout to Use list, select the latest version of Payment Z-Force Layout , e.g. Payment Z-Force Layout v2.7.
5.  Click Save .
