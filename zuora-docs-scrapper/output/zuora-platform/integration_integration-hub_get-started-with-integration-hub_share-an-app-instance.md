# Share an app instance

Learn how to share an app instance you created to enable other Zuora Connect users to access the same app configuration.

You should perform this procedure in Zuora Connect, and not in your Zuora tenant. You can only manage app instances from within Zuora Connect if you have installed more than one instance of the app.

1.  Log into [Zuora Connect](https://connect.zuora.com/).
2.  Navigate to . The Tenant Logins page opens.
3.  Locate the Zuora Connect login for your Zuora tenant, then hover and click Apps.

    ![Tenant example](https://zuora.deploy.heretto.com/v4/deployments/QPAZk6lsgXwvotedNERE/object/43fb2f7c-cbf5-4886-9c7a-6a03301a8518?jwt=eyJ0eXAiOiJKV1QiLCJhbGciOiJIUzI1NiJ9.eyJodHRwczovL2pvcnNlay5jb20vZXpkX29yZ2FuaXphdGlvbiI6Inp1b3JhIiwiaHR0cHM6Ly9qb3JzZWsuY29tL2V6ZC9vYmplY3RfdXVpZCI6IjQzZmIyZjdjLWNiZjUtNDg4Ni05YzdhLTZhMDMzMDFhODUxOCIsImV4cCI6MTc2NjY0MTU2NCwianRpIjoiZjkxYzBjNzYyOTI2NDNjZTlkY2VhOTJkM2NlNjMxOGYiLCJodHRwczovL2pvcnNlay5jb20vZXpkX2ZpbGVzZXQiOiJWZHdCeUJjM0lBa01wRU9LSFdxZCJ9.td_28MpjEB0zUzLzmY4Xqf3E3oF6JFw-z31Ah0hmYe4) An overview page for your Zuora tenant opens.

4.  Click the Apps tab.

    Zuora Connect displays the available app instances for your Zuora tenant.

5.  Hover over the app instance that you want to share, then click Share Task.
6.  On the Authorized Instance Users dialog box, enter the details of the Zuora Connect users that you want to share the app instance with, then click Save Permissions.
