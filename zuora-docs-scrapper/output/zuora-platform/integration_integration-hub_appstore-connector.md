# Appstore Connector

The Appstore Connector is in the Early Availability phase, allowing early adopters to provide feedback before a full release. It includes key capabilities, supported app stores, and setup instructions for integrating Apple, Google, and Roku products with Zuora.

Note:

This feature is in the Early Availability phase. We are actively soliciting feedback from a small set of early adopters before releasing it to all customers. To join this early availability program, submit a request at Zuora Global Support .
