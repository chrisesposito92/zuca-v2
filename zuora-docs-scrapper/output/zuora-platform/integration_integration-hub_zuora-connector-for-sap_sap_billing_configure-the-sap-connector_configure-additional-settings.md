# Configure additional settings

Learn how to configure additional settings in the Integration Hub for SAP GL (Billing) to ensure successful transfer operations.

1.  Navigate to Marketplace > Integration Hub .
2.  Search and select the SAP GL (Billing).
3.  Select the Settings tab.
4.  Find the Additional Settings section.
5.  Check the box to retry any failed transfer operations.
6.  Click Save .
