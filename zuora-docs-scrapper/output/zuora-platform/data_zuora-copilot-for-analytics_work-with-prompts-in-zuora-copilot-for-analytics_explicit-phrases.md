# Explicit phrases

Explicit phrases ensure the Copilot returns data exactly as specified, without additional context.

For explicit phrases, the Copilot returns data exactly as specified in the prompt without the need for any additional context.

The following table lists a few examples of explicit phrases used in a search prompt and their expected results.

| Explicit phrases used in a prompt | Results |
| --- | --- |
| January to December 2023 | Returns data from January to December 2023. |
| May to June 2023 | Returns data from May to June 2023. |
