# Overview of Zuora Connector for SQL Server

Introduction to the Zuora Connector for SQL Server

Enable your organization to access Zuora data directly through SQL Server. This integration simplifies the setup process, eliminating common technical barriers related to data extracts, integrations, and schema updates.

Once configured, your data is automatically refreshed daily within SQL Server, granting your teams immediate access to extensive subscription operational and analytical data from Zuora Billing. This seamless access enables your teams to swiftly make informed decisions based on valuable insights.

Note:

The Zuora Connector for Data Warehouses feature is in the Early availability phase. This is a paid add-on. If you are interested in joining our early availability program, please reach out to your Zuora Representative for further details.

## How the Zuora Connector for SQL Server works

The Zuora Connector for SQL Server facilitates a direct connection between your Zuora account and your SQL Server environment. This connection is established through a one-time setup process during which you configure your MySQL credentials.

Once configured, the connector automates the synchronization of your Zuora data into your specified SQL Server environment on a daily basis. Tailored to seamlessly integrate with SQL Server, this connector enhances your data storage and analytics capabilities within your infrastructure.
