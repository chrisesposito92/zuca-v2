# Identify legacy custom fields on a standard object

Learn how to distinct legacy custom fields from advanced custom fields on standard objects.

1.  Navigate to in the left navigation menu.
2.  In the Objects tab, locate the standard object you want to view and click the object name in the Label column to open the object detail page. Zuora lists all custom fields in the Custom Fields tab on the object detail page. If the value in the Legacy column is true , the custom field is a legacy custom field. Otherwise, it is an advanced custom field.
