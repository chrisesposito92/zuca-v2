# Define trigger

Learn how to select the trigger type when creating a notification definition.

1.  Open the notification editor:
    1.  Navigate to in the left navigation menu.
    2.  In the NotificationCs tab, click \+ Add New Notification.
2.  Select one of the following trigger type for the notification:
    | Option | Description |
    | --- | --- |
    | On An Event Occurrence | The notification will be triggered when a specific event occurs, such as Invoice Posted, Order Action Processed, or any custom event. |
    | On A Daily Schedule | The notification will be triggered when the date matches the criteria of a daily-scheduled event, such as Invoice Due, Payment Method Expiration, or any custom scheduled event. |


See [Select a related event](/zuora-platform/extensibility/events-and-notifications/notifications/create-a-notification/select-a-related-event).
