# Add destination for AWS S3

Learn how to add destination for Zuora Connector for AWS S3

After completing the prerequisites, provide the Zuora tenant ID for which the AWS S3 connector needs to be set up. Once the tenant ID is shared, the Zuora team will enable the AWS S3 connector configuration UI for your tenant, allowing you to complete the setup using a self-serve flow.

Ensure the required permissions are enabled.

1.  Go to Marketplace > Integration Hub UI.
2.  Search for Zuora Connector for AWS S3.
3.  Open the connector. You will see three tabs: Overview, Setup, and Status.
4.  In the Setup tab, click Configure, enter your destination details, and select the objects to sync.
5.  Click Test Connection.
6.  If the test is successful, click Save Destination to trigger onboarding.

Refer to [Post-onboarding flow for AWS S3](/zuora-platform/integration/integration-hub/zuora-connectors-for-data-warehouses/zuora-connector-for-aws-s3/set-up-zuora-connector-for-aws-s3/post-onboarding-flow-for-aws-s3).
