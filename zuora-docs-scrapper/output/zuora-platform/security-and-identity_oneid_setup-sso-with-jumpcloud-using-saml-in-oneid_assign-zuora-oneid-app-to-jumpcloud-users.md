# Assign Zuora OneID app to JumpCloud users

Assign the Zuora OneID app to JumpCloud users by creating user groups, selecting users, and granting application access.

To create a new user group that requires access to the application

1.  Log in to the JumpCloud Administrator Portal.
2.  Go to User Groups .
3.  Click (+), then select Create a Group of Users.
4.  Name the new user group, then select Users.
5.  Select the users you would like to add to the group.
6.  Go to Applications.
