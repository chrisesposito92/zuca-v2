# Create a customer account and add subscriptions

Learn how to create a customer account for a single org unit and add subscriptions.

Each customer account that you create can belong to only one org unit. If you wish to have the same customer account details for multiple org units, you must create a separate customer account with the same details in each org unit.

1.  Navigate to Customers > Customer Accounts in the left-hand navigation section and click Add New Account.
2.  In the Basic Information and Contacts section, use the Org Picker to select the org unit that the customer account must belong to
3.  Enter the other Basic Information and Contacts for the account to complete customer account creation.
