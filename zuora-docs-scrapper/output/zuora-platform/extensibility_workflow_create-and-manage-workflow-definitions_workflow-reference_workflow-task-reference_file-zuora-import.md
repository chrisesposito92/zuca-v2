# File: Zuora Import

This reference explains the File: Zuora Import task.

The Zuora Import task uploads content especially when you have a large amount of records.

## Task settings

-   File: Select a file to be uploaded.

-   Action Type: Select one of the following types for import.

    -   Update Accounting Code

    -   Create Revenue Schedule

    -   Update Revenue Schedule

    -   Delete Revenue Schedule

    -   Import FX Rate

    -   MPU
