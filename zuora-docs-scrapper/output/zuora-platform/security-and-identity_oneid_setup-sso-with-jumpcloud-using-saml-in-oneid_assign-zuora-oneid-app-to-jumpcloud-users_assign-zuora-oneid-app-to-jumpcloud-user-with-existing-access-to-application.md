# Assign Zuora OneID app to JumpCloud user with existing access to application

Assign the Zuora OneID app to a JumpCloud user group with existing access to the application.

To provide an existing user group with access to the application

1.  Select the group of users you would like to grant access to the application.
2.  In the Groups Details panel, select Applications.
3.  Select the application you would like the user group to have access to.
4.  Click Save.
