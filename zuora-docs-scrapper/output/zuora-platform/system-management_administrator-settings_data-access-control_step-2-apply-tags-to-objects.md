# Step 2: Apply tags to objects

Learn how to apply tags to objects as part of the Data Access Control implementation process.

A tag hierarchy has been created. See [Create a tag hierarchy](/zuora-platform/system-management/administrator-settings/data-access-control/step-1-create-a-tag-hierarchy) for details.

1.  Navigate to .
2.  Select the account for which you want to apply the tag.
3.  Click edit. The Data Access Control tag hierarchy you created will display as a drop-down list.
4.  Select the value for which you want to tag the object.
5.  Click save.

[Apply tags to users](/zuora-platform/system-management/administrator-settings/data-access-control/step-3-apply-tags-to-users).
