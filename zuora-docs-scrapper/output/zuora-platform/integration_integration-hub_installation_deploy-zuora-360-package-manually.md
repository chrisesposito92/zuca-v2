# Deploy Zuora 360 package manually

Guides you through manually deploying the Zuora 360 package when the installation process is stuck in "Processing" status.

If you receive an email notifying you of a successful installation, but the installation process for the package continues to show the "Processing" status, you can deploy the Zuora 360 package manually.

1.  Navigate to user name \> Setup \> Installed Packages .
2.  Verify that the Installed Packages list includes the correct version of the Zuora 360.
3.  Click the Zuora CPQ 360 package.
4.  On the Package Details page, click Deploy .
