# Manage User Access

Learn how administrators can manage user accounts and activate features in Zuora environments.

Administrators can manage Zuora user accounts in your tenant. The self-service interface for Feature Management allows you to activate Early Adopter, Beta, and controlled Zuora features in both your sandbox and production environments.
