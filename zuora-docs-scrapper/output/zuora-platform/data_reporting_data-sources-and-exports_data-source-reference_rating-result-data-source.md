# Rating Result data source

Data source to retrieve the rating result of a billing period in near real time

Use this data source to retrieve the rating result of a billing period in near real time.â€‹

Navigate to Reporting > Data Sources and select Rating Result from the Data Source list in the Basic Information section.

## Base object

The following table lists the fields available in the Rating Result base object.

| Object | Description |
| --- | --- |
| Rating Result | The rating result of a billing period. It contains the following fields:AmountCharge Segment NumberCreated By IDCreated DateIDIs PartialQuantityService End DateService Start DateStatusUpdated By IDUpdated Date |
