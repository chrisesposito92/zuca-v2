# Notification History - Callout

Data source to report on information for callout notification histories.

Use the Notification History - Callout data source to report on information for callout notification histories.

The following objects are available in the data source.

| Object | Description |
| --- | --- |
| CalloutHistory | Information about callout notification histories.This is the base object for the Notification History - Callout data source export. Contains the following fields:IDNotification IDContact IDObject IDEvent CategoryRequest URLResponse CodeResponse ContentCreate TimeAttempt LimitAttempt Number |
| Account | Information about the customer account that owns the callout notification histories. |
| Bill To | The contact associated with the account to whom your product or service is billed. |
| Default Payment Method | The default payment method used to make payments. |
| Parent Account | Refers to the parent account associated with the customer account, if applicable. |
| Ship To | The contact associated with the account to whom your product or service is shipped. |
| Sold To | The contact associated with the account to whom your product or service is sold. |
