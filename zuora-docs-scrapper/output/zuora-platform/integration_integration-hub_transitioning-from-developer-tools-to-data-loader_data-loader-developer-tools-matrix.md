# Data Loader - Developer Tools Matrix

Object and functions support matrix for Developer Tools, Data Loader, and alternatives.

## Object and Functions Support Matrix

| Developer Tools | Data Loader | Alternatives |
| --- | --- | --- |
| Accounts and Contacts API Loader | Import, Update, Delete and Export | Programmatic Rest APIs |
| Accounting Period API Loader | Import and Delete | Deployment Manager for migrating configurations tenant to tenantProgrammatic Rest APIs |
| Amendment Loader | Import, Update and Delete | Programmatic Rest APIs |
| Invoice API Loader | Import, Update, Delete and Export | Programmatic Rest APIs |
| Product Catalog Loader | Import, Update, Delete and Export Full Catalog Import and Export not supported | Deployment Manager for migrating full catalog configurations tenant to tenantProgrammatic Rest APIs |
| Orders API Loader | Import, DeleteAdvance Orders with multiple Order Actions - Currently in EA | Programmatic Rest APIs |
| Subscription API Loader | Import, Update, Delete and Export | Programmatic Rest APIs |
| Billing Preview | No direct equivalent | Programmatic Rest APIs |
