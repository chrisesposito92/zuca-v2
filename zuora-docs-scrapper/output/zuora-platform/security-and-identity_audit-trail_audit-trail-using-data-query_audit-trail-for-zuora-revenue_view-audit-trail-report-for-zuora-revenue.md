# View Audit trail report for Zuora Revenue

Learn how to view the Audit Trail Report in Zuora Revenue by navigating through the Reports section and applying necessary filters.

To access the audit trail, complete the following steps:

1.  Navigate to Reports \> Run Reports .
2.  Click Sox And Audit \> Audit Trail Report.
3.  Select the applicable report layout from the Select Layout drop-down field.
4.  Click Report Detail .
5.  In the Filter section, specify the criteria to filter the report data.
6.  Filter criteria are specific to each report.
7.  Click Run Reports . The report data is displayed in the lower pane under the Filters section on the report details page.

For more information, see [Work with reports](/zuora-revenue/month-end-process/reports/work-with-reports).
