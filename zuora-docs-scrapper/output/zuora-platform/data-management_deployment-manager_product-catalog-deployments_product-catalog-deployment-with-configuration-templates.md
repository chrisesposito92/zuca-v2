# Product Catalog deployment with configuration templates

Learn how to deploy product catalogs using configuration templates, including options for full or selective product configurations and API availability.

-   Configuration Templates support deployment of Product Catalog. For more information on understanding how to use Configuration Templates, click [here](/zuora-platform/data-management/deployment-manager/deployment-manager-features-and-operations/about-configuration-templates).
-   You can create Product Catalog Configuration Template without pairing it with other components.
-   There are two options for creating product catalog configuration templates.
    -   You can select the full configuration for product catalog from the source.
    -   You can select the 40 or less recently modified products in the source tenant.
-   The Advance Options which allow you to select products and rate plans are available only when you select products less than or up to 40.
-   To know more about deployment via configuration templates, click [here](/zuora-platform/data-management/deployment-manager/deployment-manager-features-and-operations/about-configuration-templates).
-   Product Catalog Deployments with Configuration Templates are also available via the configuration template APIs. To know more about these APIs, click [here](https://developer.zuora.com/v1-api-reference/api/tag/Configuration-Templates/).
