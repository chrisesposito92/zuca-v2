# Create and download source template

Learn how to create and download a source template from the Template List View screen.

You can create a source template from the Template List View screen (landing page of the Configuration Templates).

1.  Click \+ New button.
2.  In the resulting New Template pop-up, provide the Template Name and Description .
3.  Click Advanced Options button to select the subset of objects child. This step is optional.
    1.  In the resulting pop-up, select the components.
    2.  Click Add .
4.  Click Create .
    Note that when you select components and click create, all the child parameters of the selected component will be automatically saved in the template.
    You can view the newly created template on the Template List View Screen. Click the template Name to view the newly created template details


Download Template

5.  From the Template List View Screen, click Download corresponding to the template.
    The JSON file of the template will be created and downloaded, you will see it at the bottom of your browser window. Click the file name to open it.
