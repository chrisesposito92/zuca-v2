# Assign a platform role to a Workflow user

Learn how to assign a platform role to a Workflow user.

You have created or configured a target platform role for the Workflow user. See [Create a custom platform role for Workflow](/zuora-platform/extensibility/workflow/configure-global-settings-of-workflow/assign-users-workflow-access/create-a-custom-platform-role-for-workflow).

1.  Navigate to Administration > Manage Users and click the user that you want to edit permissions for.
2.  Click edit in the Basic Information tab.
3.  Select the Zuora Platform Role that you want to assign to the user.
    For example, if you want to a user to only have the Workflow view access, you can assign the Workflow Viewer to the user.

4.  Click Save .
