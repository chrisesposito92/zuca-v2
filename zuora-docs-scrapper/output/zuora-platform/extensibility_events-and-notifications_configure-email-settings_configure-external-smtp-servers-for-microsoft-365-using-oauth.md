# Configure external SMTP servers for Microsoft 365 using OAuth

Describes how to configure the external SMTP server for Microsoft 365 using OAuth 2.0 authentication.

After the configuration is completed, Zuora sends emails to your customers using Microsoft's SMTP server when email notifications are triggered.

1.  [Create an OAuth 2.0 Provider using the Authentication Code grant type](/zuora-platform/extensibility/events-and-notifications/configure-email-settings/configure-external-smtp-servers-for-microsoft-365-using-oauth/create-an-oauth-2.0-provider-using-the-authentication-code-grant-type).

2.  [Configure the external SMTP server settings for Microsoft 365](/zuora-platform/extensibility/events-and-notifications/configure-email-settings/configure-external-smtp-servers-for-microsoft-365-using-oauth/configure-the-external-smtp-server-settings-for-microsoft-365).
