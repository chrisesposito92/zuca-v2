# Change the default Org unit for the user

Learn how to change the default Org Unit assigned to a user.

While a default Org Unit is assigned to each user at the time of user creation/definition, the user can change their default Org Unit upon first login in accordance with their personalization requirements.

1.  Navigate to the Org Switcher on the right-hand top of your screen.
2.  Click the dropdown and click the gear icon in the search bar.
3.  In the Settings popup, select the Org unit you want to set as default
4.  Click Save.
