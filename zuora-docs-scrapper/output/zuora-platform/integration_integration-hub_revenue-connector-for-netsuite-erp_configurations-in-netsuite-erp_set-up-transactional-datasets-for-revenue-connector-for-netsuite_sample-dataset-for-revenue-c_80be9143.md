# Sample dataset for Revenue Connector for NetSuite

You can set up and select fields in NetSuite based on your business requirements. Refer to this example of fields from a Sales Order Lines dataset.

| Column | Alias |
| --- | --- |
| Date | TransactionDate |
| Memo | Memo |
| Posting Period | PostingPeriod |
| ID | SO_LINE_ID |
| Amount (Net) (Transaction Currency) | NetAmount |
| Line Number | LineSequenceNumber |
| Item Name/Number | ItemID |
| Number | REV_SEGMENTS |
| Number | DEF_SEGEMENTS |
| Source | TransactionSource |
| Status | Status |
| Transaction Number | TransactionNumber |
| Name | altname |
| Entity | Entity |
| Document Number/ID | TransactionID |
| Currency | Transaction Currency |
| Quantity | Quantity |
| Transaction | TransactionDisplayName |
| Type | TransactionType |
| Total Amount (Transaction Currency) | ForeignTotal |
