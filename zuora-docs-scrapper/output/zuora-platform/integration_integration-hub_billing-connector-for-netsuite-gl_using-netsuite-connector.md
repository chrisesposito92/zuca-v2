# Using NetSuite Connector

Information on using NetSuite Connector

-   [Run an integration sync](/zuora-platform/integration/integration-hub/billing-connector-for-netsuite-gl/using-netsuite-connector/run-an-integration-sync)

-   [View sync activity](/zuora-platform/integration/integration-hub/billing-connector-for-netsuite-gl/using-netsuite-connector/view-sync-activity)

-   [Edit the settings of Zuora Connector for NetSuite](/zuora-platform/integration/integration-hub/billing-connector-for-netsuite-gl/using-netsuite-connector/edit-the-configuration-settings)

## Troubleshooting

See NetSuite Connector Troubleshooting for information about troubleshooting any synchronization errors that may occur.

## Uninstalling the NetSuite Connector Bundle

If you need to uninstall NetSuite Connector, select Setup > Customization > Install Bundle > List in NetSuite. Click the Action menu next to the name of the bundle ( NetSuite Connector Product ) and select Uninstall. Follow the prompts to complete the uninstallation process.
