# Work with Zuora Connector for Salesforce CRM

This topic explains about using the Zuora Connector for Salesforce CRM to sync object data in near real-time and explore common tasks associated with it.

Zuora Connector for Salesforce CRM uses the Near Real-Time Sync mode to sync object data, which allows record changes to be synced automatically from Zuora to Salesforce in near real-time. To understand the Near Real-Time Sync and its limitations, see Real-Time Sync .
