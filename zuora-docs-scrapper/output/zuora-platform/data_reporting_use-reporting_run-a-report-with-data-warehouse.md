# Run a report with data warehouse

Learn how to run report with data warehouse

This article explains how to run reports with data warehouses, such as Zuora Warehouse, for better performance.

Note:

This is only applicable to reports with saved data query.

Currently, only Zuora Warehouse of size X-Small is available. For more information about Zuora Warehouse and warehouse size, see [Zuora Warehouse overview](/zuora-platform/data/zuora-warehouse/zuora-warehouse-overview).

You can run reports with data warehouse on the following occasions:

-   [Creating reports](/zuora-platform/data/reporting/use-reporting/run-a-report-with-data-warehouse/create-and-run-a-report-with-data-warehouse)

-   [Editing reports](/zuora-platform/data/reporting/use-reporting/run-a-report-with-data-warehouse/edit-and-run-a-report-with-data-warehouse)

-   [In the report list](/zuora-platform/data/reporting/use-reporting/run-a-report-with-data-warehouse/run-a-report-with-data-warehouse-from-the-report-list)
