# Set up transaction upload file template

Learn how to set up transaction upload file template for use in Revenue connector for NetSuite.

Define the fields in each transaction template based on your business need and use case. See [Upload transactions](/zuora-revenue/data-management/transaction-processing/upload-transactions).

There must be a transaction file template defined for each object type of NetSuite transactions such as template for Orders, Invoices and Credit Memos.
