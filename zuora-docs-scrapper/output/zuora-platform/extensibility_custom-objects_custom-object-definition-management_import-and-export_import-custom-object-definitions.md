# Import custom object definitions

Learn how to import custom object definitions in the Zuora UI.

1.  Navigate to in the left navigation.
2.  Click Custom Object Actions in the upper right of the page, and then click Import from the dropdown list. The Import Definitions window pops up.
3.  Paste the JSON format object definitions to the text editor. See the request schema of the [Create custom object definitions](https://developer.zuora.com/v1-api-reference/api/operation/POST_CustomObjectDefinitions/) API for the required JSON structure. You can find a JSON example in [JSON example for import](/zuora-platform/extensibility/custom-objects/custom-object-definition-management/import-and-export/json-example-for-import).
4.  Click Import.
