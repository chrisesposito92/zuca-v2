# Presentment metrics

Descriptions of the Presentment metrics

The following table provides descriptions of the Presentment metrics.

| Metric | Description |
| --- | --- |
| Presentment Usage | The number of billing document presentment calls sent to Zuora within a specified time range, including successful and failed billing document presentment calls.When hovering over each time interval bar, you can view the breakdown data of that time range. You can drill down to deeper levels of details by clicking a time interval bar.success : The bar displays the number of successful billing document presentment calls on a specific day or in a specific hour or minute, and the period-over-period percentage in the number accordingly.error : The bar displays the number of failed billing document presentment calls on a specific day or in a specific hour or minute, and the period-over-period percentage in the number accordingly. |
| Presentment Failures | The number of failed billing document presentment calls within a specified time range.When hovering over each time interval bar, you can view the breakdown data of that time range. You can drill down to deeper levels of details by clicking a time interval bar. |
| Presentment Performance | Presentment call handling time for a given vendor within a specified time range. Presentment call handling time means the sum of the request time and the response time of a billing document presentment call.P95 : 95th percentile of the response time dataP90 : 90th percentile of the response time dataP50 : 50th percentile of the response time dataWhen hovering over each time interval bar, you can view the breakdown data of that time range. You can drill down to deeper levels of details by clicking a time interval bar. |
