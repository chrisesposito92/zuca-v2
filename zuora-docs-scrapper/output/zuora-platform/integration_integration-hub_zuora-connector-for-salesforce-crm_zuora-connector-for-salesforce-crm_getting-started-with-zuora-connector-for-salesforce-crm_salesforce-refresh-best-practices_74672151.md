# 2\. Configuration of Salesforce user credentials

Ensure correct configuration of Salesforce user credentials in your Zuora tenant for a secure connection.

With the refreshed Salesforce org, ensure that your Salesforce user credentials are correctly configured within your Zuora tenant. This includes the Security token, username, and password. Proper configuration is vital for establishing a secure and seamless connection.
