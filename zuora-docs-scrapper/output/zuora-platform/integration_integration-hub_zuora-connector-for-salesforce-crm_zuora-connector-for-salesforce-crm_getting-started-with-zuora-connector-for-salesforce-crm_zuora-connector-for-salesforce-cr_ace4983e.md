# Deploy Zuora 360 package manually

Learn how to manually deploy the Zuora 360 package using these steps.

If you receive an email notifying you of a successful installation, but the installation process for the package continues to show the "Processing" status, you can deploy the package manually. See Troubleshooting Zuora CPQ for additional troubleshooting information.

1.  Navigate to .
2.  Verify that the Installed Packages list includes the correct version of the Zuora for Salesforce 360.
3.  Click the Zuora CPQ 360 package.
4.  On the Package Details page, click Deploy .
