# Custom Exchange Rate data source

data source to export your custom foreign exchange rates that you imported using Mass Updater

Use this data source to export your custom foreign exchange rates that you imported using [Mass Updater](/accounts-receivable/finance/mass-updater/mass-updater-overview). This data source is not available to the customers who are using the default Zuora exchange rate provider (Oanda).

## Base object and fields available in the data source

The Custom Exchange Rate data source object is not joined to any other object. This base object includes the following fields:

-   Created By ID

-   Created Date

-   Currency From

-   Currency To

-   Exchange Rate

-   ID

-   Provider Exchange Rate Date

-   Custome Rate Set

-   Updated By ID

-   Updated Date
