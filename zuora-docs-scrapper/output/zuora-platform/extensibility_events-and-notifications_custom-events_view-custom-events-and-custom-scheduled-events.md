# View custom events and custom scheduled events

Learn how to view custom events and custom scheduled events in the Zuora UI.

1.  Navigate to in the left navigation menu.
2.  Click the Custom Events tab.

    All custom events and custom scheduled events information is listed on the Custom Events page, including the name, description, type, base object, and status of each event.

    You can narrow down the event items by using filters. For more information, see [Reinvented list views](/basics/get-familiar-with-zuora-user-interface/zuora-user-interface).

    To view the detail page of a specific custom event or custom scheduled event, click the event name.
