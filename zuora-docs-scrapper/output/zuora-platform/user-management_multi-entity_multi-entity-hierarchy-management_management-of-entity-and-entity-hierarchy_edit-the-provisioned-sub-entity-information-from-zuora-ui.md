# Edit the provisioned sub-entity information from Zuora UI

Learn to edit the name and display name of provisioned sub-entities in the Zuora UI.

Only the global entity administrators can edit name and display name of the provisioned sub-entities.

To edit the name and display name of a provisioned sub-entity from the Zuora UI:

1.  Log in as an administrator of the global entity.
2.  Navigate to the global entity from the [entity switcher.](/zuora-platform/user-management/multi-entity/overview-of-multi-entity)
3.  Click your user name at the top right and navigate to Administration > Manage Entity Hierarchy.
4.  On the Manage Hierarchy of Entities page, click Edit next to the entity that you want to edit.
5.  In the Edit Entity dialog, edit the Name and Display Name fields of the entity.
6.  Click save.

To edit the other information of a provisioned sub-entity from the Zuora UI:

7.  Log in as one of the following users:
8.  Navigate to the sub-entity you want to edit from the [entity switcher](/zuora-platform/user-management/multi-entity/overview-of-multi-entity).
9.  Click your user name at the top right and navigate to Administration > Manage Entity Profile.
10.  On the Manage Entity Profile page, click edit in the Entity Information panel to edit the entity information.
11.  Click Save.
