# Edit the global entity information from Zuora UI

Learn to edit global and sub-entity information using the Zuora UI and REST API.

You can edit the information of the global entity and sub-entities from the Zuora UI and REST API.

The display name of the global entity is the tenant name. To change the tenant name, you must submit a request at [Zuora Global Support](http://support.zuora.com/) .

Edit the name of the global entity from Zuora UI:

1.  Log in as an administrator of the global entity.
2.  Navigate to the global entity from the [entity switcher](/zuora-platform/user-management/multi-entity/overview-of-multi-entity).
3.  Click your user name at the top right and navigate to Administration > Manage Entity Hierarchy .
4.  On the Manage Hierarchy of Entities page, click Edit next to the global entity.
5.  In the Edit Entity dialog, edit the Name field of the entity.
6.  Click Save.

Edit other global entity information from the Zuora UI:

7.  Log in as a global entity administrator.
8.  Navigate to the global entity from the [entity switcher](https://knowledgecenter.zuora.com/Zuora_Platform/User_Management/Multi-entity/A_Overview_of_Multi-entity#Entity_Switcher).
9.  Click your user name at the top right and navigate to Administration > Manage Entity Profile.
10.  On the Manage Entity Profile page, click edit in the Entity Information panel to update the entity information.
11.  Click Save.
