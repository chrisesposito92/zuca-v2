# Authenticate your Zuora and connector environments

Learn how to authenticate your Zuora and connector environments by navigating through the Integration Hub, selecting SAP GL, and entering necessary credentials.

1.  Navigate to Marketplace > Integration Hub .

2.  Search and select SAP GL (Revenue) .
3.  Click Authentication .
4.  Turn on the Enable Integration toggle to activate the connector.
5.  Enter the following credential details of your API user/service account:

    -   User Name

    -   Password

    -   Role

    -   Client name


6.  Click Test Connection to validate your authentication details.
7.  Click Save .
    Ensure Zuora Revenue APIs are enabled/provisioned on your tenant. If you have any issues creating API users, submit a request to [Zuora Global Support](https://www.zuora.com/support-center/) .
