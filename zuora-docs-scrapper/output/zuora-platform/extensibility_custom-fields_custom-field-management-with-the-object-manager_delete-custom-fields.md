# Delete custom fields

Learn how to delete a custom field from a standard object via the Object Manager.

1.  Navigate to in the left navigation menu.
2.  Click the name of the object that contains the custom field you want to delete. The object detail page opens.
3.  Locate the custom field you want to delete and then click the Delete icon ![Delete icon](https://zuora.deploy.heretto.com/v4/deployments/QPAZk6lsgXwvotedNERE/object/27058e0f-c08d-4fed-a45b-366f7f3de8f3?jwt=eyJ0eXAiOiJKV1QiLCJhbGciOiJIUzI1NiJ9.eyJodHRwczovL2pvcnNlay5jb20vZXpkX29yZ2FuaXphdGlvbiI6Inp1b3JhIiwiaHR0cHM6Ly9qb3JzZWsuY29tL2V6ZC9vYmplY3RfdXVpZCI6IjI3MDU4ZTBmLWMwOGQtNGZlZC1hNDViLTM2NmY3ZjNkZThmMyIsImV4cCI6MTc2NjY0MDA3NiwianRpIjoiOTJkN2NlYmExYTdiNGU3M2I3ODM2ZDNkNmU3ZDY4NTIiLCJodHRwczovL2pvcnNlay5jb20vZXpkX2ZpbGVzZXQiOiJWZHdCeUJjM0lBa01wRU9LSFdxZCJ9.jDuuamsWq3FJCK45N6WRkuEgqIistTvPac1RzfYfLT8).
4.  In the confirmation dialog, click Delete. Note that you cannot edit the custom field while deletion is in progress. The label and API name of the custom field are displayed in gray during the deletion. The deletion time varies based on the data volume of this field in your Zuora tenant. For example, deleting a custom field with 10 million records might take more than two hours.

When the deletion is completed, this custom field will be removed from the list after reloading the page.
