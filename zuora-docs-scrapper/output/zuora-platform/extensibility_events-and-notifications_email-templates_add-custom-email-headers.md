# Add custom email headers

Learn how to add custom email headers in email templates to include additional information in email notifications.

1.  When creating or editing email templates, click Add Custom Email Header in the Additional Settings section.
2.  Specify the following fields: You can add merge fields in the header name or header value.

    -   Header Name: Enter the header name of the custom email header.

    -   Header Value: Enter the header value of the custom email header.


3.  Repeat steps 1 and 2 to add more custom email headers as needed.
