# View your user profile

Learn how to access and view your user profile, including basic information, linked tenants, and user groups in the OneID portal.

To view your user profile, take the following steps:

1.  Click your avatar in the upper right of the OneID portal, and click Profile . The user profile page opens.
2.  View your user information in the Basic Information section.
3.  View the following linked tenant user and tenant information in the Linked Users section. For more information about linking the Zuora tenants to your user profile, see Link tenant users to OneID. .

    -   Tenant Id : you can click the tenant ID to access this tenant. For more information, see [Access Zuora tenants with OneID](/zuora-platform/user-management/oneid/access-zuora-tenants-through-the-oneid-portal) .

    -   Tenant Name : you can click the tenant name to access this tenant. For more information, see [Access Zuora tenants with OneID](/zuora-platform/user-management/oneid/access-zuora-tenants-through-the-oneid-portal) .

    -   Entity Id : the entity ID.

    -   Entity Name : the entity name.

    -   Domain : the URL to access the tenant.

    -   Status : the linking status of the tenant.

    -   Default tenant : indicates the default tenant of your user profile. When you log in to OneID via single sign-on (SSO), OneID will automatically access the default tenant.


4.  View user group information in the User Groups section. All the user groups you belong to are listed here. Each row in the table represents a tenant that you have access to.

    -   Tenants : the entity name

    -   User Group : the user group that contains access to the tenant

    -   Tenant Id : the tenant ID

    -   User Role : the user role from which you obtain permissions for the tenant
