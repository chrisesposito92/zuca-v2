# Activate or deactivate all communication profiles

Learn how to activate or deactivate all communication profile.

1.  Navigate to in the left navigation menu.
2.  Click Manage Communication Profiles.
3.  In the Manage Communication Profiles dialog, click Mute All or Unmute All. Note that Unmute All is available only if all communication profiles are inactive.
4.  Click Mute All or Unmute All in the confirmation dialog.
