# Configure Active Directory Federation Services for SSO SAML

Describes the configuration tasks you have to perform in Microsoft Windows Active Directory Federation Services (AD FS) 2.0 to enable single sign-on to Zuora.

Before you start configuring AD FS, see [Configure single sign-on for Zuora](/zuora-platform/system-management/administrator-settings/configure-single-sign-on-for-zuora) for the general requirements and the [provisioning process for enabling Zuora single sign-on](/zuora-platform/system-management/administrator-settings/configure-single-sign-on-for-zuora/provision-sso-for-your-tenant).
