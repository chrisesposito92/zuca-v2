# Credential Setup

Learn how to set up credentials for integrating Roku with Zuora, including obtaining and configuring necessary identifiers and keys.

Authenticate and connect to Roku Pay by providing necessary credentials in Zuora. :

1.  Obtain your Roku channel\_id (the unique identifier for your Roku channel).
2.  Obtain your Roku api\_key (the API key for accessing Roku Pay services).
3.  Add the channel\_id and api\_key to the Zuora configuration in the appropriate settings section.
