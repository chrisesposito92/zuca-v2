# Assign a role to a user

Learn how to assign a role to a user.

You can assign roles to users, including custom roles that you have created .

1.  Click your username at the top right, and navigate to .
2.  Click the user's First Name , Last Name , or User Name . Zuora displays the user's Basic Information.
3.  Click edit .
4.  Select the roles that you want to assign to the user.
5.  Click save .
