# Insert multimedia components into email templates

Learn how to insert multimedia components, such as countdown timers, social media icons, or embedded videos, into email templates.

1.  Navigate to in the left navigation menu. The Setup Profiles, Notifications and Email Templates page opens.
2.  Click the Email Templates tab.
3.  Locate the email template that you want to insert multimedia components into, and then click Edit.
4.  Scroll down to the Email Body section.
5.  In the email body editor, drag the multimedia components you want to insert from the Content tab to the desired location in the email body. See the following table for more information about each multimedia component:

    | Component | Description |
    | --- | --- |
    | Social | Adds social media links. You can select the desired social media channels and specify the style and URL. |
    | Timer | Adds a countdown timer. You can specify the end time, time zone, language, font size and color, and the behavior when recipients click it. |
    | Video | Embed a YouTube or Vimeo video. You can specify the video URL. |

6.  Click Save.
