# Procedure

Learn how to create custom fields for the Usage object using the Zuora UI.

To create custom fields for the Usage object through the Zuora UI, complete the following steps:

1.  Navigate to in the left navigation menu.
2.  In the Objects tab, locate the Usages object and click Usages in the Label column. The Usages detail page opens.
3.  In the Custom Fields tab, click the Create icon. The Add New Custom Field window opens.
4.  Enter the following information for the Type custom field:

    -   Name : Type

    -   API Name : Type\_\_c

    -   Field Type : Text

    -   Max Length : Specify the maximum length of characters that can be included for the value of the custom field.


5.  Click Save & Add Another .
6.  Repeat step 5 to enter information for the State custom field.
7.  Click Save & Close .

The Type and State custom fields are created and listed on the page.
