# Configurable Invoice Grouping supported objects and fields

You may use standard or custom fields from the Subscription, Rate Plan Charge, and Order Line Item objects when defining grouping rules.

## Subscription

-   Account Id (subscription owner account ID)

-   Name (subscription number)

-   Term Type (coming soon)

-   Sold To Contact Id (subscription level sold to contact)

-   Custom Fields


## Rate Plan Charge

-   Product Rate Plan Charge Id

-   Billing Cycle Day

-   Billing Period

-   Charge Model

-   Charge Type

-   Custom Fields


## Order Line Item

-   Item Name

-   Item Number

-   Item Type

-   Order Id

-   Sold To Contact Id

-   Custom Fields
