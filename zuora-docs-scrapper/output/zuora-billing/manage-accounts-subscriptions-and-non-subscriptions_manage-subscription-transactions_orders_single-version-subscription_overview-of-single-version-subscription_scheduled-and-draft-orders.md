# Scheduled and draft orders

This topic provides an overview of scheduled and draft orders, detailing their execution dates and available information in the UI and API.

The following table lists the scheduled and draft orders in the change history in UI.

| Scheduled or draft orders | Description |
| --- | --- |
| Scheduled orders | Scheduled Execution Date: The scheduled execution date. |
| Draft order | Draft order: Only displays the order number.In API, there is no subscription change information for draft orders. |
