# Products

Explore the Products section to view and select pricing options, including the latest and as of today charges, and access detailed product and charge information.

In the Products section, you can select Latest Pricing or As of Today in the drop-down list.

-   If the Latest Pricing option is selected, for charges, the latest charge segments are listed.

-   If the As of Today option is selected, for charges, charge segments as of today are listed.


The table below lists fields and descriptions for products.

| Field | Description |
| --- | --- |
| Plan | You can perform the following steps:Click the charge model link to view the plan details displayed in the right panel.Hover the product name to view the product summary or directly click the name to view product details. |
| Charge Name | Click one of the following to view details:Product rate plan charge name: If clicked, a right panel displays. You can switch between the following tabs to view the corresponding charge details.Basic InformationCharge AmountTiming & FrequencyFinanceThe charge details vary based on the charge model and types. For general explanations of the charge information, see View charge details.For recurring charges, the Price Change On Subscription Renewal field is not editable if the subscription is canceled or expired.History icon: Click this icon to view charge segments of the charge |
| Type/Model | Displays the charge type and charge model . |
| Quantity | Displays the quantity for the charge.The value is always 1 for the Delivery Pricing charges. If all charges under a rate plan are the Delivery Pricing charges, this field is not available. |
| Price | Charge price, for example, unit price. |
| Start - End | Start and end dates of displayed charge segments.If Latest Pricing is selected, this field indicates the period between the start date and the end date of the latest charge segment.If As of Today is selected, this field indicates the period between the start date of the first charge segment and the end date of the latest charge segment. |
