# E-Invoice integration process

Learn how to integrate with Avalara for E-Invoicing, including enabling features in Zuora, setting up accounts, and configuring environments.

Before integrating with Avalara, complete the following prerequisite tasks:

-   Enable the E-Invoicing feature in Zuora. For more information, see [E-Invoicing Overview](/zuora-billing/bill-your-customer/leverage-advanced-capabilities/e-invoicing/e-invoicing-overview).
-   Set up an Avalara E-Invoicing and Live Reporting account, see [Avalara E-Invoicing and Live Reporting account](https://knowledge.avalara.com/bundle/nsz1680268923536_nsz1680268923536/page/uhv1681300004942.html).

You can follow the tasks below to complete integration with Avalara and customize e-invoice file templates:

1.  [Configure Avalara sandbox environment](/zuora-billing/bill-your-customer/leverage-advanced-capabilities/e-invoicing/e-invoicing-with-avalara/configure-avalara-sandbox-environment)
2.  [Configure Zuora E-Invoicing feature](/zuora-billing/bill-your-customer/leverage-advanced-capabilities/e-invoicing/e-invoicing-with-avalara/configure-zuora-e-invoicing-feature)
3.  [Configure e-invoice file template and complete end-to-end testing](/zuora-billing/bill-your-customer/leverage-advanced-capabilities/e-invoicing/e-invoicing-with-avalara/e-invoice-file-template-configuration-and-complete-end-to-end-testing/end-to-end-testing-data-preparation)
4.  [Launch on production environment](/zuora-billing/bill-your-customer/leverage-advanced-capabilities/e-invoicing/e-invoicing-with-avalara/launch-on-production-environment)
