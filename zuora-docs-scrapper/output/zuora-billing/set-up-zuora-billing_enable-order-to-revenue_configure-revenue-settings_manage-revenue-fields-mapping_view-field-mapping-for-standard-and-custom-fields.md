# View field mapping for standard and custom fields

Learn how to view field mapping for standard and custom fields in booking and billing transactions.

To view the field mapping for the Order to Revenue feature, do the following:

1.  Navigate to the tab corresponding to the object on which you want to view field mapping:

    -   Booking Template tab for the Booking Transaction object

    -   Billing Template tab for the Billing Transaction object


2.  Navigate to either section according to the type of fields that you want to view, as follows:

    -   Standard Fields

    -   Custom Fields



For more information about field mapping on booking transactions, see Booking data mapping . For more information about field mapping on billing transactions, see Billing data mapping .
