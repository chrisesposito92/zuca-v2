# Delete HTML templates

Learn how to delete HTML templates through the Zuora UI when they are no longer needed.

If an HTML template is no longer maintained and used, you can delete it through the Zuora UI if needed.

To delete an HTML template through the Zuora UI, perform the following steps:

1.  Click your avatar at the top right and navigate to .
2.  On the Manage Invoice, Credit/Debit Memo Templates tab, click the HTML template that you want to delete. The selected template is displayed in the online HTML template editor.

In the Manage Invoice, Credit/Debit Memo Templates tab, you can also click Remove next to an existing HTML template to delete it.
