# Transactions II

This topic details the fields included in a debit memo, such as the debit memo number, invoice, amount, balance, date, and status.

Each debit memo includes the following information:

| Field | Description |
| --- | --- |
| Debit Memo Number | Displays the unique identifier for the debit memo. For example, DM00000007. |
| Invoice | Displays the invoice number to which the debit memo applies. |
| Amount | Displays the debit amount for the invoice. |
| Balance | Displays the debit memo balance. |
| Debit Memo Date | Displays the date when this memo is created. If the credit memo is created by bill runs, the memo date is the invoice date. |
| Status | Displays the status of the debit memo. The available status is draft, generating, error, posted, canceled, cancel in progress, or pending for tax. |
