# Update prepayment charge in subscription - Zuora UI

Learn how to update a prepayment charge in a subscription using the Zuora UI.

You can use the Zuora UI to update a prepayment charge in a subscription.

1.  [Update a product in a subscription](/zuora-billing/manage-accounts-subscriptions-and-non-subscriptions/manage-subscription-transactions/orders/order-actions-tutorials/product-update-in-subscriptions/update-a-rate-plan-using-the-zuora-ui).
2.  Select the product to be updated in this order.
3.  Update the charge details. For a prepayment charge, you can update Price and Prepaid Quantity.
