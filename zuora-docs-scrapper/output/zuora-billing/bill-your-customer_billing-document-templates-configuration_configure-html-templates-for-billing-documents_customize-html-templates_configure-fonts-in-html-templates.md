# Configure fonts in HTML templates

Learn how to configure default fonts in HTML templates, including a comprehensive list of available fonts and guidance on accessing the latest font options.

The following fonts are available in the HTML template editor by default:

| Andale Mono | Franklin Gothic | Optima |
| --- | --- | --- |
| Arial | Garamond | Oswald |
| Arial Black | Georgia | Pacifico |
| Baskerville | Gill Sans | Palatino |
| Bodoni MT / Bodoni 72 | Goudy Old Style | Perpetua |
| Book Antiqua | Helvetica | Playfair Display |
| Cabin | Impact | Poppins |
| Calibri | Lato | Raleway |
| Calisto MT | Lobster Two | Roboto |
| Cambria | Lora | Rockwell |
| Candara | Lucida Bright | Rubik |
| Century Gothic | Lucida Sans | Segoe UI |
| Cinzel | Microsoft Sans Serif | Source Sans Pro |
| Comic Sans MS | Montserrat | Space Mono |
| Consolas | Mulish | Spectral |
| Copperplate Gothic | Noto Sans | Symbol |
| Courier New | Old Standard TT | Tahoma |
| Crimson Text | Open Sans | Terminal |
| Dejavu Sans | Noto Sans Mono CJK | Times New Roman |
| Didot | Noto Serif CJK | Trebuchet MS |
| Fauna One | Old Standard TT | Verdana |
| Fjalla One | Open Sans | Verdana |

Note:

The Noto Sans CJK, Noto Sans Mono CJK, and Noto Serif CJK fonts offer support for Chinese , Japanese , and Korean ( CJK ) characters, improving visual consistency and readability of your templates across different languages and regions.​​​

The list of default fonts might change over time. To get the latest list of available default fonts, click the Text block in one HTML template and then get the list from the font family in the toolbar displayed next to the Text block.

Note that if you want irregular currency symbols displayed on the generated PDF, use the Roboto font from the font family.
