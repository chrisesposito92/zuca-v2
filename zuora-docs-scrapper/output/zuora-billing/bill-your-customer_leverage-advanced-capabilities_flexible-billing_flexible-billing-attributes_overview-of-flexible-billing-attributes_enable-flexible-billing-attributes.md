# Enable Flexible Billing Attributes

Learn how to enable Flexible Billing Attributes in your system.

Ensure that you have the Orders or Orders Harmonization feature enabled.

To enable Flexible Billing Attributes, perform the following steps:

1.  Check the known restrictions and limitations of Flexible Billing Attributes.
2.  Enable this feature in Sandbox and Production environments with the self-serve interface. For more information, see Enable billing features by yourself.
