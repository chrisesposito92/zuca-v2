# Export the SubscriptionDetails file

This task topic explains how to export the SubscriptionDetails file from Zuora to analyze TCV data, including one-time and recurring charges.

The `SubscriptionDetails` export file provides more details than the Zuora UI, and can be a useful resource.

To export the file, perform the following steps:

1.  Navigate to Customers > Subscriptions in the left-hand navigation.
2.  On the Subscriptions page, click the Export icon.

    ![export-sub.png](https://zuora.deploy.heretto.com/v4/deployments/QPAZk6lsgXwvotedNERE/object/a284f5be-b471-409e-b7b8-ebf84edbc493?jwt=eyJ0eXAiOiJKV1QiLCJhbGciOiJIUzI1NiJ9.eyJodHRwczovL2pvcnNlay5jb20vZXpkX29yZ2FuaXphdGlvbiI6Inp1b3JhIiwiaHR0cHM6Ly9qb3JzZWsuY29tL2V6ZC9vYmplY3RfdXVpZCI6ImEyODRmNWJlLWI0NzEtNDA5ZS1iN2I4LWViZjg0ZWRiYzQ5MyIsImV4cCI6MTc2NjYzOTY5MiwianRpIjoiOGNlMDkyNzQzN2VhNDY0Nzg2YWJlZDY4NGZhZTA0YTEiLCJodHRwczovL2pvcnNlay5jb20vZXpkX2ZpbGVzZXQiOiJWZHdCeUJjM0lBa01wRU9LSFdxZCJ9.fX8HUqSPyNUCLs2fKVAigHjMPWIFI2ukkaHe0miKm1g)

3.  On the Exports page, select Subscriptions Zip from the Export Data list, and then click Export . You will be prompted to confirm the export operation.
4.  Click OK to confirm the action. Zuora will prepare the file to be exported. You can refresh the Exports page to check the status in the Export Status column.
5.  Click the file name to download the .zip file containing the export file named `SubscriptionDetails.csv` after the status is completed.
