# Basic Information IV

Scheduled actions information includes:

| Field | Description |
| --- | --- |
| Scheduled Cancellation | The date on which the subscription will be canceled. |
| Scheduled Resumption | The date on which the subscription will be resumed. |
| Scheduled Suspension | The date on which the subscription will be suspended. |

Note: You can also edit the above details if required.
