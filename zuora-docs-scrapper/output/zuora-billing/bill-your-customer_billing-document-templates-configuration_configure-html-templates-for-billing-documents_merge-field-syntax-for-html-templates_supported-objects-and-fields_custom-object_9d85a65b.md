# Custom objects and custom fields

Learn about using custom objects and fields in HTML templates, including naming conventions in the Merge Fields drop-down list.

HTML templates also support custom objects and custom fields. Similar to the standard objects and fields, you can find available custom objects and custom fields in the Merge Fields drop-down list of the Text component.

In the Merge Fields drop-down list, custom object names are prefixed with default\_\_, and custom field names are suffixed with \_\_c by Zuora naming conventions.![](https://zuora.deploy.heretto.com/v4/deployments/QPAZk6lsgXwvotedNERE/object/76f5a020-9e21-4746-a3d0-434090dc558b?jwt=eyJ0eXAiOiJKV1QiLCJhbGciOiJIUzI1NiJ9.eyJodHRwczovL2pvcnNlay5jb20vZXpkX29yZ2FuaXphdGlvbiI6Inp1b3JhIiwiaHR0cHM6Ly9qb3JzZWsuY29tL2V6ZC9vYmplY3RfdXVpZCI6Ijc2ZjVhMDIwLTllMjEtNDc0Ni1hM2QwLTQzNDA5MGRjNTU4YiIsImV4cCI6MTc2NjY0MTUzMiwianRpIjoiMjFkYTI0ZGU5NDRmNDYzOGI1N2QxODEwNmMwNmU1ZmIiLCJodHRwczovL2pvcnNlay5jb20vZXpkX2ZpbGVzZXQiOiJWZHdCeUJjM0lBa01wRU9LSFdxZCJ9.FtOwWGFxJ_zjIIZAAndJR5oH318Cc7U5SEaXA18PdVg)

When new custom fields are added, or existing custom fields are updated or removed, it takes around 5 minutes to be visible in the merge fields list.

For common use cases, see Use custom objects in HTML templates.
