# View changelog for adding or updating custom field mapping

Learn how to view the changelog for adding or updating custom field mappings in different transaction objects.

To view the changelog for adding or updating custom field mapping, do the following:

1.  Navigate to the tab corresponding to the object for which you want to view changelog:

    -   Booking Template tab for the Booking Transaction object

    -   Billing Template tab for the Billing Transaction object


2.  Navigate to the Change Logs section and view the changelog.
