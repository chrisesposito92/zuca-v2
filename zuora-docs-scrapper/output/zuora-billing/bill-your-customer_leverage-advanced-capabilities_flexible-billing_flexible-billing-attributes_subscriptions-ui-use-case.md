# Subscriptions UI use case

This use case demonstrates how to view and edit billing attributes and contacts on subscription details pages.

The following use case shows you how to view the billing attributes of the Flexible Billing Attributes feature on the subscription details pages. This use case also shows you how to edit bill-to, sold-to, and ship-to contacts on the subscription details pages, and the considerations before editing the bill-to, sold-to, and ship-to contacts.
