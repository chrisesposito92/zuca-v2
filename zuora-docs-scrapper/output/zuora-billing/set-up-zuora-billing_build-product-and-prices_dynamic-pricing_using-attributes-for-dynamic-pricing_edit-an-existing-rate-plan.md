# Edit an existing rate plan

Learn how to update an existing product rate plan and its charges in the Catalog screen, including modifying pricing, attributes, and enabling Dynamic Pricing.

You can update an existing product rate plan and its charges directly in the Catalog screen. Use this workflow when you need to modify pricing, attributes, or enable Dynamic Pricing for a charge.

1.  Navigate to in the left navigation bar.
2.  Locate and click the product name that contains the rate plan you want to update.
3.  Under Plans, find the target plan and click the Vertical ellipsis (â‹®).
4.  Click Edit Plan.
5.  Update the plan name, description, dates, or any other fields as needed.
6.  Add charges to your plan, if required. For detailed steps, see [Create a product plan](/zuora-billing/set-up-zuora-billing/build-product-and-prices/dynamic-pricing/create-a-product-plan).
7.  Add discounts to your plan, if required. For detailed information, see [Create a product rate plan discount](/zuora-billing/set-up-zuora-billing/build-product-and-prices/dynamic-pricing/create-a-product-rate-plan-discount).
8.  Add existing attributes to your plan. For detailed steps on configuring attributes in a rate plan, see [Using Attributes for Dynamic Pricing](/zuora-billing/set-up-zuora-billing/build-product-and-prices/dynamic-pricing/using-attributes-for-dynamic-pricing).
