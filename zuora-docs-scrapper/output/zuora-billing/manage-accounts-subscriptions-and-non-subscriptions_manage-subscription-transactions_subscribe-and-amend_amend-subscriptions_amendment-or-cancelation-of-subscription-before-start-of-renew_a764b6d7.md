# Create early renewals

Learn how to create early renewal amendments for subscriptions by renewing subscriptions, creating a bill run, and posting invoices.

To create early renewal amendments on the subscriptions, complete the following steps:

1.  Renew subscriptions.
2.  Create a bill run for the renewal term.
3.  Post the invoices.
