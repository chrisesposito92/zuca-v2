# Add field mapping for custom fields

Learn how to add field mapping for custom fields by navigating to the appropriate tab, accessing the Custom Fields section, and configuring the mapping relation.

To add a field mapping for a custom field, do the following:

1.  Navigate to the tab corresponding to the object on which you want to add field mapping:

    -   Booking Template tab for the Booking Transaction object

    -   Billing Template tab for the Billing Transaction object


2.  Navigate to the Custom Fields section.
3.  Click +Add New . A new field mapping line is displayed. ![FieldMappingLine.png](https://zuora.deploy.heretto.com/v4/deployments/QPAZk6lsgXwvotedNERE/object/9a47e120-5d8e-40e0-8eed-4edd779fb5ce?jwt=eyJ0eXAiOiJKV1QiLCJhbGciOiJIUzI1NiJ9.eyJodHRwczovL2pvcnNlay5jb20vZXpkX29yZ2FuaXphdGlvbiI6Inp1b3JhIiwiaHR0cHM6Ly9qb3JzZWsuY29tL2V6ZC9vYmplY3RfdXVpZCI6IjlhNDdlMTIwLTVkOGUtNDBlMC04ZWVkLTRlZGQ3NzlmYjVjZSIsImV4cCI6MTc2NjYzOTU5NywianRpIjoiOWI2NzA5OTQxNzJkNGQ1MjgyZTI4OWE3MWUwNGIyYWYiLCJodHRwczovL2pvcnNlay5jb20vZXpkX2ZpbGVzZXQiOiJWZHdCeUJjM0lBa01wRU9LSFdxZCJ9.FMyWWrVihFnfEObM0pjQp0ylUWJ_EpPU0UHhhyPc6Jo)
4.  Add the mapping relation by selecting proper values for each of the following dropdown fields in the field mapping line:

    -   Source Billing Object
    -   Source Billing Field
    -   Target Revenue Field

5.  Click the save icon to save your addition.

Note:

To add multiple fields, use the multi-select data type functions when mapping the fields between Zuora Billing and Zuora Revenue.
