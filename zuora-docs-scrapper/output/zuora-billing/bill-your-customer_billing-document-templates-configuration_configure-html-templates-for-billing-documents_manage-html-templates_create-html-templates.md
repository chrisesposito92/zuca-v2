# Create HTML templates

Create and customize HTML templates for billing documents using the Zuora UI.

With the HTML Templates feature, you can create HTML templates for billing documents as needed through the Zuora UI, including invoices, credit memos, and debit memos.

To create an HTML template through the Zuora UI, perform the following steps:

1.  Click your avatar at the top right and navigate to .
2.  On the Manage Invoice, Credit/Debit Memo Templates tab, click Add New Html Template . The Choose a Template from the Library page is displayed.
3.  In the Invoice , Credit Memo , or Debit Memo tab, hover over one template, and then click Use .
    1.  If you want to start from scratch, choose the blank template.
    2.  If you want to start from a sample template, choose a predefined sample template as needed.
4.  Click Save. The Set Template Number dialog box opens. Choose one of the following:

    -   Use system-generated number: Let the system set a natural key for your usage.

    -   Specify template number: Enter a natural or external key to support CRUD operations.



A blank template or predefined sample template is displayed in the online HTML template editor. You can use the online editor to start customizing the HTML template.
