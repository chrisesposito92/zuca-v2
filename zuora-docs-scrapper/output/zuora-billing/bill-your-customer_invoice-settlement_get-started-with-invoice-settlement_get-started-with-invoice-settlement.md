# Get started with Invoice Settlement

Get started with a basic understanding of this feature

To get started with the Invoice Settlement feature, it is recommended to have a basic understanding of this feature. This feature is to replace the previous Invoice Item Adjustment feature. A migration checklist and guide are also provided to help you enable the Invoice Settlement feature for your existing environments.
