# Set the custom memo template as default

Learn how to set the custom memo template as default

You can create multiple credit and debit memo templates. By default, the default memo templates are used to generate memo PDFs for customer accounts. To use custom memo templates for all customers, set the templates as default in Zuora Billing setting. To set custom memo templates for a particular customer, set the templates in the customer's profile.
