# Add an attachment to customer accounts and subscriptions

This task topic explains how to add an attachment to a detail page, such as a Customer Account, Subscription, Invoice, Credit Memo, or Debit Memo.

1.  Open the detail page of the target object, for example, the account detail page.
2.  Scroll down to the Attachments section and click New or New Attachment.
3.  Click Choose File and select the file you want to attach.
4.  In the Description field, give a description of the attachment.
5.  Click Save or Submit to start uploading the file.

    Note: If you upload attachments with malicious contents, an error message will be displayed indicating the uploaded file is not safe
