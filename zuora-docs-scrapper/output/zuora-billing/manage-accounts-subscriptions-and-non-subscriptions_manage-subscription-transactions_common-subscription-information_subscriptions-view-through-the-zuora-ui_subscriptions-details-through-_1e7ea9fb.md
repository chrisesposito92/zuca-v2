# Basic information section

This section provides detailed information on field groups such as Owners, Terms & Conditions, Quotes, and Others, including descriptions and settings for each.

This section includes the following information.

| Field group | Field and description |
| --- | --- |
| Owners | Customer Account: Displays the account name and account number for the customer account.Invoice Owner: Displays the account name and account number for the invoice owner. |
| Terms&Conditions | Term Setting: Displays whether the subscription is a termed subscription or evergreen subscription . If the subscription is a termed subscription, you can click View Term History to display and view subscription creation and renewal history.The following fields are only available to a termed subscription:Subscription Start/End Date: Displays the subscription start/end date.Current Term: Displays the length of the period for the current subscription term.Term Start/End Date: Displays the term start/end date of the current term.Renewal Settings and Renewal Term: The Renewal Settings field displays whether the subscription is renewed with a specific term or renewed to evergreen .Auto Renew: Displays whether auto-renewal of subscription is enabled or not. |
| Quotes | The following fields are used in the Zuora Reporting Data Sources to report on Subscription metrics. If the subscription is originated from Zuora Quotes, the following values are populated with the values from Zuora Quotes.Quote Number: Displays the unique identifier of the Quote.Quote Type: Displays the Quote type that represents the subscription lifecycle stage, such as New, Amendment, Renew, or Cancel.Quote Business Type: Displays the specific identifier for the type of business transaction the Quote represents, such as New, Upsell, Downsell, Renewal, or Churn.Opportunity Name: Displays the unique identifier of the Opportunity.Opportunity Close Date: Displays the closing date of the Opportunity. |
| Others | Invoice separately?: Displays whether to create independent invoices per subscription .Notes: Displays commentary for the subscription. |
