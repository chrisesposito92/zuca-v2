# Direct Integration Tax Apps

Zuora's Direct Integration Tax Apps simplify tax calculation and reporting with pre-built integrations and customizable options.

In addition to Marketplace apps, Zuora offers Direct Integration Tax Apps which make it easy to calculate and report taxes. It includes a pre-built integration out of the box, and also gives you the flexibility to customize it using Tax Templates and Flexible Tax Mappings .

Zuora provides an easy-to-use Tax Engine interface within Zuora Billing, allowing you to set up and manage tax engines without needing to access Zuora Connect. The Tax Engine UI includes:

-   Simple and advanced setup options.

-   Ready-to-use templates (such as Tax Sales, Tax Void).

-   Configuration options for specific vendors.

-   Feature to test connections and validate credentials.

-   Built-in support for popular vendors like Avalara AvaTax, with no custom coding needed.


The direct integration tax apps include:

-   Vertex Advantage Tax App V2

-   ONESOURCE Indirect Tax Determination App V2

-   Avalara for Brazil App
