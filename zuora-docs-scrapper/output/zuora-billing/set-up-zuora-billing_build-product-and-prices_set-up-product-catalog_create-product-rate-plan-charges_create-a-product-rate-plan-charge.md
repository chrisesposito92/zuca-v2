# Create a product rate plan charge

Create and configure a product rate plan charge by selecting the appropriate charge type and model.

To create a product rate plan charge, complete the following steps:

1.  On the product details page, find the rate plan that you want to edit.
2.  Depending on the type of the charge to create, click add new under the One-Time Charges , Recurring Charges/Period , or Usage Charges/UOM section.
3.  In the Name section, specify a name for the charge and optionally provide a description.
4.  In the Charge Amount section, select the type of charge model. The charge model determines how the charge is calculated. For more information about supported charge models, see Charge Models .
5.  Complete the fields to create the new charge.
6.  Click Save .
