# Overwrite tax code and tax mode in a subscription rate plan

Learn how to overwrite the tax code and tax mode for a subscription rate plan charge by adjusting tax preferences and selecting appropriate tax options.

To overwrite tax mode and tax code for a subscription rate plan charge:

1.  Perform the Create Subscription or Add Product operation.
2.  After clicking Add Product in the bottom right corner of the Products and Charges area, click the charge name to expand the charge details, and update the subscription rate plan charge:
    1.  In the Taxation section, select Adjust Tax Preference checkbox.
    2.  Select Taxable checkbox.
    3.  Select a Tax Mode , either `Tax Inclusive` or `Tax Exclusive` .
    4.  Select an active Tax Code Name . To be taxable, the charge must be associated with an active tax code.
    5.  Click Done to complete the charge update.
3.  Click Review Order in the bottom right corner and complete the rest steps in the Create Subscription or Add Product operation.
