# Overview of Flexible Billing

This section provides an introduction to the Flexible Billing functionality, enabling efficient management of billing, taxation, and payments with features like Flexible Billing Attributes, and Billing Schedule.

The Flexible Billing functionality comprises the [Flexible Billing Attributes](/zuora-billing/bill-your-customer/leverage-advanced-capabilities/flexible-billing/flexible-billing-attributes/overview-of-flexible-billing-attributes), [Billing Schedule](/zuora-billing/bill-your-customer/leverage-advanced-capabilities/flexible-billing/billing-schedule/overview-of-billing-schedule), [Multiple Currencies](/zuora-billing/bill-your-customer/leverage-advanced-capabilities/flexible-billing/multiple-currencies/multiple-currencies-overview), and [Invoice Grouping](/zuora-billing/bill-your-customer/leverage-advanced-capabilities/flexible-billing/invoice-grouping/invoice-grouping-overview) features. All the features are generally available. You can enable the features in Sandbox and Production environments with the self-serve interface. For more information, see [Enable billing features by yourself](/zuora-billing/set-up-zuora-billing/billing-settings-configuration/system-settings/enable-billing-features-by-yourself).
