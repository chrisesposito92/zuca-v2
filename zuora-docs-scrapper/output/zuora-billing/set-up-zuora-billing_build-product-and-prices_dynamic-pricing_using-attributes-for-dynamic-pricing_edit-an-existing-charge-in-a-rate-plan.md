# Edit an existing charge in a rate plan

Learn how to edit an existing charge in a rate plan by navigating through the product catalog and modifying the plan details.

1.  Navigate to in the left navigation bar.
2.  Locate and click the product name that contains the rate plan you want to update.
3.  Under Plans, find the target plan and click the Vertical ellipsis (â‹®).
4.  Click Edit Plan.
5.  Scroll to the Pricing section and click the Edit icon next to the required plan you want to edit.
6.  Modify the information as required. For detailed steps, see [Create a product plan](/zuora-billing/set-up-zuora-billing/build-product-and-prices/dynamic-pricing/create-a-product-plan).
7.  Click Save.
