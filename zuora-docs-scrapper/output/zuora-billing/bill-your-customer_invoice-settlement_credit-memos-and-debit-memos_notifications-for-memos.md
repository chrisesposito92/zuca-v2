# Notifications for memos

Configure credit memo and debit memo notifications from the Zuora UI

You can configure credit memo and debit memo notifications from the Zuora UI. The email and callout notifications can be sent automatically when the following events are triggered:

-   A credit memo is created

-   A credit memo is posted

-   A debit memo is created

-   A debit memo is posted


For more information about the credit memo and debit memo events, see Standard events for Zuora Payments .

Also, you can send the generated credit memo PDFs or debit memo PDFs in the emails and callouts. See Download and Email Memo Documents .
