# Bar Code Fields

This reference details the use of barcode merge fields in invoice templates, including support for Code 128 and Australia Post barcodes.

| Merge Field | Description |
| --- | --- |
| BarCode.Code128 | In order to have a barcode on your invoice, you need to use a barcode merge field in your template. Currently, Zuora supports up to five Code 128 barcodes per invoice template.This field cannot be translated. |
| BarCode.AustraliaPost | Zuora also supports barcodes generated by Australia Post. See Using Barcodes on Invoice Templates .This field cannot be translated. |
