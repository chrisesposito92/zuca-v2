# Basic Information V

This section provides details about quote information, including quote number, type, business type, opportunity name, and close date.

Quote information includes:

| Field | Description |
| --- | --- |
| Quote Number | Displays the unique identifier of the Quote. |
| Quote Type | Displays the Quote type that represents the subscription lifecycle stage, such as New, Amendment, Renew, or Cancel. |
| Quote Business Type | Displays the specific identifier for the type of business transaction the Quote represents, such as New, Upsel, Downsell, Renewal, or Churn. |
| Opportunity Name | Displays the unique identifier of the Opportunity. |
| Opportunity Close Date | Displays the closing date of the Opportunity. |
