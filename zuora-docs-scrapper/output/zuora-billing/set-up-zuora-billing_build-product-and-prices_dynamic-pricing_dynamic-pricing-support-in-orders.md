# Dynamic pricing support in Orders

Add dynamic pricing-enabled products to subscriptions by creating orders via Zuora UI or REST APIs.

You can add a dynamic pricing-enabled product to a subscription by creating an order, either through the Zuora UI or the REST APIs.
