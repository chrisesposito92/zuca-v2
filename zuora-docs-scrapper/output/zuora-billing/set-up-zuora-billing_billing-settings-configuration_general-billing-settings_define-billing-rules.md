# Define billing rules

Learn how to define billing rules for proration, billing in advance, billing in arrears, and usage billing.

You can use the Billing Rules page to define your company's billing policies concerning proration, billing in advance, billing in arrears, and billing for individual or aggregated usage.

To define a billing rule, perform the following steps:

1.  Click your avatar in the upper right and click Settings > Billing .
2.  On the Billing Settings page, click Define Billing Rules .
3.  On the Billing Rules page, navigate to the specific section where the billing rule you want to define is located.
4.  In the corresponding row of the billing rule, click the Edit icon to start the configuration.
5.  From the Option list, select an option for the billing rule.
6.  Click the Save icon to save the configuration.

You can also type keywords in the search bar at the top of the Billing Rules page to quickly identify the billing rule.
