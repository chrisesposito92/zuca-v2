# Delete product charge definitions

You can delete product charge definitions using the Zuora UI or REST API.

You can delete a charge definition either from the Zuora UI or through REST API.

## With REST API

To delete a charge definition, use the [Delete a product rate plan definition](https://www.zuora.com/developer/api-references/api/operation/DELETE_ProductChargeDefnition/) operation.
