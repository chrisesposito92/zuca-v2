# Preview billing documents in HTML templates

Learn how to preview billing documents in HTML templates using the Zuora UI.

After an HTML template is created, you can preview billing documents in the HTML template through the Zuora UI.

To preview billing documents in HTML templates through the Zuora UI, perform the following steps:

1.  Click your avatar at the top right and navigate to .
2.  On the Manage Invoice, Credit/Debit Memo Templates tab, click the HTML template that you want to use for preview. The selected template is displayed in the online HTML template editor.
3.  In the HTML template editor, click Preview to switch to the Preview mode from the Edit mode.
4.  In the Preview Settings area, configure the following fields:
    1.  In the Account field, search for account keywords and select the account that you want to preview billing documents for.
    2.  From the Invoice , Credit Memo , or Debit Memo list, select the invoice, credit memo, or debit memo that you want to preview in the HTML template.

The displayed HTML template changes to a PDF file with actual data for you to preview.
