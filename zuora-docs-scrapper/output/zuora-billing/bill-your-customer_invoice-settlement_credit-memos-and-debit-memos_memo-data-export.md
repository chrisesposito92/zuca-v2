# Memo data export

Learn about memo data export

You can export credit memo and debit memo data from Zuora data sources. The following data sources are available for you to download:

-   Application Group

-   Credit Memo

-   Credit Memo Item

-   Credit Memo Application

-   Credit Memo Application Item

-   Credit Memo Part

-   Credit Memo Part Item

-   Debit Memo

-   Debit Memo Item

-   Credit Taxation Item

-   Debit Taxation Item

-   Revenue Schedule Item Credit Memo Item

-   Revenue Schedule Item Debit Memo Item

-   Revenue Event Item Credit Memo Item

-   Revenue Event Item Debit Memo Item


These data sources provide detailed information on your credit and debit memos. You can export and add the data to other reporting, accounting, business intelligence, and analytic tools.

For more information on exporting data from the above data sources, see Generate a Data Source Export .

## Deprecated data sources

The following data sources are deprecated after the Credit and Debit Memos feature is enabled:

-   Credit Balance Adjustment

-   Invoice Adjustment

-   Invoice Item Adjustment
