# Configure headers and footers in HTML templates

Learn how to configure headers and footers in HTML templates using predefined options or customizable settings.

Note:

The HTML Templates feature is generally available for invoices as of Zuora Release 2022.02.R2, and for credit and debit memos as of Zuora Release 2022.05.R1.

After an HTML template is created, you can configure a header and footer for the HTML template in the following approaches:

-   Using predefined headers and footers

-   Configuring a header and footer with a predefined page number

-   Configuring a header and footer with a customizable page number
