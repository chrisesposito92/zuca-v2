# Set up product catalog

Learn how to set up your product catalog to manage products and pricing for subscription businesses.

Before you can create subscriptions for your end customers, setting up your product catalog is essential so that products and rate plans can be selected and added to a subscription. Use the Product Catalog page under the Product section in the UI to manage products and pricing for subscription businesses.
