# Details view of single version subscriptions

Explore tasks and sections available on the subscription detail page for single version subscriptions, including canceling subscriptions, creating orders, and viewing change history.

You can perform the following tasks on the top right of the subscription detail page of single version subscriptions.

-   Cancel Subscriptions

-   Create Order

-   More > Preview

-   More > Create Bill Run

-   More > Attachments

-   More > Create Delivery Adjustment


The subscription detail page of single version subscriptions includes the following sections:

-   Basic Information and Change History

-   Products

-   Transactions

-   Orders
