# Product rate plans

To support this scenario in the catalog, you will define a Product and its corresponding Product Rate Plans (PRPs), which represent the different marketable subscription tiers.

Each product rate plan is created as a marketable plan and might have different effective dates.

| Name | PRP Number |
| --- | --- |
| Bronze | PRP-01 |
| Silver | PRP-02 |
| Gold | PRP-03 |
