# View the status and details of usage records

Learn how to view the status and details of usage records after they are created and billed in the Zuora UI.

After the usage records are successfully created and billed, you can view the status and details of the usage records.

To view information about usage records through the Zuora UI:

1.  In the left-hand navigation section, navigate to .
2.  On the Usage page, view the information about the usage record that you created.

The status of the uploaded usage records is Processed.

If the usage records have not been successfully billed, for example, the bill run finished in error, please see Retrieve rating details about usage charges or records to understand some of the reasons why and how to correct the errors.
