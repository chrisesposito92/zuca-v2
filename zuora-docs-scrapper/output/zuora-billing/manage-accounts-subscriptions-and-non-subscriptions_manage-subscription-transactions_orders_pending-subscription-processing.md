# Pending Subscription Processing

This topic covers the Early Adopter phase of the Pending Subscription Processing feature, inviting feedback from early adopters.

This guide introduces the Pending Subscription Processing feature.

Note:

The Pending Subscription Processing feature is in the Early Adopter phase. We are actively soliciting feedback from a small set of early adopters before releasing it as generally available. If you want to join this early adopter program, submit a request at Zuora Global Support .
