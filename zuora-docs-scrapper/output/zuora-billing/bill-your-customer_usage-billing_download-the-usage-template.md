# Download the usage template

Learn how to download and populate the Export Usage File template to upload usage charges.

To upload usage charges, you must first download and populate the Export Usage File template.

To download the template:

1.  Navigate to Billing > Usage in the left-hand navigation section.
2.  At the upper-right corner of the page, click add new usage records .
3.  Next to Download a usage file template, click either Excel or CSV. The template will open in the selected format.

Do not delete any columns in the Usage template file. Although the values are optional and the fields are empty, the Usage file will not import successfully if any of the column field names in the downloaded Usage template do not exist in the uploaded Usage file.
