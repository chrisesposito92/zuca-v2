# Configure page setup for HTML templates

Learn how to configure page setup settings for PDF files generated from HTML templates in the Zuora UI.

In the HTML template editor, you can configure page setup settings to apply to the PDF files that are generated from HTML templates for billing documents, including invoices, credit memos, and debit memos.

To configure page setup settings for the PDF files generated from an HTML template through the Zuora UI, perform the following steps:

1.  Click your avatar at the top right and navigate to .
2.  On the Manage Invoice, Credit/Debit Memo Templates tab, click Add New Html Template . The Choose a Template from the Library page is displayed.
3.  In the Invoice , Credit Memo , and Debit Memos tab, hover over the blank template, and then click Use .
4.  In the HTML template editor, click the more options icon and then click Setup .
5.  In the Page Setup dialog that is displayed, perform the following steps:
    1.  In the Page Orientation section, click Portrait or Landscape .
    2.  From the Page Size list, select one size option to apply to the PDF files generated from the HTML template.
    3.  Configure the page margins in four directions.
    4.  Click Save to save the configurations.
6.  In the HTML template editor, click Save to save the HTML template.

    The page setup settings cannot be saved until you save the HTML template. You can preview the template to see how the page setup takes effect on PDF files.
