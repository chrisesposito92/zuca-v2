# Edit an attachment

This task explains how to edit an attachment.

Complete the following steps to edit an attachment:

1.  On the detail page of the target object, scroll down to the Attachments section.
2.  Click the title of the attachment you want to edit. The file name and description fields become editable.
3.  Make the changes and click Save.

    Note: You cannot edit the content in the attachment file. To edit the file content, delete the attachment and upload another updated attachment.
