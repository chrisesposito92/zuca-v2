# Suspending a Subscription from the Zuora UI

This article explains how to suspend an active subscription using the Zuora UI, including specifying booking and suspend dates, and optionally setting a resume date.

You can suspend an active subscription from the Zuora UI. If you have enabled the Orders feature, see Suspend a Subscription for how to suspend a subscription in Orders.

To suspend an active subscription:

1.  Click on the subscription you want to change from the subscription list view page.
2.  Click more > Suspend . Suspend Subscription dialog is displayed.
3.  Specify the Booking Date . The Booking Date field is set for the amendment contract and is only available when the new UI experience is enabled .
4.  Select one of the following options to specify a suspend date:

    -   Today
    -   End of Last Invoice Period
    -   The number of periods from today
    -   Specific date

5.  Specify a date in the Amendment Contract Effective Date field. This is the date that your customer provided as the notice of suspension.
6.  (Optional) You can select the You can also optionally choose a date to resume automatically check box to set a subscription resume date. If you select this check box, select one of the following options to specify a resume date: See Resume a Subscription for more information.

    -   Today
    -   The number of periods from the date suspended
    -   The number of periods from today
    -   Specific date


7.  Click ok .

A Suspend Subscription amendment is automatically created. You can view it in the Subscriptions detail screen .
