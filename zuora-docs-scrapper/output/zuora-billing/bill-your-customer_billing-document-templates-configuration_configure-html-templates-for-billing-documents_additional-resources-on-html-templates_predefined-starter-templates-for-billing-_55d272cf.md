# Predefined starter templates for debit memos

This section provides an overview of the Basic starter template for debit memos, including predefined data for company information, charge details, and memo summaries.

Currently, only one starter template called Basic is available for debit memos.

## Basic

The template called Basic contains predefined data including basic company and contact information, a predefined charge details table, and debit memo details.

## Header section

The header section contains a sample company logo image and billing document type.

-   Sample company logo image in the upper left of the template

-   Billing document type in the upper right: Debit Note


## Basic information section

The basic information section contains predefined text and merge fields for company and contact information.

-   Sample text for the company name, address, and contact information

-   Sample text and merge fields for balance due

-   Sample text and merge fields for bill-to contact information

-   Sample text and merge fields for account and memo summary information


## Charge Details section

The Charge Details section contains a predefined charge details table.

To add more columns to the table, click the Data Table block for the charge details table, and then configure column information in the Content panel that is displayed on the right of the template editor.

## Total section

The Total section contains predefined text and merge fields for debit memo summary information, including comments, amount without tax, tax amount, total amount, and balance due.
