# Predefined starter templates for billing documents

This article introduces the starter templates predefined in the online HTML template library for billing documents, including invoices, credit memos, and debit memos.

## Predefined starter templates for invoices

Currently, nine starter templates are available for invoices.

## Basic

The template called Basic contains predefined data including basic company and contact information, a predefined charge details table, and invoice details.

## Header section

The header section contains a sample company logo image and billing document type.

-   Sample company logo image in the upper left of the template

-   Billing document type in the upper right: Invoice


## Basic information section

The basic information section contains predefined text and merge fields for company, invoice, and contact information.

-   Sample text for the company name, address, and contact information

-   Sample text and merge fields for account and invoice summary information

-   Sample text and merge fields for bill-to contact information

-   Sample text and merge fields for sold-to contact information


## Charge Details section

The Charge Details section contains a predefined charge details table. For the descriptions of the column headers in this table, see Charge Details table .

To make adjustments to the table, click the Data Table block for the charge details table, and then configure information in the Content panel that is displayed on the right of the template editor.

## Invoice Total section

The Invoice Total section contains predefined text and merge fields for invoice details information, including comments, amount without tax, tax amount, total amount, and balance.

## Basic - Header with Page Number

The template called Basic - Header with Page Number is identical to the Basic template, except for a predefined page number field that is directly below the billing document type in the header section.

To make adjustments to this field, click the Page Number block, and then configure information in the Content panel that is displayed on the right of the template editor. For example, you can choose from one of three formats of page numbers to display.

## Basic - Header with Customizable Page Number

The template called Basic - Header with customizable Page Number is identical to the Basic template, except for a predefined HTML field that is directly below the billing document type in the header section.

To customize the header and footer, click the HTML block, and then write custom HTML code in the Content panel that is displayed on the right of the template editor.

## Basic - Watermark

The template called Basic - Watermark is identical to the Basic template, except for a predefined HTML field that defines a watermark displayed in the middle of the template. The predefined watermark is a capitalized word TESTING in red.

To customize the watermark, click the HTML block, and then write custom HTML code in the Content panel that is displayed on the right of the template editor.

For more information about watermark configuration in HTML invoice templates, see How to add watermark to HTML invoice templates .

## Discount Inline with Charge Amount

The template called Discount Inline with Charge Amount is identical to the Basic template, except for the Charge Details section.

The charge details table in the Discount Inline with Charge Amount template is the same one as it is described in the predefined Charge Details table for invoices .

## Charge Details section

The Charge Details section contains a predefined regular charge items table and a bundled charge items table. Both tables are configured by HTML code.

To make adjustments to any of these tables, click the HTML block for the usage summary table, and then write custom HTML code in the Content panel that is displayed on the right of the template editor.

Note that some information about each table is above the HTML block of each table. Such information is for your reference, you might remove it when you preview the invoice.

## Invoice Total section

The Invoice Total section contains predefined text and merge fields for invoice details information, including comments, amount without tax, tax amount, total amount, and balance.

## Basic - Payment Sequence

The template called Basic - Payment Sequence is similar to the Basic template. The major difference is the Charge Detailed section and the instructions above it.

The charge details table in the Basic - Payment Sequence template introduces payment sequences as a column.

For more information about payment sequence configuration in HTML templates, see How to display payment sequences on invoices .
