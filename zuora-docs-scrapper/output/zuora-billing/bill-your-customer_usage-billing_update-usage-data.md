# Update usage data

Learn how to update custom usage fields with the correct permissions.

With the correct permissions assigned, you can update custom usage fields. To update usage data:

1.  Navigate to Billing > Usage in the left-hand navigation section.
2.  Find the record to update and click Status . Note that you can only update records in the Pending status.
3.  Edit the fields and click Update .

The updated usage information is saved.
