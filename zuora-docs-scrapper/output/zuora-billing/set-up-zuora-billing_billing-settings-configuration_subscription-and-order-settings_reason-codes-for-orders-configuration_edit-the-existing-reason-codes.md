# Edit the existing reason codes

This topic explains how to edit existing reason codes by following a series of steps to update and activate them automatically.

To edit an existing reason code, perform the following steps:

1.  In the reason code row that you want to edit, click Edit .
2.  In the Reason Code Name column, enter your reason for returning an order in the empty field, then click Save . Once saved, the reason code is activated automatically.
