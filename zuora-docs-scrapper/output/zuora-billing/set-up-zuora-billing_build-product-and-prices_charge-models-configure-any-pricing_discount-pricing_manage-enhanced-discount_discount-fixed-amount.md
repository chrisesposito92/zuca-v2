# Discount-fixed amount

Provides guidance on configuring the fixed amount discount, including charge models, discount application, and charge types.

When configuring the Charge Amount section for the fixed amount charge, refer to the following table:

| Field | Description |
| --- | --- |
| Charge Model | Discount-Fixed Amount |
| Discount Amount | Enter the discount amount for the billing period of the fixed amount discount. |
| Apply Discount To | Select one of the following:All charges in the rate planAll charges in the subscriptionNote:The charges to be applied with the enhanced discount must not be delivery pricing, prepaid and drawdown, or discount charges.For the enhanced discount of the Discount-Fixed Amount charge model, you can apply the enhanced discount to only one charge in the subscription. |
| Allow apply to billing period partially | You must check this checkbox to change your current discounts to enhanced discounts. |
| Select one of the following charge types:One-TimeRecurring |  |
