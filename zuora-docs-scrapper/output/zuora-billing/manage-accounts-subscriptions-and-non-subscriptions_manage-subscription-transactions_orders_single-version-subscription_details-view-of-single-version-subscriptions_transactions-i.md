# Transactions I

The Transactions section provides details on invoices, credit memos, debit memos, and order line items, including fields such as invoice number, date, due date, amount, balance, and status.

The Transactions section includes the following tabs and information:

-   Invoices


-   Credit Memos


-   Debit Memos


-   Order Line Items


Each invoice includes the following information:

| Field | Description |
| --- | --- |
| Invoice Number | Displays the unique identifier for the invoice. For example, INV00000001. |
| Invoice Date | Displays the invoice date. |
| Due Date | Displays the invoice due date. |
| Amount | Displays the net amount of the invoice. |
| Balance | Displays the invoice balance. |
| Status | Displays whether the invoice is in a draft, posted, or canceled status . |
