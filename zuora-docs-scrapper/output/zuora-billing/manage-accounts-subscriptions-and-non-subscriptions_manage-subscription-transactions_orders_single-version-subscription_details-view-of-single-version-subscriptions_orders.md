# Orders

This topic explains how to view and manage orders by status in the Orders section.

In the Orders section, you can view all related orders in the All tab, or you can click one of the following tabs to view orders in the following order statuses.

-   Pending

-   Draft

-   Reverted

-   Cancelled


For more information, see order status .
