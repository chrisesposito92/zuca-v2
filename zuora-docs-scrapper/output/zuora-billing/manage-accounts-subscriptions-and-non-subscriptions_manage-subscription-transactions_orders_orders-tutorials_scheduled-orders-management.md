# Scheduled orders management

This topic explains how to manage and schedule orders for subscriptions using various order actions through the Zuora UI or REST API.

For a subscription, you can schedule one or more orders in a scheduler to be executed on future dates. For more information, see Scheduled orders .

You can create orders and save the orders as scheduled orders for the following order actions:

-   Create Subscription

-   Add Product

-   Update Product

-   Remove Product

-   Terms And Conditions

-   Suspend

-   Resume

-   Renewal

-   Owner Transfer

-   Cancelation


You can create orders and save the orders as scheduled orders using the Zuora UI or REST API.
