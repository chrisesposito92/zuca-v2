# Memo status

Description of memo statuses

The following table describes the memo statuses.

| Field | Description |
| --- | --- |
| Draft | The credit or debit memo that has been created but has not been posted. |
| Posted | The credit or debit memo has been posted.For a credit memo, the memo is now ready to be applied to pay invoices or debit memos.For a debit memo, the memo is now ready for payments to be collected. |
| Canceled | The credit or debit memo has been canceled. |
