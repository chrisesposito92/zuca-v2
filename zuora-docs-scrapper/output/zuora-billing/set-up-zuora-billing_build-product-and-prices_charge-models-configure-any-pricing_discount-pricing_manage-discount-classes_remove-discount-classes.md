# Remove discount classes

Learn how to remove a discount class by deactivating it and ensuring it is not used in any discount rate plan charges.

You can remove a discount class only when it is deactivated and not used in any discount rate plan charges.

To remove a discount class, complete the following steps:

1.  Deactivate the discount class that you want to remove.
2.  In the Actions column of the existing discount class, click Remove .
3.  When you are prompted from confirmation, click OK .
