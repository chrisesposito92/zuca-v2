# Add a Feature to a Product in an Amendment

This article explains how to add features to a subscription product by creating an amendment and selecting the desired features.

To add a feature to a subscription product :

1.  Create an amendment of the type, New Product or Update a Product .
2.  In the Amendment Detail section, click add features .
3.  Select the features you want to add to this product. If you have many features, use the Search field to narrow your selection. A subscription product rate plan can contain up to 100 features.
4.  Click add selected feature(s) .
