# Configure fulfillment settings

This topic explains how to configure fulfillment settings for order line items in Zuora Billing, including carriers, systems, and locations.

On the Fulfillment Settings page, you can configure the following fulfillment settings for the order line items in Zuora Billing:

-   Fulfillment carrier

-   Fulfillment system

-   Fulfillment location


To navigate to the Fulfillment Settings page, click your username at the top right and navigate to Billing > Fulfillment Settings.

See the following sections for detailed steps to configure each fulfillment setting:

## Configure fulfillment carriers

The Fulfillment Carrier section on the Fulfillment Settings page lists all the fulfillment carrier options available in your system.

You can perform the following operations in this area:

-   To add a new carrier, click Add New Carrier, enter the name of the carrier you are to add to the system, and then click Save .

-   To modify an existing carrier, click Edit corresponding to the carrier you will change, edit the carrier name in the text box, and then click Save .

-   To delete an existing carrier, click Delete corresponding to the carrier you will delete and click Yes to confirm the deletion.


## Configure fulfillment systems

The Fulfillment System section on the Fulfillment Settings page lists all the fulfillment system options available in your system.

You can perform the following operations in this area:

-   To add a new fulfillment system, click Add New System, enter the system name, and click Save.

-   To modify an existing system, click Edit corresponding to the system you are to change, edit the system name in the text box, and then click Save .

-   To delete an existing system, click Delete corresponding to the system you are to delete and click Yes to confirm the deletion.


## Configure fulfillment locations

The Fulfillment Location section on the Fulfillment Settings page lists all the fulfillment location options available in your system.

You can perform the following operations in this area:

-   To add a new location, click Add New Location, enter the name of the location you will add to the system, and then click Save .

-   To modify an existing location, click Edit corresponding to the location you are to change, edit the location name in the text box, and then click Save.

-   To delete an existing location, click Delete corresponding to the location you are to delete and click Yes to confirm the deletion.
