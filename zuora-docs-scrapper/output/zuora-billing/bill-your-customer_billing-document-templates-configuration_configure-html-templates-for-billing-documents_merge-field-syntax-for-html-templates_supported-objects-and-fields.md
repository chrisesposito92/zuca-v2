# Supported objects and fields

Learn about the types of objects supported by Zuora for customizing HTML templates, including invoice-related and custom objects.

Zuora supports two types of objects when customizing HTML templates.

1.  Objects related to invoices, credit memos, or debit memos.
2.  Custom objects.

    All the available objects and fields can be viewed in the data schema tree.
