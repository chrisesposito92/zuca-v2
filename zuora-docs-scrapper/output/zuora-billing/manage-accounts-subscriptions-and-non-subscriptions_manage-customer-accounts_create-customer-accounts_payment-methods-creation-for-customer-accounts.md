# Payment methods creation for customer accounts

This topic explains how to set up and manage payment methods for customer accounts, with a limit of 100 methods per account.

You can set up payment methods for both new customer accounts during the account creation process, or by editing an existing account. You can add a maximum of 100 payment methods to an account. See Define and set up payment methods for detailed instructions.
