# Configure attributes in a rate plan

Learn how to configure and add multiple attributes to a rate plan using the Plan screen in Zuora.

1.  Click Attributes from the Plan screen.
2.  Select the required attribute from the drop-down list. These attributes must be pre-defined in Zuora (created under the Context and Attributes section under the Extension Studio option).
3.  Enter the required value for the selected attribute.
4.  Use the + icon to add multiple attributes to the rate plan.
