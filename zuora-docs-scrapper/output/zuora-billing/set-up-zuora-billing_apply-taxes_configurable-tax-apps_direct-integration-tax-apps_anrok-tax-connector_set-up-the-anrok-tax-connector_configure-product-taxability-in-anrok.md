# Configure product taxability in Anrok

Learn how to configure product taxability in Anrok to ensure accurate taxation before processing transactions in Zuora.

Configure product taxability in Anrok to ensure products are taxed correctly before processing transactions in Zuora.

1.  Sign in to the Anrok portal.
2.  Navigate to the Product IDs section.
3.  Link your Zuora account with Anrok to synchronize the Zuora product catalog and transaction data automatically.
4.  For additional configuration options, see the [Anrok documentation](https://help-center.anrok.com/hc/en-us/articles/20709175221011-Integrate-Anrok-Zuora).
