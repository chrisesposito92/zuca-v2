# Generate memo PDFs with custom memo template

Learn how to generate memo PDFs with custom memo template

Zuora generates the credit memo and debit memo PDFs based on the templates you set. You can set either Zuora default memo templates or your custom templates to generate memo PDFs. By default, Zuora default memo templates are used.

To use your custom memo template for generating memo PDFs, complete the following steps:

1.  Create a custom memo template.
2.  Upload the custom memo template to Zuora.
3.  Set the custom memo template as the default template.
