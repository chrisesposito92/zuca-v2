# Fulfillments management

This topic explains about managing fulfillments by adding, updating, canceling, and viewing details for order line items.

You can perform the following tasks on fulfillments:

-   Add fulfillments to order line items

-   Update or cancel fulfillments

-   View details of fulfillments
