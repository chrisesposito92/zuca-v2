# Create discount classes

Create and configure discount classes to manage the application priority of discount product rate plan charges.

You can create discount classes to define the sequence in which discount product rate plan charges are applied.

1.  Click your username at the top right, and navigate to .
2.  If you want to create a discount class that defines the highest application priority, click add new discount class .
3.  If you want to create a discount class that defines a lower application priority than an existing discount class, click Add in the Actions column of the existing discount class.
4.  Configure the following information for the new discount class.
    1.  In the Discount Class column, enter the discount class name.
    2.  In the Active column, specify the status of the discount class. If you want the discount class to be active, keep the default setting unchanged. If you want the discount class to be inactive, clear the checkbox. By default, the checkbox is selected.
    3.  In the Actions column, click Save to save the configurations.
