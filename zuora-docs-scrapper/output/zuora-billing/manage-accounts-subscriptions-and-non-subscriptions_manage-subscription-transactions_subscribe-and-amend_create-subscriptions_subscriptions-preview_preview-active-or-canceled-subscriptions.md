# Preview active or canceled subscriptions

You can preview a subscription at any time (when the subscription is in draft, active, or canceled status) to see the invoice charges that will be generated by the subscription.

To preview an active or canceled subscription.

1.  Access the subscription you want to preview.
2.  Click Preview.

    Note: If the subscription is active or canceled, the subscription's term start date, term end date, and contract effective date will be pre-populated on the preview page, and you will not be able to edit these values.

3.  Specify a preview start date in the Preview Start Date field.
4.  Select one of the following options to specify the preview through date in the Preview Through Date area.
5.  Specify a preview start date in the Preview Start Date field.
6.  Click refresh to preview the charges.
7.  Select one of the following options to specify the preview through date in the Preview Through Date area:
    1.  Next number billing periods
    2.  End of Term
    3.  Specific Date
8.  Click preview to preview the charges.
9.  Click done when you are finished previewing the charges to return to the subscription entry page.
