# Edit a Mail Merge field in a billing document template

Learn how to edit merge fields in a billing document template using Edit Field or Toggle Field Codes on Windows and Mac.

1.  To update a merge field using Edit Field on Windows:
    1.  In your template document, right-click on the merge field you want to edit and select Edit Field.
    2.  In the Field name field, enter the new merge field name.
    3.  Click OK .
2.  To update a merge field using Toggle Field Codes on Windows and Mac,
    1.  In your template document, click the merge field you want to edit.
    2.  Toggle the field code and result:

        -   Press Shift + F9 on Windows.

        -   Press option + fn + F9 on Mac.


    3.  Edit the field name in the field code.
    4.  Update the field:

        -   Press F9 on Windows.

        -   Press option + fn + F9 on Mac.


    5.  Repeat Step 2 to verify that your change has been saved.

        Note:

        You cannot change a merge field by simply typing a new merge field over the current field.
