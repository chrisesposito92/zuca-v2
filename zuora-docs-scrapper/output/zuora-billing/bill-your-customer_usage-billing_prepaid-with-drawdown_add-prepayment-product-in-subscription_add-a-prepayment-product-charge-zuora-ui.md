# Add a prepayment product charge - Zuora UI

Learn how to add a product prepayment charge to a subscription using the Zuora UI.

You can use the Zuora UI to add a product prepayment charge to a subscription.

1.  [Add a product to a subscription](/zuora-billing/manage-accounts-subscriptions-and-non-subscriptions/manage-subscription-transactions/orders/order-actions-tutorials/add-products-to-subscriptions/add-a-product-rate-plan-using-the-zuora-ui).
2.  Select the product to be added to the order.

    ![Add one time product](https://zuora.deploy.heretto.com/v4/deployments/QPAZk6lsgXwvotedNERE/object/9c704fba-b39c-4f58-ab12-7d05884d9cf4?jwt=eyJ0eXAiOiJKV1QiLCJhbGciOiJIUzI1NiJ9.eyJodHRwczovL2pvcnNlay5jb20vZXpkX29yZ2FuaXphdGlvbiI6Inp1b3JhIiwiaHR0cHM6Ly9qb3JzZWsuY29tL2V6ZC9vYmplY3RfdXVpZCI6IjljNzA0ZmJhLWIzOWMtNGY1OC1hYjEyLTdkMDU4ODRkOWNmNCIsImV4cCI6MTc2NjY1MTM4OSwianRpIjoiOTMyNzI3ZjRhYWUxNDNhMTk1YmIyZGI5ODIwZWZmZDUiLCJodHRwczovL2pvcnNlay5jb20vZXpkX2ZpbGVzZXQiOiJWZHdCeUJjM0lBa01wRU9LSFdxZCJ9.VTfHAmQUMvsv8OzHJHPhNnIrjzZIRe0Xi0xT4RK-wpM)

3.  Update the one-time charge details if needed. In the example, you can increase the prepaid quantity to 10 if your customer needs an extra 10 Million API calls on an ad-hoc basis.

    -   For a prepayment charge , you can update Price, Prepaid Quantity, and Validity Period.

    -   For a drawdown charge, you can update Price and Drawdown Rate.
