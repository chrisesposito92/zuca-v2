# Download e-invoice

Learn how to download e-invoices in various formats approved by tax authorities, using the Zuora e-invoicing feature integrated with Avalara.

If your Zuora e-invoicing feature is integrated with Avalara, you can download billing documents in multiple formats that the tax authorities or networks have approved. Depending on the country, the system supports the file formats XML, UBL, PNG, or QR codes that each country requires or supports.

1.  In the Invoice Details page, click the more options ![icon-more-options.png](https://zuora.deploy.heretto.com/v4/deployments/QPAZk6lsgXwvotedNERE/object/32b4c7a7-4cc5-4018-a985-43a366869c65?jwt=eyJ0eXAiOiJKV1QiLCJhbGciOiJIUzI1NiJ9.eyJodHRwczovL2pvcnNlay5jb20vZXpkX29yZ2FuaXphdGlvbiI6Inp1b3JhIiwiaHR0cHM6Ly9qb3JzZWsuY29tL2V6ZC9vYmplY3RfdXVpZCI6IjMyYjRjN2E3LTRjYzUtNDAxOC1hOTg1LTQzYTM2Njg2OWM2NSIsImV4cCI6MTc2NjY4NzU4MSwianRpIjoiZWExMzk3NDNhMDM2NDFhNWIwZTUzYWMyOWU5ZjlkZDIiLCJodHRwczovL2pvcnNlay5jb20vZXpkX2ZpbGVzZXQiOiJWZHdCeUJjM0lBa01wRU9LSFdxZCJ9.7ybiefqrMQXWjRHnjSbLb0B9JVIEb-2ln-63ZGf07w4) icon.
2.  Select Download E-Invoice.
3.  In the Download E-Invoice dialog window, select a file format to download the file.

    Note:

    -   All the file formats that are configured in the Business region settings are available for selection. The default file format shows with the 'Default File Format' label along with the file format. When an invoice is processed by Sovos or PEPPOL, you can download the e-invoice only in the default file format
    -   The e-invoice file can be downloaded when the e-invoice status is Success, ConditionalSuccess, or Generated.
