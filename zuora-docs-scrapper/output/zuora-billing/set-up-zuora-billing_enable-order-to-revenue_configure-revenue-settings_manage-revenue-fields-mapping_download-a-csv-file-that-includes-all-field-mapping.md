# Download a CSV file that includes all field mapping

Learn how to download a CSV file containing all field mappings for an Order to Revenue object.

To download a CSV file that includes all field mapping on an Order to Revenue object, do the following:

1.  Navigate to the tab corresponding to the object for which you want to download a CSV:

    -   Booking Template tab for the Booking Transaction object

    -   Billing Template tab for the Billing Transaction object


2.  Click Download Field Mapping at the top right.
