# Use case 3: bill run date recurs on end of month

Explains how to schedule a bill run to execute on the last day of each month, with the invoice and target dates recurring on the first day of the following month.

Suppose you want to schedule a bill run to execute on the end day of each month, the invoice date and target date recur on the 1st day of each following month.

| Execution # | Bill Run Date | Invoice Date | Target Date |
| --- | --- | --- | --- |
| 1 | 06/30/2024 | 07/01/2024 | 07/01/2024 |
| 2 | 07/31/2024 | 08/01/2024 | 08/01/2024 |
| 3 | 08/31/2024 | 09/01/2024 | 09/01/2024 |
