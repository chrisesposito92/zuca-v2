# Tax request mapping between Zuora and Avalara

This topic outlines the mapping of data fields between Zuora and Avalara for tax calculations in various billing scenarios.

This section provides mapping details on how data fields in Zuora are used in Avalara for tax calculation when posting billing documents. The mapping details vary depending on the scenarios:

-   Calculate tax for a charge

-   Calculate tax when issuing a credit

-   Calculate tax when the default tax date is overridden

-   Calculate tax for credit memo and override tax amount calculated by Avalara
