# Use predefined headers and footers

Learn how to use predefined headers and footers in an HTML template through the Zuora UI by following a series of steps.

To use the predefined header and footer in an HTML template through the Zuora UI, perform the following steps:

1.  Follow the steps in Configure page setup for HTML templates to set the top and bottom margins to have space to add a header and footer. The following image shows how the header and footer work with the content within the whole page, given the page size is A4 (210mm x 297mm), left margin is 5 mm, right margin is 5 mm, top margin is 30 mm, and bottom margin is 30 mm:

    ![](https://zuora.deploy.heretto.com/v4/deployments/QPAZk6lsgXwvotedNERE/object/73f8ae9b-7391-47ec-919f-2022ab5fcc46?jwt=eyJ0eXAiOiJKV1QiLCJhbGciOiJIUzI1NiJ9.eyJodHRwczovL2pvcnNlay5jb20vZXpkX29yZ2FuaXphdGlvbiI6Inp1b3JhIiwiaHR0cHM6Ly9qb3JzZWsuY29tL2V6ZC9vYmplY3RfdXVpZCI6IjczZjhhZTliLTczOTEtNDdlYy05MTlmLTIwMjJhYjVmY2M0NiIsImV4cCI6MTc2NjY0MTM0NSwianRpIjoiNDU5YWYwYWMwZWFjNGE5ZWEzNjYwOTNjMjg3M2E1MGEiLCJodHRwczovL2pvcnNlay5jb20vZXpkX2ZpbGVzZXQiOiJWZHdCeUJjM0lBa01wRU9LSFdxZCJ9.KtSFHH1vy5ixfGWAB3W-Yu48km_hsbxCS2DHLOZv-QE)

2.  In the Blocks tab, navigate to to see all the predefined headers.
3.  Drag one of the predefined headers into the HTML template, and place it at the top of the HTML template.
4.  In the HTML template, click the header Row block. You can add additional columns if the default columns cannot meet your needs.
5.  In the header Row block, modify the content for each column. For example, you can drag an image component to add your company logo, or update the text to add the detailed addresses for your company.
6.  Follow the same steps to configure a footer for the template.
