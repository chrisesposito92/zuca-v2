# From Zoura UI

Learn how to navigate the Zoura UI to manage and edit charge definitions.

1.  Navigate to .
2.  Find the charge that contains the target charge definition and enter the edit charge mode.
3.  At the bottom of the Charge Amount section, click Manage Charge Definitions . The Manage Charge Definitions page opens.
4.  In the Charge Definition Mapping List section, click the charge definition number. The Charge Definition page opens, displaying all the information about this charge definition.
