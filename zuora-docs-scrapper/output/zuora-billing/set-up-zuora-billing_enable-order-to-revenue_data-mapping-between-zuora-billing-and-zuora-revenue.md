# Data mapping between Zuora Billing and Zuora Revenue

Understand the data mapping process between Zuora Billing and Zuora Revenue.

With the Orders to Revenue feature, source data from Zuora Billing can be extracted, transformed, and populated into Zuora Revenue. Considering the differences between the two systems, it is important to understand the mapping between Zuora Billing and Zuora Revenue in terms of transaction types and data fields.

-   Predefined field mappings between the two systems fall into the following categories:

    -   Billing data mapping

    -   Booking data mapping

    -   Revenue Recognition Events data mapping

    -   Billing Transaction type mapping
