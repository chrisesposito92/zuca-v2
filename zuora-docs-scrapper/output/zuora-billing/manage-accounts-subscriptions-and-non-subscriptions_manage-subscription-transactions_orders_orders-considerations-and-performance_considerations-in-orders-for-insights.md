# Considerations in Orders for Insights

This article outlines the limitations of the Zuora Insights add-on when the Orders feature is enabled.

The table below describes known limitations in the Zuora Insights add-on when the Orders feature is enabled.

| Feature | Limitations |
| --- | --- |
| Insights Analysis | Insights Analysis is not supported when the Orders feature is enabled. |
