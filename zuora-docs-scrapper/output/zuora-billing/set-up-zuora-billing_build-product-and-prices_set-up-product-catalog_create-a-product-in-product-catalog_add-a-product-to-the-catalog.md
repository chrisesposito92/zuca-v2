# Add a product to the catalog

Learn how to add a new product to your catalog, including specifying product details, setting effective dates, and categorizing the product.

Complete the following steps to add a new product to your product catalog:

1.  Navigate to in the left-hand navigation section.
2.  Click the Products tab if there are tabs at the top of the page.
3.  Click Add New Product . A New Product page opens.
4.  Specify a name for the product and optionally enter a description for it.
5.  Modify the SKU as necessary, or use the automatically-generated SKU.
6.  Select a date for the Effective Start Date field. The effective start date is the day your product becomes available, which is when you can begin adding the product to a subscription. The valid value is in MM/DD/YYYY format.
7.  Select a date for the Effective End Date field.
8.  For the Category field, select one of the following categories, which is used to organize this product:

    -   Base Products : The main list of products that you offer.

    -   Add On Services : This can include setup or consulting fees.

    -   Miscellaneous Products : This can include delinquent payment charges.


9.  Click Save .
