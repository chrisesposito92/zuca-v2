# Renew a subscription using the Zuora UI

This topic explains how to renew a subscription using the Zuora UI by navigating through the Customers and Subscriptions sections, selecting the appropriate amendment type, and activating the renewal.

1.  Navigate to Customers > Subscriptions and select the subscription you want to renew.
2.  Click Amendment .
3.  Enter an Amendment Name . Optionally, enter a Change Description .
4.  Select Renewal from the Amendment Type list, then click Save .
5.  The Amendment Detail section outlines the effect of the renewal.
6.  Click Active . In the Activate Amendment window that pops up, enter a Customer Acceptance Date , then click Save.
