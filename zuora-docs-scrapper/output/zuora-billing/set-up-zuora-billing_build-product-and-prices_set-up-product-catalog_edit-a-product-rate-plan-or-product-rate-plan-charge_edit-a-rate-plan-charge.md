# Edit a rate plan charge

Edit a rate plan charge by accessing the product detail page, making necessary changes, and saving them.

1.  On the product detail page, hover over the target rate plan charge, and click the Edit (pencil) icon.
2.  Make changes to the rate plan charge.
3.  Click Save .

    Note:

    Billing timing cannot be changed if the rate plan is used by a subscription.
