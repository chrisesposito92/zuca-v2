# Enable Multiple Currencies

Learn how to enable the Multiple Currencies feature in your environment.

Ensure that you have the following features enabled:

-   Orders or Orders Harmonization

-   Invoice Settlement

-   Flexible Billing Attributes


To enable the Multiple Currencies feature, perform the following steps:

1.  Check the known restrictions and limitations of the Multiple Currencies feature.
2.  Enable this feature in Sandbox and Production environments with the self-serve interface.

    Note: For more information, see Enable billing features by yourself .


After enabling this feature, you can configure multiple currencies through the Zuora UI and REST API.
