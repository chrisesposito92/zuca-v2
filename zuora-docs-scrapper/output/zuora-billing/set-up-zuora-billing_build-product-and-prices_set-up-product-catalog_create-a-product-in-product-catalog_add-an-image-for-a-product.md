# Add an image for a product

Learn how to upload an image to identify a product in the catalog.

After the product is added to the catalog, complete the following steps to upload an image to identify the product:

1.  On the Products page, click the product you want to update.
2.  In the Product Information section, click Edit .
3.  In the Upload new image field, click Choose File to search for the image you want to upload.
4.  Select the image, then click Upload . Zuora Billing supports the following image file formats: .jpg, .gif, .png, or .bmp.
5.  Click Save .
