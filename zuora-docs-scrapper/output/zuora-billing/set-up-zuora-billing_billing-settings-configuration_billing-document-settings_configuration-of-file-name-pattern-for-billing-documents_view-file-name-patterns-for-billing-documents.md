# View file name patterns for billing documents

Learn how to view and manage file name patterns for billing documents such as invoices, credit memos, and debit memos through the Zuora UI.

You can view detailed information about the name patterns for the files that are generated for billing documents through the Zuora UI. Billing documents include invoices, credit memos, and debit memos.

The Credit and Debit Memos feature is only available if you have the Invoice Settlement feature enabled. The Invoice Settlement feature is in Limited Availability. If you wish to have access to the feature, submit a request at Zuora Global Support .

To view detailed information about file name patterns for billing documents through the Zuora UI:

1.  Click your username at the top right, and navigate to .
2.  On the Document Filename Patterns tab, view information about file name patterns for different types of billing documents.

The Document Filename Pattern List area displays information about the file name patterns configured for billing documents.

| Column Field | Description |
| --- | --- |
| Document Type | The type of billing documents. |
| Filename Pattern | The current file name pattern configured for the corresponding type of billing documents. |
| Action | The action that can be performed to manage the file name pattern. |
