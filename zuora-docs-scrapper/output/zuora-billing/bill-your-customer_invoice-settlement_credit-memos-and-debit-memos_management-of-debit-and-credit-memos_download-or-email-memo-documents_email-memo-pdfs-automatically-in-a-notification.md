# Email memo PDFs automatically in a notification

Learn how to email memo PDFs automatically in a notification

You can send memo PDFs automatically to your customers by configuring the memo notifications. When a certain memo event is triggered, the generated memo PDF is sent to your specified email addresses.

To auto-send the memo PDFs in a notification, complete the following steps:

1.  Navigate to Extension Studio > Events & Notifications in the left navigation menu.
2.  Locate the notification you want to edit and click the Edit icon ![icon-edit-gray.png](https://zuora.deploy.heretto.com/v4/deployments/QPAZk6lsgXwvotedNERE/object/8e1f025d-0818-475a-998e-aeb6e880baa6?jwt=eyJ0eXAiOiJKV1QiLCJhbGciOiJIUzI1NiJ9.eyJodHRwczovL2pvcnNlay5jb20vZXpkX29yZ2FuaXphdGlvbiI6Inp1b3JhIiwiaHR0cHM6Ly9qb3JzZWsuY29tL2V6ZC9vYmplY3RfdXVpZCI6IjhlMWYwMjVkLTA4MTgtNDc1YS05OThlLWFlYjZlODgwYmFhNiIsImV4cCI6MTc2NjY1MTcxMCwianRpIjoiMjEwNjJiYTNjMGVmNGQyMWE2ZWI3NjlmY2ZhY2U1ZTkiLCJodHRwczovL2pvcnNlay5jb20vZXpkX2ZpbGVzZXQiOiJWZHdCeUJjM0lBa01wRU9LSFdxZCJ9.D5TM6zjJL74D66dAGrZpVOCAkRSH-D5eoK1a9JcvjC0) .
3.  In the Edit Notification Definition dialog, select Include Credit Memo PDF or Include Debit Memo PDF check box in the Delivery Options panel.
4.  Click Save.

    For more information about editing email addresses, see Create and Edit Email Templates. For more information about the memo events, see Standard events for Zuora Payments.
