# Add new reason codes

This topic explains how to add and activate new reason codes for order returns.

To add a new reason code, perform the following steps:

1.  Click Add New Reason Code .
2.  In the Reason Code Name column, enter your reason for returning an order in the empty field, then click Save . Once saved, the reason code is activated automatically.
