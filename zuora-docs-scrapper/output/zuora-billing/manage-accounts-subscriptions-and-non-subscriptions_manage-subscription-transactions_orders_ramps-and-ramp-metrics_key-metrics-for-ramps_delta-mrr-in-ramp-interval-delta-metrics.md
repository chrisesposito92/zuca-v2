# Delta MRR in Ramp Interval Delta Metrics

Delta MRR metrics illustrate the changes in Monthly Recurring Revenue between different subscription versions when an order is created, focusing on intervals where MRR changes occur.

Delta MRR (Monthly Recurring Revenue) in Ramp Interval Delta Metrics measures the change in recurring revenue between two consecutive ramp intervals in a subscription deal.
