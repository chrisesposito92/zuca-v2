# Basic information

The following table lists the information about the invoice schedule displayed in the Basic Information section.

| Field | Description |
| --- | --- |
| Customer Account | The number of the customer account that the invoice schedule belongs to. |
| Order Number | The number of the order associated with the invoice schedule. |
| Notes | The comments on the invoice schedule. |
