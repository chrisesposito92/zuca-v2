# Configure Zuora E-Invoicing feature

Learn how to configure the Zuora E-Invoicing feature in both sandbox and production environments.

You must configure E-Invoicing feature in Zuora sandbox and production environments separately.

To configure the Zuora E-Invoicing feature, complete the following tasks:

1.  Configure Sovos as the service provider.

    -   In Zuora sandbox environment, configure Sovos as the service provider and select the Test Environment checkbox. This service provider is used for the Sovos sandbox environment.

    -   In Zuora production environment, configure Sovos as the service provider and clear the Test Environment checkbox. This service provider is used for the Sovos production environment. For more information, see [Configure e-invoicing service providers](/zuora-billing/bill-your-customer/leverage-advanced-capabilities/e-invoicing/e-invoicing-feature-configuration/e-invoicing-service-providers-configuration).


2.  Configure the business region.

    -   In Zuora sandbox environment, configure a business region and associate it with the Sovos service provider.

    -   In Zuora production environment, configure a business region and associate it with the Sovos service provider. For more information, see [Configure e-invoicing business regions](/zuora-billing/bill-your-customer/leverage-advanced-capabilities/e-invoicing/e-invoicing-feature-configuration/e-invoicing-business-regions-configuration).
