# Define custom fields for Vertex flexible fields

Learn how to define custom fields for Vertex flexible fields in Zuora.

To define custom fields for Vertex flexible fields:

1.  In your Zuora tenant, go to .
2.  Select Invoice Tax Fields.
3.  Add a custom field with the API name in the following format: `tax_vertex_flexible_code_field_[Field ID]__c`
4.  Click Save.
