# Billing document templates customization using Word Mail Merge

This topic guides you through customizing billing document templates, such as invoices and memos, using Microsoft Word Mail Merge.

This article explains how to create a custom billing document ([invoices](/zuora-billing/bill-your-customer/billing-document-templates-configuration/create-word-templates-for-invoices/overview-of-creating-word-template), [credit memo, and debit memo](/zuora-billing/bill-your-customer/invoice-settlement/credit-memos-and-debit-memos/overview-of-credit-memos-and-debit-memos)) template using the Microsoft Mail Merge feature. It also explains the Mail Merge features that apply to the billing document template. For more specific information about Mail Merge, refer to the Microsoft Office Word documentation. If you are using a version of Word that this article does not cover, refer to the Microsoft Office Word documentation for the exact steps to work with the Mail Merge features.
