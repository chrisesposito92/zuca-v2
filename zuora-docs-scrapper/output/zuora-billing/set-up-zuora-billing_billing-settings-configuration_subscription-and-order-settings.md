# Subscription and order settings

This topic explains about configuring subscription and order settings to define default values and payment terms for end customers.

The subscription and order settings include configurations that have an influence when you create a subscription or order for your end customers. For example, you can define the default values used in a new subscription or order or define the payment terms that can be selected for the customer.
