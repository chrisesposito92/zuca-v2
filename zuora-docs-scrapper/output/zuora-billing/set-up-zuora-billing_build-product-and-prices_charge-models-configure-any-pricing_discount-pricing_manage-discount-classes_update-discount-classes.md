# Update discount classes

Learn how to update settings and manage the priority of existing discount classes.

If you want to change some settings of an existing discount class, complete the following steps:

1.  In the Actions column of the existing discount class, click Edit .
2.  Update the following information for the discount class.
    1.  In the Discount Class column, enter the discount class name.
    2.  In the Active column, specify the status of the discount class.
    3.  In the Actions column, click Save to save the configurations.

        You can also change the sequence of existing discount classes. If you want to update an existing discount class to have a higher priority, click the up arrow at the left to move it to the preceding row. If you want to update an existing discount class to have a lower priority, click the down arrow at the left to move it to the next row.

        Note that a discount class cannot be renamed after being created.
