# Best practices for using the FilterByRef function

Enhancements to the FilterByRef function improve performance in HTML templates using standalone custom objects for faster PDF generation.

The performance of HTML templates that use the FilterByRef function with standalone custom objects has been enhanced for faster PDF generation.

A standalone custom object is a custom object that does not have a relationship with any standard Zuora business object, for example, Invoice, Invoice Item, Payment, Credit Memo, or Debit Memo.
