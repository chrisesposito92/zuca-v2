# Allow feature changes in subscriptions

Learn how to enable feature changes in subscriptions by editing product settings in Zuora Billing.

For each product, you can control whether to allow your users to add or remove features while creating or amending a subscription. After adding a product to the product catalog, you edit the product to specify this setting.

1.  If you haven't done so, enable the Feature Specifications setting in the Zuora Billing settings. Otherwise, this setting is not available on the product details page.
2.  Navigate to the Products page and click the product you want to update. The product details page opens.
3.  In the Product Information section, click Edit .
4.  Select the checkbox for the Allow Feature Changes in Subscriptions field to enable feature changes.
5.  Click Save .
