# Export HTML templates

Learn how to export customized HTML templates through the Zuora UI.

After an HTML template is created and customized, you can export it through the Zuora UI if needed.

To export an HTML template through the Zuora UI, perform the following steps:

1.  Click your avatar at the top right and navigate to .
2.  On the Manage Invoice, Credit/Debit Memo Templates tab, click the HTML template that you want to export. The selected template is displayed in the online HTML template editor.

4.  In the Export Template dialog that is displayed, click Save As File to export it.

A JSON file is exported and downloaded automatically with the existing template name.

In the Export Template dialog, you can also choose to click Copy to copy the template data displayed in the Template section, and then paste the data into a file to save it.
